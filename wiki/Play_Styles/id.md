Halaman ini akan mencantumkan secara terperinci berbagai macam cara yang berbeda di mana pemain dapat memilih untuk bermain di mode permainannya masing-masing.

![osu!std](Osu.gif "osu!std") osu! Standard
========================================

![Mouse](Playstyles M.png "Mouse") Mouse-saja
-------------------------------------------------------

Ini dulunya satu-satunya cara untuk bermain osu! standar sampai pengembang menambahkan metode lain. Juga dikenal sebagai mouse-murni,pemain mouse-saja menavigasi kursor di dalam game dengan mouse dan juga mengklik iramanya, mengikuti slider dan memutar spinner dengan hanya klik kiri / kanan mouse. Secara luas dianggap sebagai metode yang paling sulit untuk dimainkan, pemain mouse-saja yang berperingkat tinggi dalam peringkat keseluruhan biasanya sangat dihormati oleh komunitas.

Hamparan kuncinya menunjukkan klik kiri / kanan sebagai M1 dan M2.

![Mouse](Playstyles M.png "Mouse")![Mouse](Playstyles M.png "Mouse") Mouse Ganda
----------------------------------------------------------------------------------------------------

Mouse Ganda adalah gaya bermain di mana pemain memiliki mouse di kedua tangannya menggunakan salah satu mouse untuk menekannya dan satu mouse lagi untuk mengarahkannya / menggerakkannya . Mouse Ganda dipakai untuk dianggap sebagai "lelucon" gaya bermain untuk kebanyakan orang sampai Mei 2013 ketika seorang pemain Kanada bernama ![Canadian Flag](Ca.gif "Canadian Flag") [Azer](http://osu.ppy.sh/u/2155578) mulai mendapatkan skor pada maps seperti [With a Dance Number](http://osu.ppy.sh/s/29691) dan [The Big Black](http://osu.ppy.sh/s/41823) ketika bermain dengan menggunakan Mouse Ganda. Hal serupa juga terjadi untuk pemain Mouse Ganda untuk memplester mouse sebelah kiri mereka ke meja mereka.

![Mouse](Playstyles M.png "Mouse")![Keyboard](Playstyles K.png "Keyboard") Mouse + Keyboard
---------------------------------------------------------------------------------------------------------

Ini adalah gaya bermain yang paling populer untuk osu! standar karena hanya menggunakan perangkat keras komputer yang paling dasar yang merupakan mouse dan keyboard. Mouse + pemain Keyboard memainkan hampir persis dengan cara yang sama seperti pemain mouse-saja dengan perbedaannya hanyalah 2 tombol keyboard (z dan x sebagai opsi biasa) yang digunakan sebagai pengganti mouse itu sendiri untuk mengklik. Beberapa pemain hanya menggunakan keyboard untuk klik sementara yang lain hanya beralih ke keyboard saat streaming (rentetan irama yang sangat cepat) dan menggunakan klik mouse untuk keadaan lainnya.

Hamparan kuncinya menunjukkan klik keyboard kiri / kanannya sebagai K1 dan K2.

![Tablet](Playstyles T.png "Tablet") Tablet tap (atau Tablet-saja)
--------------------------------------------------------------------------

![osu!tablet](Osu!tablet.jpg "osu!tablet") Sebuah gaya bermain bagi mereka yang memiliki tablet grafis (Digitizer). Pemain tablet-saja menggunakan pena tablet bukannya mouse untuk menavigasi kursor di dalam game tapi mengklik iramanya dengan menekan pena ke permukaan tablet grafis itu sendiri tanpa perlu keyboard.

Hamparan input akan menampilkan semua klik Anda sebagai M1.

![Tablet](Playstyles T.png "Tablet")![Keyboard](Playstyles K.png "Keyboard") Tablet + Keyboard
----------------------------------------------------------------------------------------------------------

Sebuah gaya bermain populer bagi mereka yang memiliki tablet grafis (Digitizer). Pemain Tablet + Keyboard menggunakan pena tablet bukannya mouse untuk menavigasi kursor di dalam game dan mengklik ketukan menggunakan keyboard atau tombol tablet. Sama seperti mouse + Keyboard, beberapa pemain juga hanya menggunakan keyboard untuk mengklik sementara yang lain akan beralih ke keyboard saat sedang streaming.

Kadang-kadang disebut sebagai 'tapx', komunitas telah menghadirkan istilah ini untuk menggambarkan bagaimana streaming dengan Tablet-saja dapat bekerja. Untuk streaming sebagai pemain Tablet-saja, Anda bergantian antara menekan pena Anda pada tablet dan menekan salah satu tombol yang ditetapkan (karena hanya tombol klik sekunder akan bekerja, dan 'x' adalah default tombol klik sekunder).

![Touchscreen](Playstyles TC.png "Touchscreen") Layar sentuh
-----------------------------------------------------------

Sebuah gaya bermain memungkinkan bagi pemain yang memiliki monitor sentuh yang aktif, layar laptop atau tablet. Pemain menyentuh layar untuk seketika menavigasi dan klik ketukan iramanya. Gaya bermain ini mungkin merasa sedikit mirip dengan bermain osu! streaming / osu! droid karena Anda akan menyentuh layar secara langsung bukan dengan dukungan hardware (mouse, tablet grafis). Beberapa pemain memilih untuk bermain gaya ini ketika mereka tidak memiliki tablet grafis.

Hamparan kuncinya menunjukkan input sentuhan Anda hanya sebagai M1.

![Touchscreen](Playstyles TC.png "Touchscreen")![Keyboard](Playstyles K.png "Keyboard") Layar sentuh + Keyboard
------------------------------------------------------------------------------------------------------------------

Sebuah gaya bermain dipakai oleh orang-orang yang memiliki layar monitor / laptop sentuhan-aktif, gaya bermain ini sebagian besar diadopsi oleh pemain layar sentuh yang tidak dapat melakukan [streams](Streams "wikilink") hanya dengan menggunakan layar sentuhnya saja. Gaya bermain ini dapat mirip dengan Tablet + Keyboard karena beberapa monitor membolehkan bagi pena untuk digunakan sebagai pengganti jari Anda.

Gaya lainnya
------------

**Panel sentuh laptop/trackpad**: Ketika secara teoritis memungkinkan, sebagian besar pemain memilih untuk tidak menggunakan trackpad karena tingkat kesulitan ekstrim yang terlibat (secara spesifik, mengendalikan gerakan kursor pada ruang yang terbatas dari laptop panel sentuh / trackpad). Trackpad digunakan (dan bukannya mouse) untuk menavigasi kursor dalam game.

**Joystick**: Sebuah cara kreatif untuk mengontrol Kursor osu!. Pemain menahan dan menggerakkan joystick sekeliling untuk memindahkan kursor osu! dan mengklik pada pemicu / permukaan tombolnya (tergantung pada desain joystick pemain) atau tombol keyboard untuk mendaftarkan ketukannya. Hal ini paling tidak disukai karena keterampilan yang diperlukan untuk mengontrol gerakan joystick dan kebanyakan tidak menggunakan itu kecuali mereka juga memainkan game lainnya ([World of Warplanes](https://wikipedia.org/wiki/World_of_Warplanes) sebagai contohnya) .

**Metal plate**: Sebuah cara yang khusus untuk menggantikan keyboard untuk mengklik ketukan. Metode ini diperkenalkan oleh [geckogates](https://osu.ppy.sh/u/geckogates) dan membutuhkan beberapa kabel dan pengetahuan elektronik untuk membuatnya. Anda akan melampirkan kawat ke jari Anda untuk mengklik (terhubung ke sirkuit keyboard untuk z, x) dan akan menyentuh pelat logam untuk mendaftar ketukannya.

![osu!taiko](Taiko.gif "osu!taiko") Taiko
====================================

![Keyboard](Playstyles K.png "Keyboard") Keyboard
-----------------------------------------------------

Ini dulunya satu-satunya cara untuk bermain Taiko sampai pengembang menambahkan metode lain. Sederhananya, Anda menekan tombol keyboard yang ditugaskan untuk "memukul" bagian-bagian drum; x, c untuk pusat gendang (Red Don) dan z, v untuk gendang rim (Blue Katsu). Biasanya, pemain Taiko terampil menggunakan kontroler Taiko mereka (baik membeli atau membuatnya sendiri) untuk bermain, sebagian besar pemain menggunakan gaya bermain ini seharusnya mereka tidak memiliki kemewahan memiliki drum Taiko.

Ada dua gaya utama untuk mengatur penempatan tombol di keyboard, KDDK [Taiko default] or (DDKK/KKDD). Ini merupakan kombinasi dari bagaimana Anda mengatur kunci Anda untuk memukul notnya. Mungkin ada yang lain, kombinasi aneh misalnya, tetapi itu hampir tidak pernah digunakan.

Untuk tombol kuncinya, opsi biasanya adalah zxcv (KDDK).

Taiko Kontrol
-------------

Opsi kontrol untuk [Taiko no Tatsujin](https://wikipedia.org/wiki/Taiko_no_Tatsujin) seri([Taiko Drum Master](https://wikipedia.org/wiki/Taiko:_Drum_Master) untuk USA). Kebanyakan pemain terampil bermain menggunakan Taiko kontroler dibeli atau dibuat dari awal. Pemain akan membawa di kedua tangannya masing-masing sebuah taiko / stik drum dan memukul permukaannya atau tepi drum untuk mendaftar not berdasarkan mana yang ia pukul. Kebanyakan kontrol Taiko terhubung ke komputer dengan menggunakan konektor Bluetooth jika tidak ada USB connecter.

Gaya lainnya
------------

**Gamepad**: Salah satu dari gaya bermain yang memungkinkan di variasi konsol Taiko no Tatsujin seri(termasuk Taiko Drum Master). Pemain mendaftarkan Don/Katsu dengan menekan permukaan [R Don], panah kiri-kanan [L Don] dan panah atas-bawah [L/R Katsu].

![osu!catch](Ctb.gif "osu!catch") Catch the Beat
=========================================

![Keyboard](Playstyles K.png "Keyboard") Keyboard
-----------------------------------------------------

Yang paling umum digunakan dan gaya bermain yang biasa untuk mode ini. Pemain menekan kiri / kanan tanda panah untuk memindahkan penangkap, dan juga dapat menekan Shift Kiri untuk memicu Dash! Mode (kecepatan x2). Seperti biasa, itu mungkin untuk menyunting kunci yang akan digunakan; beberapa pemain menikmati a / d untuk menggerakannya dan spasi / Kanan Shift untuk Dash! efek.

Joystick
--------

Sebuah gaya bermain yang memungkinkan bagi pemain yang menggunakan gaya bermain lawas seperti skema kontrol permainan arkade yang berusia puluhan tahun. Pemain memindahkan kepala joystick dan menekan tombol permukaan / pemicunya / tombol keyboard untuk memicu Dash! efek.

Gamepad
-------

Sebuah gaya bermain yang memungkinkan bagi pemain yang merasa nyaman bermain dengan gamepad. Pemain menekan kiri / kanan anak panah dari gamepad untuk memindahkan penangkap kiri / kanan dan menekannya kemudian menahan tombol permukaan untuk memicu Dash! efek.

![osu!mania](Mania.gif "osu!mania") osu!mania
========================================

![Keyboard](Playstyles K.png "Keyboard") Keyboard
-----------------------------------------------------

[Panduan eksternal untuk Keyboard Mekanik](http://www.daskeyboard.com/blog/mechanical-keyboard-guide/)
Ini dulunya adalah satu-satunya cara untuk bermain osu!mania sampai pengembang menambahkan metode lain. Pemain menempatkan jari mereka pada tombol keyboard sehubungan dengan pengaturan kuncinya dan saat ini bermain menggunakan pengaturan kunci. Keuntungan dari gaya bermain ini adalah bahwa hal itu pada dasarnya dapat menampung semua pengaturan 4/5/6/7/8 kunci. Satu-satunya titik tangkap mulai dapat terbiasa dan efek ghosting pada keyboard biasa ini / tombol membran (kecuali jika Anda menggunakan keyboard gaming yang layak).

[DJMAX](https://wikipedia.org/wiki/DJMax)/O2Jam/[Beatmania IIDX](https://en.wikipedia.org/wiki/Beatmania_IIDX)/dll. (7K) kontrol arkade
------------------------------------------------------------------------------------------------------------------------------------------------------

Gaya bermain disetel untuk veteran berpengalaman di game irama layaknya bergaya DJ. Kebanyakan pemain keras yang disiplin ini biasanya memiliki kontroler arkade pribadi mereka sendiri sesuai pilihan mereka. Sejak 8 kunci memerlukan keputusan mapper atau Pengubah Permainan, sebagian besar pemain yang memiliki kontroler arkade ini dapat menangani hingga 8 kunci (maksimum biasa) dan 1 khusus.

Gaya lainnya
------------

**Dancepad**(4/8K): Opsi biasa untuk [Dance Dance Revolution]([Dance Dance Revolution](https://wikipedia.org/wiki/Dance_Dance_Revolution)) seri ([Stepmania](https://wikipedia.org/wiki/StepMania) untuk berbasis komunitas). Pemain menari di depan dancepad untuk memasukan perintahnya. Tergantung pada dancepad dan preferensi pemainnya, itu memungkinkan untuk menggunakan tombol permukaan sebagai perintah masukan daripada / bawah / atas / tombol panah kanan kiri yang biasa. Kelemahan terbesar adalah input maksimum ketika bisa langsung 4 kali pada satu waktu (tangan dan kaki), yang sangat merugikan mengingat beatmaps saat ini bisa melaju ke lebih dari itu.

**Gitar/set Drum**(5/6K): Opsi biasa untuk game bertemakan gitar([Guitar Hero](https://wikipedia.org/wiki/Guitar_Hero))/bertemakan drum([Drummania](https://wikipedia.org/wiki/Drum_Mania)) dalam game irama. Pemain stream pada gitar mereka / memukul gendang atau simbal untuk perintah masukan. Pemain drum memiliki kelemahan dibandingkan dengan pemain gitar karena mereka hanya dapat memasukkan dua perintah (dua tangan) pada satu waktu tanpa tendangan pedal khusus.

**Kontrol Para Para**(5K): Opsi biasa untuk [Para Para Paradise](https://wikipedia.org/wiki/Para_Para_Paradise) seri. Pemain menggerakkan lengan mereka, siku dan tangan lurus di atas sensor dekat dengan perintah masukan berdasarkan posisi tangan melalui sensor dekat. Kekurangannya adalah bahwa Anda hanya memiliki dua lengan untuk perintah input, tidak termasuk penggunaan kaki.

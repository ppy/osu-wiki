# Versión (desambiguación)

**Versión** puede tener múltiples significados:

- Una [dificultad](/wiki/Beatmap/Difficulty) en un [beatmap](/wiki/Beatmap)
- El número de versión de un [archivo `.osu`](/wiki/Client/File_formats/osu_(file_format))
- El número de versión de una [skin](/wiki/Skinning)
- El número o fecha de compilación de un cliente de osu!
- El [canal de lanzamiento](/wiki/Client/Release_stream) de un cliente de osu!
  - `Stable` o `Cutting Edge (Experimental)` para osu!(stable)
  - `Lazer` o `Tachyon (Unstable)` para osu!(lazer)

# 版本（消歧义）

**版本 (Version)** 可以指：

- [谱面](/wiki/Beatmap)中的一个[难度](/wiki/Beatmap/Difficulty)
- [`.osu` 文件](/wiki/Client/File_formats/osu_(file_format))的版本号
- [皮肤](/wiki/Skinning)的版本号
- osu! 客户端的版本号或构建日期
- osu! 客户端的[发行通道](/wiki/Client/Release_stream)
  - 对于 osu!(stable)：`正式版 (Stable)` 或 `测试版 (Cutting Edge)`
  - 对于 osu!(lazer)：`Lazer` 或 `Tachyon (Unstable)`

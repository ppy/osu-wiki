# osu! FumoFumo Cup: cirno

![OFFC: cirno logo](img/logo.png)

The **osu! FumoFumo Cup: cirno** (***OFFC: cirno***) is a 4v4 double-elimination osu!standard tournament hosted by ::{ flag=CN }:: [kitakore](https://osu.ppy.sh/users/33887163). It is the inaugural season of the osu! FumoFumo Cup tournament series.

## Tournament schedule

| Event | Timestamp |
| --: | :-- |
| Registration phase | 2024-08-10/2024-08-30 |
| Team-up phase | 2024-08-20/2024-09-06 |
| Quarterfinals | 2024-09-09/2024-09-17 |
| Semifinals | 2024-09-18/2024-09-30 |
| Finals | 2024-10-01/2024-10-07 |
| Grand Finals | 2024-10-08/2024-10-20 |

## Prizes

| Placing | Prize(s) |
| :-: | :-- |
| ![Gold crown](/wiki/shared/crown-gold.png "1st place") | One [MeowPad](https://desu.life#device) for each team member (provided by *DESU.Life*) |
| Top three teams | Corresponding KanonBot badge for each team member |
| All players | Title in OFFC official QQ group, animated profile banners (team customized) |

Apart from the prizes above, the following players as chosen by the members of the official OFFC QQ group will also be awarded with:

| Player | Prize |
| :-: | :-- |
| MVP (from 1st place team) | One AmiAmi Cirno Fumo |
| SVP (from 2nd place team) | Blue Archive Official Artworks Collection |
| TVP (from 3rd place team) | One osu! themed mug |

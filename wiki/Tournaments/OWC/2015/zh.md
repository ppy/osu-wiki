# osu! 世界杯 2015

![OWC 2015 标志](img/logo.jpg)

**osu! 世界杯 2015** (***OWC 2015***) 是由 [osu! 团队](/wiki/People/osu!_team)举办的，基于国家/地区的 osu! 锦标赛。本届 osu! 世界杯是第六届。

## 赛程

| 阶段 | 时间 |
| --: | :-- |
| 报名 | 2015-10-01/2015-10-18 |
| 直播抽签 | 2015-11-01 (14:00 UTC) |
| 小组赛 | 2015-11-07/2015-11-08 |
| 十六强赛 | 2015-11-15 |
| 四分之一决赛 | 2015-11-21/2015-11-22 |
| 半决赛 | 2015-11-28/2015-11-29 |
| 决赛第一周 | 2015-12-05/2015-12-06 |
| 决赛第二周 | 2015-12-13 |

## 规则

### 锦标赛规则

1. osu! 世界杯是按国家/地区组队的，4v4 的团队比赛。
2. 比赛谱面计分采用称作 Score V2 的新机制。**[了解更多信息！](https://osu.ppy.sh/community/forums/topics/375428)**
3. 每一轮的比赛选图由选图人员在比赛开始之前的周日提前公布。只有这些图会在相应的比赛中使用。
   - 有一张图会作为决胜 (tiebreaker) 图，只有仅剩一分且平局时才使用这张图。
4. 比赛日程将由比赛委员会拟定（见下）。
5. 若组织者或裁判均无空闲时间，比赛将会推迟。
6. 失败玩家的分数不会计入队伍总分。
   - 若玩家在游玩期间复活并最终通过，则其分数计入队伍总分。
7. 允许调整[视觉设置](/wiki/Client/Interface/Visual_settings)选项。
8. 若某局比赛出现同分，该局比赛无效。
9. 若有任何玩家掉线，视作该玩家失败。
   - 若玩家在游玩开始的 30 秒内掉线，则可进行重赛。重赛决定由裁判给出。
10. 在有效局比赛中使用过的图，不可使用第二次。
11. 若参与比赛的玩家不足 4 人，比赛可推迟最多 10 分钟。
12. 允许比赛中换人。
13. 卡顿不能作为重赛某张图的理由。
14. 所有玩家都应维持比赛顺畅无延误地进行。若玩家有意拖延比赛，裁判可对其施加惩罚。
15. 若玩家在非游玩期间掉线，且队伍无法换人，比赛可推迟最多 10 分钟。
16. 必须对所有玩家与裁判持有敬意。请遵循裁判与比赛委员会的指令。标注为“最终”的决定不得提出异议。
17. 严令禁止通过作弊扰乱比赛，选用不合适的热手图（见下），辱骂或挑衅其他玩家与裁判，耽误比赛进程，以及蓄意做出其它不当行为。
18. 多人游戏聊天遵循 [osu! 社区规定](/wiki/Rules)。所有聊天规定亦适用于多人游戏聊天。
    - 违反聊天规定将导致禁言。被禁言的玩家无法参与多人比赛，必须暂时换下。
19. 在小组赛阶段，“默认获胜”记作 4:0 获胜，+1.0 倍的分数计算。
20. 意外事故由比赛委员会进行处理。裁判可酌情放宽标准，这由他们决定。
21. 违反锦标赛规则的惩罚包括：
    - 指定玩家不得游玩一张图
    - 指定玩家不得参与整局比赛
    - 宣布“默认输掉”比赛
    - 取消整场锦标赛参赛资格
    - 取消本场及未来所有锦标赛的参赛资格，除非进行申诉
22. 对此规则的任何修改都会进行通知。

### 比赛报名

1. 所有有意向加入国家/地区队的用户单独报名。
   - 比赛委员会将建立一个名单，列出国家/地区队的潜在候选人。
   - 比赛委员会指定一位候选人担任所在国家/地区队的队长（尽管是临时的）。
   - 被指定的队长可以从其国家/地区的候选人名单中选人，建立起一支队伍。
2. 为保证报名正规有效，所有报名用户均会经过比赛委员会核查。
   - 所有报名用户将被分配到对应国家的候选人名单中。
   - 为获准列入名单，请确保 osu! 模式的全球排名高于 \#5000。
   - 为获准列入名单，请确保在最近 12 个月内未违反 [osu! 社区规定](/wiki/Rules)。
3. 所有成功建立的队伍将于报名阶段后公布。
4. 被拒绝的玩家可通过联系 [tournaments@ppy.sh](mailto:tournaments@ppy.sh) 进行申诉。
5. 选图人员不可作为玩家参与比赛。

### 阶段说明

1. 在第一个阶段（小组赛）中，所有队伍分成 8 组，每组有 4 支队伍。
2. 每小组中的任一队伍都会跟同一小组中的其他队伍进行比赛。
3. 通过以下优先级对每个队伍进行排名：
   - 最高比赛胜场数。
   - 最高 `{(谱面胜场数) - (谱面败场数)}`。
   - 最高谱面胜场数。
   - 最高 `∑{(总分差) / (最高分数)}`。
   - 重赛的获胜者。
4. 小组赛中每组排名前二的队伍将进入双败淘汰赛阶段。
5. 接下来的阶段均采用双败淘汰制。获胜的队伍将会晋级并进入下一场比赛，没有获胜的队伍将归入败者组。
6. 根据[该图片](/wiki/shared/stages-visual.png)将擂台划分如下：

| 阶段 | 比赛 ID |
| :-- | :-- |
| 十六强赛 | A, B, C, D, E, F, G, H |
| 四分之一决赛 | I, J, K, L & R, S, T, U |
| 半决赛 | M, N & V, W, X, Y, Z,AA |
| 决赛第一周 | O & AB, AC, AD, AE |
| 决赛第二周 | P, Q |

7. **获胜条件：**
   - 小组赛中，你需要在比赛中获得四张图的胜利。（七局四胜）
   - 在十六强赛与四分之一决赛中，你需要在比赛中获得五张图的胜利。（九局五胜）
   - 在半决赛与第一周决赛中，你需要在比赛中获得六张图的胜利。（十一局六胜）
   - 在第二周决赛中，你需要在比赛中获得七张图的胜利。（十三局七胜）

### 比赛说明

1. 裁判将提前 15 分钟创建多人游戏房间。参赛选手必须在这段时间内集合。
   - 房间设置为 osu!、Team-Vs、获胜条件：“Score V2”。房间名必须是“OWC 2015: 蓝队 vs 红队”（**未来可能会更改**）。
   - 房间名中提到的第一个队伍必须是蓝队，第二个队伍必须是红队。
2. 玩家可自由选择最多两张热手谱面。禁止使用带有存疑内容的谱面，包括其视觉元素或歌曲中存在亵渎语言、裸体、暴力或其他工作场所不宜内容，无论语言为何。
3. 每位队长可从图池中选定两张谱面禁用。在整场比赛中，这些谱面不可被任何队伍选出。
   - 必须总是禁用两张谱面。
4. 选图由两位队长轮流进行。
5. 每位队长必须在 `#multiplayer` 频道中使用一次 `!roll` 指令。
   - `!roll` 胜者选择比赛使用的首张谱面。
   - `!roll` 败者先选禁两张谱面。
6. 队长可从任意分类中选择谱面。
   - 仅剩一分且平局时，必须使用决胜 (tiebreaker) 图。
7. 比赛结果将通过统计网站发布。

### 选图说明

1. 小组赛、十六强赛、四分之一决赛、半决赛、决赛各使用一个独立图池。
   - 决赛的两周共用同一个图池。
2. 每个图池含有 5 个分类：NoMod、Hidden、HardRock、DoubleTime 与 FreeMod。
3. 每个图池均包含 19 张图。
4. 每个图池均有一张决胜 (tiebreaker) 图。
5. 游玩 NoMod 图时，不使用任何模组。
6. Hidden、HardRock 以及 DoubleTime 图需使用相应模组游玩。
7. FreeMod 图将会激活所有模组，任何玩家都可以选用 Hidden、HardRock 或者不使用任何模组。
   - 玩家可选择多个模组。
   - 游玩 FreeMod 图时，每队须至少有 2 位玩家使用至少一个模组。
8. 决胜图按 FreeMod 规则游玩。
   - 游玩决胜图时，不强制玩家使用模组。
9. 所有阶段的 NoMod 图数量为 6。
10. 所有阶段的模组指定图数量为 3。

### 赛程安排说明

1. 每个阶段将会使用**一个周末**的时间进行。
2. 小组赛的时间安排可相互重叠。
3. 所有淘汰赛将会在星期六或者星期日 (UTC+0) 进行。
4. 赛事将由赛事管理者进行调度。时间安排将会在第一场赛事前的一个星期日公布。赛事管理者将会尝试根据参赛者所在时区调整比赛时间。
   - 对于四分之一决赛及后续阶段：若下一周内某段时间无法参赛，请在周日前告知比赛委员会。委员会将尽力满足诉求，但不作任何保证。
5. 赛程安排在 osu!wiki 上发布后，任何情况下均不允许重排赛程。
6. 每支队伍的队长负责保证队伍能够参赛。成员更多的队伍更容易确保凑齐至少四名选手进行每场比赛。如果某支队伍无法提供四名选手进行比赛，本场比赛将记为认输。

# Jaye's One Key Event

This is the index page for all tournaments in the Jaye's One Key Event (*JOKE*) series. Tournaments in the JOKE series feature 1-key maps for the [osu!mania](/wiki/Game_mode/osu!mania) game mode.

Closely related to Jaye's One Key Event and organised in part by the same team is the [Sparxe's Two Key Event](/wiki/Tournaments/STKE) tournament series, featuring 2-key maps instead.

# osu! Taiko 世界杯 2014

![TWC 2014 标志](img/logo.jpg)

**osu! Taiko 世界杯 2014** (***TWC 2014***) 是由 [osu! 团队](/wiki/People/osu!_team)举办的，基于国家/地区的 osu!taiko 锦标赛。本届 osu!taiko 世界杯是第四届。

## 赛程

| 阶段 | 时间 |
| --: | :-- |
| 报名 | 2014-01-16/2014-01-26 |
| 直播抽签 | 2014-02-01 (13:00 UTC) |
| 小组赛 | 2014-02-07/2014-02-09 |
| 十六强赛 | 2014-02-15/2014-02-16 |
| 四分之一决赛 | 2014-02-22/2014-02-23 |
| 半决赛 | 2014-03-01/2014-03-02 |
| 决赛第一周 | 2014-03-08/2014-03-09 |
| 决赛第二周 | 2014-03-15 |

## 规则

### 比赛规则

1. osu! Taiko 世界杯 2014 是按国家/地区组队的，3v3 的团队比赛。
2. 每一轮的比赛选图由选图人员在比赛开始之前的周日提前公布。只有这些图会在相应的比赛中使用。
   - 有一张图会作为决胜 (tiebreaker) 图，只有平局时才使用这张图。
   - 此外，还会有 [Hidden](/wiki/Gameplay/Game_modifier/Hidden)、[HardRock](/wiki/Gameplay/Game_modifier/Hard_Rock)、[DoubleTime](/wiki/Gameplay/Game_modifier/Double_Time) 和 FreeMod 的图池分类。
3. 比赛日程将由比赛委员会拟定（见下）。
4. 若组织者或裁判均无空闲时间，比赛将会推迟。
5. 失败玩家的分数不会计入队伍总分。
6. 允许调整[视觉设置](/wiki/Client/Interface/Visual_settings)选项。
7. 若某局比赛出现同分，该局比赛无效。
8. 若有任何玩家掉线，视作该玩家失败。
9. 在有效局比赛中使用过的图，不可使用第二次。
10. 若参与比赛的玩家不足 3 人，比赛可推迟最多 10 分钟。
11. 允许比赛中换人。
    - 每张图只允许更换一名选手。
12. 卡顿不能作为重赛某张图的理由。
13. 在小组赛阶段，“默认获胜”记作 4:0 获胜，+1.0 倍的分数计算。
14. 意外事故由比赛委员会进行处理。
15. 对此规则的任何修改都会进行通知。

### 比赛报名

1. 你的队伍至少需要 **3 名玩家** 才能参赛。
   - 队伍最大人数为 6 人。
   - 你必须指定一名代表队伍的队长。
2. 每支队伍代表一个国家/地区。必须与同一国家/地区的玩家组队。
3. 若要报名队伍，请先[填写此表格](https://docs.google.com/forms/d/1C58_JWKBVVUrJr5so9LBBsqp6LqZktJjGVSZ4pbRhEo/viewform)。然后[向 Loctav 发送](https://osu.ppy.sh/home/messages/users/71366)一条标题为“TWC Registration”来的消息以确认报名。
   - 队长也可以通过[通知委员会](https://osu.ppy.sh/home/messages/users/71366)来修改阵容。
   - 若报名成功送达，你会收到一条确认回复，届时报名状态变为待定。
4. 为确保报名真实有效，任何报名与改动均会经赛事委员会审核。
5. 参赛队伍总数为 20 支。
6. 报名阶段结束后，将公布所有报名成功的队伍。
   - 队长会收到阵容被接受或被拒绝的通知。
7. 选图人员不得以选手身份参加本次比赛。

### 阶段说明

1. 在第一个阶段（小组赛）中，所有队伍分成 4 组，每组有 5 支队伍。
2. 每小组中的任一队伍都会跟同一小组中的其他队伍进行比赛。
3. 通过以下优先级对每个队伍进行排名：
   1. 最高比赛胜场数。
   2. 最高 `{(谱面胜场数) - (谱面败场数)}`。
   3. 最高谱面胜场数。
   4. 最高 `∑{(总分差) / (最高分数)}`。
   5. 重赛的获胜者。
4. 小组赛中每组排名前四的队伍将进入双败淘汰赛阶段。
5. 接下来的阶段均采用双败淘汰制。获胜的队伍将会晋级并进入下一场比赛，未获胜的队伍淘汰出局。
6. **获胜条件：**
   - 小组赛中，你需要在比赛中获得四张图的胜利。（七局四胜）
   - 在十六强赛与四分之一决赛中，你需要在比赛中获得五张图的胜利。（九局五胜）
   - 在半决赛与决赛中，你需要在比赛中获得六张图的胜利。（十一局六胜）

### 比赛说明

1. 裁判将提前 20 分钟创建多人游戏房间。参赛选手必须在这段时间内集合。
   - 房间将被锁定。密码和多人邀请会尽快发送给两位队长。
   - 房间设置为 osu!、Team-Vs.、获胜条件：“Score”。房间名称必须是“Taiko World Cup 2014: 蓝队 vs 红队”。
   - 房间名中提到的第一个队伍必须是蓝队，第二个队伍必须是红队。
2. 选手可自由选择最多两张热手谱面。
3. 选图由两位队长轮流进行。每位队长必须在 `#multiplayer` 中使用一次 `!roll` 来决定先选图一方。
   - 队长可自由选择 NoMod 和 FreeMod 分组中的谱面。
   - 模组限定分组中的谱面有选择次数限制。在整场比赛中，每位队长在每个分组中只能选择一张谱面。
     - FreeMod 分组无选图限制。
   - 仅剩一分且平局时，必须使用决胜 (tiebreaker) 图。
4. 比赛结果将通过统计网站发布。

### 地图说明

1. 小组赛、十六强赛、四分之一决赛、半决赛、决赛各使用一个独立图池。
2. 每个图池含有 5 个分类：NoMod、[Hidden](/wiki/Gameplay/Game_modifier/Hidden)、[HardRock](/wiki/Gameplay/Game_modifier/Hard_Rock)、[DoubleTime](/wiki/Gameplay/Game_modifier/Double_Time) 和 FreeMod。
3. 每个图池均包含 23 张图。
4. 每个图池均有一张决胜 (tiebreaker) 图。
5. 游玩 NoMod 图时，不使用任何模组。
6. Hidden、HardRock 以及 DoubleTime 图需使用相应模组游玩。
7. FreeMod 图将会激活所有模组，任何玩家都可以选用 Hidden、HardRock 或者不使用任何模组。玩家**不得**使用超过一个模组。
   - 游玩 FreeMod 图时，每队须至少有 2 位玩家使用模组。
8. 决胜图按 FreeMod 规则游玩。
   - 游玩决胜图时，不强制玩家使用模组。
9. NoMod 分组的谱面数量为：
   - 小组赛为 10 张
   - 淘汰赛阶段为 6 张
10. 模组限定分组的谱面数量为：
    - 小组赛为 3 张
    - 淘汰赛阶段为 4 张

### 赛程说明

1. 每个阶段将会使用**一个周末**的时间进行。
2. 小组赛比赛可重叠进行。
3. 所有淘汰赛将会在星期六或者星期日 (UTC+0) 进行（参见赛程）。
4. 赛程将由赛事委员会负责安排。赛程会在实际阶段第一场比赛前一个星期日公布（例如，小组赛会在 2 月 2 日公布）。赛事委员会会尽量根据参赛者时区安排赛程。
5. 每支队伍的队长负责保证队伍能够参赛。成员更多的队伍更容易确保凑齐至少三名选手进行每场比赛。如果某支队伍无法提供三名选手进行比赛，本场比赛将记为认输。

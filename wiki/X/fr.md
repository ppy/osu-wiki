# X

osu! possède plusieurs comptes sur **X** (anciennement Twitter), ayant chacun des objectifs spécifiques.
La plupart ont une activité très faible, mais ils permettent de vous tenir informé en dehors de l'écosystème d'osu!.

## Service

| Avatar | Compte | Description |
| :-: | :-: | :-- |
| ![osu!status avatar](img/osustatus.jpg) | [@osustatus](https://x.com/osustatus) | Notifications de problèmes liés au site web et à l'utilisation du site. [Bancho](/wiki/Bancho_(server)). |
| ![osu! support avatar](img/osu-support.png) | [@osusupport](https://x.com/osusupport) | Aide pour les problèmes de compte et/ou de communauté. Géré par l'[équipe d'assistance aux comptes](/wiki/People/Account_support_team). |

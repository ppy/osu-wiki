# ความสัมพันธ์ระหว่างเวลากับระยะทาง (Time-distance equality)

::: Infobox
![](/wiki/shared/editor/editor-draw-distsnap.png "ไอคอนเครื่องมือ Distance snap")
:::

**ความสัมพันธ์ระหว่างเวลากับระยะทาง (Time-distance equality)** คือเทคนิค [การสร้างแมพ](/wiki/Beatmapping/Mapping_techniques) ในโหมด [osu!](/wiki/Game_mode/osu!) ที่ซึ่งระยะห่างระหว่าง [Hit objects](/wiki/Gameplay/Hit_object) ในสนามเล่นจะเป็นสัดส่วนโดยตรงกับระยะเวลาที่ห่างกันของโน้ตเหล่านั้น ภายในตัวแก้ไข Beatmap เทคนิคนี้สามารถบังคับใช้ได้โดยการใช้เครื่องมือ [Distance snap](/wiki/Client/Beatmap_editor/Distance_snap) ซึ่งจะช่วยรักษาสัดส่วนตามค่าตัวคูณระยะห่าง (Distance spacing multiplier) ที่กำหนดไว้

## การใช้งาน

หลักการของความสัมพันธ์นี้สร้างขึ้นจากแนวคิดที่ว่า ความหนาแน่นทางสายตาของ Hit objects ควรจะสอดคล้องกับโครงสร้างของดนตรี: จังหวะที่รวดเร็วหมายถึงวัตถุควรจะวางอยู่ใกล้กัน และในทางกลับกัน การนำหลักความสัมพันธ์ระหว่างเวลากับระยะทางมาใช้จะช่วยให้ Beatmap อ่านง่ายขึ้นและดูเป็นธรรมชาติมากขึ้น เนื่องจากมันช่วยให้ผู้เล่นมีเบาะแสทางสายตาที่สม่ำเสมอว่าควรจะกด Hit object นั้นๆ เมื่อไหร่

แม้ว่าในระดับความยากที่ต่ำจะได้รับคำแนะนำอย่างมากให้ใช้เทคนิคนี้[^rc-note] แต่ในระดับความยากที่สูงขึ้น การทำตามหลักการนี้อย่างเคร่งครัดอาจทำให้ผู้เล่นที่มีประสบการณ์รู้สึกไม่ตื่นเต้นเนื่องจากแมพจะคาดเดาได้ง่ายเกินไป เพื่อแก้ไขปัญหานี้ Mapper มักจะสร้าง [Jump (การกระโดด)](/wiki/Beatmapping/Mapping_techniques/Jumps) ในระดับต่างๆ เพื่อเพิ่มโอกาสในการเคลื่อนที่และสร้างความหลากหลายให้กับแมพ

### ข้อดี

แม้ว่าความสัมพันธ์ระหว่างเวลากับระยะทางในความหมายที่เคร่งครัดจะไม่ได้ถูกนำมาใช้ตลอดทั้งระดับความยาก แต่เทคนิคนี้ก็มีประโยชน์ในหลายด้าน:

- โดยทั่วไป สัดส่วนที่พอดีจะช่วยเพิ่มความสามารถในการอ่าน [จังหวะ (Beats)](/wiki/Music_theory/Beat) ที่อยู่ติดกันซึ่งมี [การแบ่งจังหวะ (Snapping)](/wiki/Client/Beatmap_editor/Beat_snap) ที่ต่างกัน ซึ่งเป็นสิ่งสำคัญมากในส่วนที่มี [จังหวะ (Rhythm)](/wiki/Music_theory/Rhythm) ที่ซับซ้อน, สลับไปมา หรือไม่เป็นไปตามธรรมชาติ
- การใช้ความสัมพันธ์นี้จะช่วยสร้าง [รูปแบบ (Patterns)](/wiki/Beatmap/Pattern) ที่มีความสมดุลทางสายตา เช่น รูปทรงเรขาคณิตที่สม่ำเสมอ หรือส่วนของ [สตรีม (Stream)](/wiki/Beatmap/Pattern/osu!/Stream) ที่เป็นระเบียบ
- เทคนิคนี้ได้รับการแนะนำให้เป็นจุดเริ่มต้นสำหรับ Mapper มือใหม่ เพื่อช่วยให้พวกเขาพัฒนาสัมผัสเรื่องความสม่ำเสมอและจังหวะได้ดีขึ้น

## หมายเหตุ

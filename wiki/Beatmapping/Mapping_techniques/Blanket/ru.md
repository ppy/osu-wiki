# Бланкет

::: Infobox
![](img/blanket.png "Бланкет вокруг ноты, сделанный с помощью округлого слайдера")
:::

**Бланкет** (от англ. *blanket* — одеяло) — эффект, при котором какой-нибудь [объект](/wiki/Gameplay/Hit_object) «завёрнут» в [тело слайдера](/wiki/Gameplay/Hit_object/Slider/Sliderbody). Бланкеты используются для визуального оформления [карты](/wiki/Beatmap); о том, как их сделать самостоятельно, см. [Создание красивых слайдеров](/wiki/Beatmapping/Mapping_techniques/Making_good_sliders#beat-blankets).

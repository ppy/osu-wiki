# Blanket

::: Infobox
![](img/blanket.png "Blanket ของ hit object 2 โดยใช้ sliderbody ของ hit object 1")
:::

**Blanket** คือเอฟเฟกต์ที่เกิดขึ้นเมื่อ [sliderbody](/wiki/Gameplay/Hit_object/Slider/Sliderbody) โอบล้อมรูปร่างของ [hit object](/wiki/Gameplay/Hit_object) อื่น Blanketing ทำได้หลายวิธีเพื่อให้เข้ากับสไตล์ภาพของ[บีตแมป](/wiki/Beatmap) คู่มือการทำสามารถดูได้ใน[บทความ mapping techniques](/wiki/Beatmapping/Mapping_techniques/Making_good_sliders#beat-blankets)

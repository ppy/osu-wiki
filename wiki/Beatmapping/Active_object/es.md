# Objeto activo

El **objeto activo** es un término usado en [osu!catch](/wiki/Game_mode/osu!catch) para todos los objetos que deben ser atrapados para mantener el [combo](/wiki/Gameplay/Combo_(score_multiplier)), esto incluye [frutas](/wiki/Gameplay/Hit_object/Fruit) y [gotas](/wiki/Gameplay/Hit_object/Juice_stream#gota).

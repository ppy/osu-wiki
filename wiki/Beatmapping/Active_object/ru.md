# Активный объект

**Активный объект** — термин из [osu!catch](/wiki/Game_mode/osu!catch), обозначающий игровой объект, который нужно поймать для сохранения [комбо](/wiki/Gameplay/Combo_(score_multiplier)). Сюда относятся [фрукты](/wiki/Gameplay/Hit_object/Fruit) и [дропы](/wiki/Gameplay/Hit_object/Juice_stream#drop).

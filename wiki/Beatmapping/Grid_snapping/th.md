# Grid snapping

**Grid snapping** หมายถึงการใช้ฟังก์ชันปุ่ม `Grid Snap` ใน [beatmap editor](/wiki/Client/Beatmap_editor) เมื่อเปิดใช้งาน hit object ทุกตัวที่ผู้สร้างโต้ตอบด้วยจะถูกบังคับให้ขยับไปยังตำแหน่งที่ใกล้ที่สุดตามจุดตัดของกริด หรือพูดอีกอย่างคือถูก "snap" ปุ่ม `Grid Snap` อยู่ทางด้านขวาของ editor

ระดับพื้นฐานของระยะห่างระหว่างแต่ละช่องของกริดสามารถปรับได้ด้วยการกด `G` หรือไปที่ `View` -> `Grid Level` **Grid snapping** ช่วยจัดออบเจกต์ให้ตรงกัน และช่วยพยายามเรียงออบเจกต์เป็น patterns ที่เรียบร้อยขึ้น

# Skinning

Skinning เป็นหนึ่งในฟีเจอร์หลักของ osu! ที่ให้ผู้เล่นปรับหน้าตาอินเทอร์เฟซของเกมใหม่ได้จาก element เริ่มต้นที่เกมเตรียมไว้ สกินที่ผู้เล่นทำเองมีได้ตั้งแต่ธีมสนุก ๆ ไปจนถึงงานออกแบบที่ดูจริงจังแบบมืออาชีพ หรือบางสกินอาจตั้งใจทำให้เล่นยากมากจนแทบเล่นไม่ได้เลยก็มี

## Skinning sets

*หน้าหลัก: [Skin set list](/wiki/Ranking_criteria/Skin_set_list)*

**เฉพาะบีตแมปเท่านั้น** ถ้าสกินในบีตแมปของคุณมี element ใด element หนึ่งจาก set ที่ระบุไว้ใน [ranking criteria](/wiki/Ranking_criteria) สกินนั้นต้องมี element อื่น ๆ ทั้งหมดใน skinning set เดียวกันด้วย กฎนี้ใช้กับสกินของบีตแมปเท่านั้น แต่ skinner คนอื่น ๆ ก็ควรอิงรายการเหล่านี้ไว้ด้วย เพื่อรักษาความครบถ้วนของสกิน

# Создание скинов

Создание скинов - одна из ключевых особенностей osu!. Она позволяет игрокам использовать оригинальные элементы оформления для создания своих собственных скинов. Скины могут быть забавными, тематическими, для повседневной игры в osu! или же невозможными для игры.

## Наборы для создания скинов

*Главная страница: [Список элементов для скина](/wiki/Ranking_criteria/Skin_set_list)*

**Только для карт,** Если ваш скин для карты содержит элемент из наборов, указанных в критериях ранкинга, то он должен содержать остальные элементы из этого набора. Это относится только к скинам для карт, но остальные скины так же могут содержать этот список.

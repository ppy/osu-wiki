# Skinning

Skinning (atau pembuatan skin) merupakan salah satu fitur utama osu!. Fitur ini memungkinkan para pemain untuk mengembangkan skin buatan mereka sendiri berdasarkan elemen skin bawaan yang tersedia. Skin ini bisa bermacam-macam, mulai dari skin yang dibuat untuk bersenang-senang, tematik, profesional, atau bahkan yang mustahil untuk dimainkan sekalipun.

## Set skin

*Halaman utama: [Daftar set skin](/wiki/Ranking_criteria/Skin_set_list)*

**Khusus untuk beatmap,** apabila skin beatmap kamu mengandung salah satu elemen dari daftar set yang tertera pada [kriteria ranking](/wiki/Ranking_criteria), skin kamu harus mengikutsertakan seluruh elemen lainnya dari set ini. Peraturan ini hanya berlaku bagi skin beatmap, namun para skinner lainnya juga disarankan untuk mengikuti daftar ini demi menjaga keutuhan skin mereka.

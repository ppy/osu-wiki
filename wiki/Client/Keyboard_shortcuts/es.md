# Referencia de las teclas de acceso rápido

Hay muchos atajos en osu! que se pueden usar mientras el jugador está en el juego o inactivo en cualquier pantalla. Se enumeran con sus funciones a continuación.

*Nota: este artículo enumera los atajos de teclado predeterminados. Puedes cambiar algunos de ellos en las [opciones](/wiki/Client/Options).*

## En todo el juego

Estos atajos funcionan en cualquier lugar:

| Atajo | Acción |
| :-- | :-- |
| `Alt` + `F4` | Cierra el cliente de juego (sin avisos). Si estás en el juego, en el editor de beatmaps o en osu!direct, vuelve al selector de canciones o al menú principal (para osu!direct). |
| `Alt` + `Flecha arriba/abajo` | Ajusta el volumen principal, usa las flechas izquierda y derecha para ajustar los volúmenes de los efectos y de la música. |
| `Ctrl` + `Alt` + `Mayús` + `S` | Vuelve a cargar la skin actual. Si se presiona durante el juego o en la pantalla de resultados, esto se ejecutará después de regresar a la pantalla del selector de canciones. |
| `Ctrl` + `F11` | Alterna el gráfico de tiempo entre fotogramas. Presionar `Ctrl` lo pausará. |
| `F7` | Cambia los limitadores de tiempo entre fotogramas. |
| `F8` | Alterna la ventana del chat. |
| `F9` | Alterna la ventana del chat extendido. |
| `F10` | Alterna todos los botones del ratón (excepto la rueda del ratón, que siempre está habilitada). |
| `F12` | Toma una captura de pantalla. Hacer clic en el cuadro de diálogo abrirá la carpeta de las capturas de pantalla. |
| `Mayús` + `F12` | Toma una captura de pantalla y se sube al servidor para compartirla. La captura de pantalla también se guardará localmente. Esto abrirá la captura de pantalla en tu navegador. |
| `Insertar` | Esta es la [tecla jefe](https://en.wikipedia.org/wiki/Boss_key). Minimiza el cliente del juego en el área de notificaciones. Esto también silenciará el juego. |
| `Mayús` + `F1` | Abre [osu!status](https://twitter.com/osustatus) en tu navegador. (Solo cuando aparece un error de conexión). |
| `Ctrl` + `O` | Abre la barra lateral de opciones. No se puede ingresar al asistente de compensación si no estás en el menú principal. |

## Menú principal

| Atajo | Acción |
| :-- | :-- |
| `Alt` + `Entrar` | Alterna el modo de pantalla completa. |
| `Flecha arriba/abajo` | Ajusta el volumen principal, usa las flechas izquierda y derecha para ajustar los volúmenes de los efectos y de la música. |
| `Escape` | Volver atrás o cancelar. |
| `P`, `E`, o `O` | Abre el menú de juego. |
| `P` tres veces | Entra a Solo. |
| `P` dos veces, luego `M` | Entra a [Multi](/wiki/Client/Interface/Multiplayer). |
| `E` dos veces | Entra al modo edición. |
| `O` | Abre la barra lateral de opciones. |
| `Q` | Muestra el mensaje de salida. |
| `Escape` | Muestra el mensaje de salida cuando las opciones y el menú de juego están cerrados. |
| `F` | Alterna el contador de FPS. |

### Gramola

| Atajo | Acción |
| :-- | :-- |
| `Z` o `F1` | Reproduce la canción anterior. |
| `X` o `F2` | Reproduce la canción. Si se está reproduciendo, la canción vuelve a empezar. |
| `C` | Alterna entre la pausa y la reproducción. |
| `F4` | Detiene la reproducción (restablece el tiempo actual). |
| `V` o `F5` | Reproduce la siguiente canción. |
| `J` o `F6` | Abre el menú Saltar a. |
| `R` | Reproduce una canción aleatoria. |

## Selector de canciones

*Nota: No todos funcionan en la pantalla de selección de canciones de multijugador.*

| Atajo | Acción |
| :-- | :-- |
| `F1` | Abre el menú de selección de los [modificadores del juego](/wiki/Gameplay/Game_modifier). |
| `F2` | Selecciona aleatoriamente un beatmap. |
| `Mayús` + `F2` | Vuelve al beatmap previamente seleccionado aleatoriamente. |
| `F3` | Abre las opciones del beatmap. |
| `F5` | Vuelve a procesar los beatmaps (puede que pida confirmación para continuar). |
| `Mayús` + `Suprimir` | Abre el cuadro de diálogo para borrar el beatmap. Te preguntará si quieres borrar el beatmap (todas las dificultades), una sola dificultad o todos los beatmaps visibles. |
| `Ctrl` + `1` | Cambia el modo de juego a [osu!](/wiki/Game_mode/osu!). |
| `Ctrl` + `2` | Cambia el modo de juego a [osu!taiko](/wiki/Game_mode/osu!taiko). |
| `Ctrl` + `3` | Cambia el modo de juego a [osu!catch](/wiki/Game_mode/osu!catch). |
| `Ctrl` + `4` | Cambia el modo de juego a [osu!mania](/wiki/Game_mode/osu!mania). |
| `Ctrl` + `Entrar` | Juega el beatmap seleccionado con el [mod Auto](/wiki/Gameplay/Game_modifier/Auto). |
| `Ctrl` + `Mayús` + `Entrar` | Juega el beatmap seleccionado con el [mod Cinema](/wiki/Gameplay/Game_modifier/Cinema). |
| `Ctrl` + `A` | Alterna al mod Auto. |
| `Ctrl` + `F3` o `Ctrl` + `+` | Aumenta la velocidad de desplazamiento de osu!mania (mientras estás en el modo osu!mania). |
| `Ctrl` + `F4` o `Ctrl` + `-` | Disminuye la velocidad de desplazamiento de osu!mania (mientras estás en el modo osu!mania). |
| `Ctrl` + `Retroceso` | Elimina la última palabra en la barra de búsqueda. |
| `Flecha arriba/abajo` | Selecciona la siguiente dificultad arriba/abajo respectivamente. |
| `Flecha izquierda/derecha` | Selecciona el siguiente beatmap arriba/abajo respectivamente. |
| `Página arriba/abajo` | Desplaza la página hacia arriba/abajo respectivamente. |
| `Entrar` | Ingresa a la selección actual (si seleccionas sobre un beatmap o grupo de colección) o estás jugando una dificultad (si seleccionas sobre un beatmap). |
| `Mayús` + `Flecha izquierda/derecha` | Ingresa al grupo siguiente/anterior. |
| `Mayús` + `Entrar` | Expande o contrae el grupo seleccionado. |

## Modo de juego

| Atajo | Acción |
| :-- | :-- |
| `+` o `-` | Ajusta la compensación local (mantén presionado `Alt` para cambiar la precisión). |
| `Escape` | Pausa el juego o vuelve al selector de canciones si estás viendo una repetición o usando el mod Auto o Cinema. |
| `Barra espaciadora` | Saltar al juego. |
| `Tabulador` | Alterna el marcador. |
| `Mayús` + `Tabulador` | Alterna la visualización de la interfaz del juego. |
| `H` | Alterna la visualización del marcador y la interfaz de la repetición (viendo una repetición o siendo espectador). |
| `F3` o `F4` | Ajusta la velocidad para osu!mania (se puede usar cerca del comienzo del juego). |
| `Ctrl` + `R` o `` ` `` | Debe mantenerse pulsado brevemente. Reintenta rápidamente el mapa actual (se salta la introducción al reintentar). |

## Modo repetición

*Aviso: debes usar el canal de lanzamiento cuttingedge para usarlos.*

| Atajo | Acción |
| :-- | :-- |
| `5` | Pausa la repetición. |
| `6` o `0` | Reanuda o establece la velocidad de reproducción en 1x. |
| `7` | Establece la velocidad de reproducción en 2x. |
| `8` | Establece la velocidad de reproducción en 3x. |
| `9` | Establece la velocidad de reproducción en 4x. |

## Editor de beatmaps

*Página principal: [Editor de beatmaps](/wiki/Client/Beatmap_editor)*

### General

Estos atajos funcionan en cualquier lugar dentro del editor de beatmaps:

| Atajo | Acción |
| :-- | :-- |
| `Z` | Salta a la primera nota o al inicio de la canción. |
| `V` | Salta a la última nota o al final de la canción. |
| `X` | Reproduce la canción desde el principio. |
| `C` o `Barra espaciadora` | Pausar o reanudar. |
| `Flecha izquierda` o `Flecha derecha` | Salta a través de la canción. |
| `Flecha arriba` o `Flecha abajo` | Salta entre los puntos de tiempo. |
| `Mayús` + `Flecha izquierda` o `Mayús` + `Flecha derecha` | Salta a través de la canción a una velocidad de 4x. |
| `Ctrl` + `B` | Agrega un marcador en la marca de tiempo actual. |
| `Ctrl` + `Mayús` + `B` | Elimina el marcador en la marca de tiempo actual. |
| `Ctrl` + `Flecha izquierda/derecha` | Salta al marcador anterior/siguiente. |
| `Ctrl` + `L` | Vuelve al último estado guardado (parcial). Esto volverá a cargar los archivos `.osu` y `.osb`. |
| `Ctrl` + `Mayús` + `L` | Vuelve al último estado guardado (completo). Esto volverá a cargar todo. |
| `Ctrl` + `O` | Abre la ventana de selección de dificultades. |
| `Ctrl` + `S` | Guarda el beatmap en su estado actual. |
| `Ctrl` + `N` | Pregunta si quieres eliminar todos los objetos. |
| `Ctrl` + `Flecha arriba/abajo` | Aumenta/disminuye la velocidad de reproducción en un 25 %. |
| `Ctrl` + `Mayús` + `Flecha arriba/abajo` | Aumenta/disminuye la velocidad de reproducción en un 5 %. |
| `F1` | Cambia a la pestaña [compose](/wiki/Client/Beatmap_editor/Compose). |
| `F2` | Cambia a la pestaña [design](/wiki/Client/Beatmap_editor/Design). |
| `F3` | Cambia a la pestaña [timing](/wiki/Client/Beatmap_editor/Timing). |
| `F4` | Abre la ventana de [configuración de la canción](/wiki/Client/Beatmap_editor/Song_setup). |
| `F5` | Entra al modo de prueba. |
| `F6` | Abre la ventana de puntos de tiempo y control. |
| `Ctrl` + `P` | Agrega un punto de tiempo no heredado en la marca de tiempo actual. Si no estás en la pestaña timing, esto también abrirá la ventana de puntos de tiempo y control. |
| `Ctrl` + `Mayús` + `P` | Agrega un punto de tiempo heredado en la marca de tiempo actual. Esto también abrirá la ventana de puntos de tiempo y control. |
| `Ctrl` + `I` | Elimina la sección actual heredada o no heredada. Esto no funcionará en el primer punto de tiempo no heredado. |

#### Ratón

| Atajo | Acción |
| :-- | :-- |
| `Rueda del ratón hacia arriba/abajo` | Busca hacia adelante/atrás. |
| `Mayús` + `Rueda del ratón hacia arriba/abajo` | Busca hacia adelante/atrás a 4x. |
| `Ctrl` + `Rueda del ratón hacia arriba/abajo` | Ajusta el divisor de ritmo. |
| `Alt` + `Rueda del ratón hacia arriba/abajo` | Ajusta el multiplicador de distancia de las notas al pasar el cursor sobre el campo de juego. Ajusta el zoom de la línea de tiempo al pasar el cursor sobre la línea de tiempo. |
| `Alt` + `Mayús` + `Rueda del ratón hacia arriba/abajo` | Ajusta con mayor precisión el multiplicador de distancia de las notas al pasar el cursor sobre el campo de juego. |

### Compose

#### General

| Atajo | Acción |
| :-- | :-- |
| `Alt` + `F4` o `Escape` | Vuelve al selector de canciones. |
| `Ctrl` + `Z` | Deshacer. |
| `Ctrl` + `Y` | Rehacer. |
| `Ctrl` + `A` | Selecciona todos los objetos. |
| `Ctrl` + `X` | Cortar. |
| `Ctrl` + `C` | Copiar. |
| `Ctrl` + `V` | Pegar. |
| `Ctrl` + `D` | Clona la selección. Esto pegará la selección 1 compás después del último objeto seleccionado. |
| `Ctrl` + `M` | Ajusta el divisor de ritmo. |
| `Suprimir` | Borrar. |
| `1`, `2`, `3` o `4` | Cambia entre el modo de colocación/selección: select, circle, slider y spinner respectivamente (osu!, osu!taiko u osu!catch). |
| `1`, `2` o `3` | Cambia entre el modo de colocación/selección: select, circle y hold respectivamente (osu!mania). |
| `Ctrl` + `Alt` + `Rueda del ratón hacia arriba/abajo` | Cambia entre los modos de colocación/selección. |
| `Ctrl` + `Mayús` + `A` | Abre [AiMod](/wiki/Client/Beatmap_editor/AiMod). |

#### Ratón

| Atajo | Acción |
| :-- | :-- |
| `Doble clic` | En círculos o slider ends para saltar a la posición del objeto en la línea de tiempo desde cualquier punto de la canción. (osu!, osu!taiko, osu!catch) |
| `Clic derecho` | Alterna los nuevos combos (al colocar nuevos objetos) o elimina objetos (al seleccionar). |

#### Campo de juego

| Atajo | Acción |
| :-- | :-- |
| `Alt` | Alterna temporalmente el [ajuste de distancia](/wiki/Client/Beatmap_editor/Distance_snap) mientras se mantiene presionado. |
| `Y` | Alterna el ajuste de distancia. |
| `Mayús` | Alterna temporalmente el ajuste a la cuadrícula mientras se mantiene presionado. |
| `T` | Alterna el ajuste a la cuadrícula. |
| `G` | Cambia el tamaño de la cuadrícula. |
| `Ctrl` + `1` | Ajusta el tamaño de la cuadrícula a diminuto. |
| `Ctrl` + `2` | Ajusta el tamaño de la cuadrícula a pequeño. |
| `Ctrl` + `3` | Ajusta el tamaño de la cuadrícula a medio. |
| `Ctrl` + `4` | Ajusta el tamaño de la cuadrícula a grande. |
| `L` | Alterna el bloqueo de objetos. |

#### Manipulación de objetos

| Atajo | Acción |
| :-- | :-- |
| `Q` | Alterna un nuevo combo. |
| `Ctrl` + `,` | Gira la selección 90 grados en el sentido contrario a las agujas del reloj. |
| `Ctrl` + `.` | Gira la selección 90 grados en el sentido de las agujas del reloj. |
| `Ctrl` + `H` | Invierte la selección horizontalmente. |
| `Ctrl` + `J` | Invierte la selección verticalmente. |
| `Ctrl` + `G` | Invierte la selección. Es decir, mueve el primer objeto seleccionado al último objeto seleccionado y así sucesivamente. |
| `Ctrl` + `Mayús` + `R` | Gira los objetos seleccionados un ángulo especificado (abre un cuadro de diálogo). |
| `Ctrl` + `Mayús` + `S` | Escala los objetos seleccionados en una cantidad especificada (abre un cuadro de diálogo). |
| `Ctrl` + `Mayús` + `D` | Genera polígonos con un número especificado de puntos (abre un cuadro de diálogo). |
| `Ctrl` + `Mayús` + `F` | Convierte el slider seleccionado en un stream de círculos (abre un cuadro de diálogo). |
| `Ctrl` + `Flecha izquierda/derecha/arriba/abajo` | Mueve las notas seleccionadas en la dirección pulsada. (osu!, osu!taiko, osu!catch). |
| `Ctrl` + `Flecha izquierda/derecha` | Mueve los objetos seleccionados una columna a la izquierda o a la derecha. (osu!mania). |
| `Ctrl` + `Flecha arriba/abajo` | Mueve los objetos seleccionados 1 divisor hacia arriba o hacia abajo. (osu!mania). |

#### Hitsounding

| Atajo | Acción |
| :-- | :-- |
| `W` | Alterna la adición de un Whistle a la selección. |
| `E` | Alterna la adición de un Finish a la selección. |
| `R` | Alterna la adición de un Clap a la selección. |
| `Mayús` + (`Q`, `W`, `E` o `R`) | Cambia el conjunto de muestras de la selección a auto, normal, soft o drum, respectivamente. |
| `Ctrl` + (`Q`, `W`, `E` o `R`) | Cambia el conjunto de adiciones de la selección a auto, normal, soft o drum, respectivamente. |
| `Ctrl` + `Mayús` + `I` | Abre el cuadro de diálogo de importación de muestras. |

#### Línea de tiempo

| Atajo | Acción |
| :-- | :-- |
| `Mayús` + (`1`, `2`, `3`, `4`, `5,`, `6`, `7`, `8` o `9`) | Ajusta el [divisor de ritmo](/wiki/Client/Beatmap_editor/Beat_snap_divisor) a 1/1, 1/2, 1/3, 1/4, 1/5, 1/6, 1/7, 1/8 y 1/9 respectivamente. |
| `Mayús` (mientras arrastras en la línea de tiempo) | Ignora el ajuste del divisor de ritmo. |
| `J` | Mueve la selección 1 divisor hacia atrás. |
| `K` | Mueve la selección 1 divisor hacia adelante. |

### Design

| Atajo | Acción |
| :-- | :-- |
| `I` | Abre la superposición de la biblioteca de sprites. |
| `Ctrl` + `Z` | Deshacer. |
| `Ctrl` + `Y` | Rehacer. |
| `Ctrl` + `X` | Corta el sprite seleccionado. |
| `Ctrl` + `C` | Copia el sprite seleccionado. |
| `Ctrl` + `V` | Pega el sprite en el tiempo actual. *Nota: Los sprites pegados no se mostrarán en la lista de capas hasta que se guarde y se vuelva a cargar el beatmap.* |
| `Suprimir` | Elimina el sprite seleccionado de la capa. |
| `W`, `A`, `S`, `D` | Mueve el objeto seleccionado hacia arriba, izquierda, abajo o derecha respectivamente un [osu!pixel](/wiki/Client/Beatmap_editor/osu!_pixel). |

### Timing

| Atajo | Acción |
| :-- | :-- |
| `T` | Establece el BPM y la compensación cuando se pulsa al ritmo de la canción. |
| `Mayús` mientras ajustas el BPM o la compensación | Incrementa 5 veces la cantidad normal. |
| `Ctrl` mientras ajustas el BPM | Incrementa en 0,25 |
| `Ctrl` mientras ajustas la compensación | Incrementa en 1. |
| `Ctrl` mientras ajustas la velocidad del slider | Incrementa en 0,1. |

# ปุ่มลัด (Shortcut key reference)

ใน osu! มีปุ่มลัดมากมายที่สามารถใช้งานได้ทั้งในระหว่างการเล่นหรือในขณะที่อยู่ที่หน้าจอต่างๆ รายการด้านล่างคือรวบรวมปุ่มลัดและหน้าที่ของมัน

*หมายเหตุ: บทความนี้ระบุปุ่มลัดพื้นฐาน (Default) คุณสามารถเปลี่ยนปุ่มบางส่วนได้ในเมนู [การตั้งค่า (Options)](/wiki/Client/Options)*

## ทั่วทั้งเกม (Game-wide)

ปุ่มลัดเหล่านี้ใช้งานได้ทุกที่ภายในเกม:

| ปุ่มลัด | การทำงาน |
| :-- | :-- |
| `Alt` + `F4` | ปิดโปรแกรมเกมทันที (ไม่มีหน้าต่างยืนยัน) หากอยู่ในโหมดการเล่น, ตัวแก้ไข หรือ osu!direct ระบบจะพาคุณกลับไปยังหน้าเลือกเพลงหรือเมนูหลักแทน |
| `Alt` + `ลูกศรขึ้น/ลง` | ปรับระดับเสียงหลัก (Master volume), ใช้ลูกศรซ้ายและขวาเพื่อปรับระดับเสียงเอฟเฟกต์และเสียงดนตรี |
| `Ctrl` + `Alt` + `Shift` + `S` | โหลด Skin ปัจจุบันใหม่อีกครั้ง หากกดในขณะเล่นหรือหน้าสรุปผล คำสั่งจะมีผลเมื่อกลับสู่หน้าเลือกเพลง |
| `Ctrl` + `F11` | เปิด/ปิด กราฟแสดงเวลาเฟรม (Frame times graph) กด `Ctrl` เพื่อหยุดกราฟไว้ชั่วคราว |
| `F7` | วนรอบตัวจำกัดเฟรมเรต (Frame rate limiters) |
| `F8` | เปิด/ปิด หน้าต่างแชท |
| `F9` | เปิด/ปิด หน้าต่างแชทแบบขยาย |
| `F10` | เปิด/ปิด การใช้งานปุ่มเมาส์ทั้งหมด (ยกเว้นล้อเมาส์ที่เปิดใช้งานเสมอ) |
| `F12` | ถ่ายภาพหน้าจอ (Screenshot) คลิกที่แถบแจ้งเตือนเพื่อเปิดโฟลเดอร์เก็บรูปภาพ |
| `Shift` + `F12` | ถ่ายภาพหน้าจอและอัปโหลดไปยังเซิร์ฟเวอร์เพื่อแบ่งปัน โดยภาพจะถูกบันทึกลงในเครื่องด้วย และระบบจะเปิดเบราว์เซอร์เพื่อแสดงภาพนั้น |
| `Insert` | นี่คือ [ปุ่มบอส (Boss key)](https://en.wikipedia.org/wiki/Boss_key) ใช้สำหรับย่อหน้าต่างเกมลงไปที่แถบแจ้งเตือน (System tray) และปิดเสียงเกมทันที |
| `Shift` + `F1` | เปิดหน้า [osu!status](https://twitter.com/osustatus) บนเบราว์เซอร์ (ใช้งานได้เฉพาะเมื่อเกิดข้อผิดพลาดในการเชื่อมต่อ) |
| `Ctrl` + `O` | เปิดเมนูการตั้งค่า (Options) จากด้านข้าง คุณจะไม่สามารถเข้า [Offset wizard](/wiki/Client/Options/Offset_Wizard) ได้หากไม่ได้อยู่ที่เมนูหลัก |

## เมนูหลัก (Main menu)

| ปุ่มลัด | การทำงาน |
| :-- | :-- |
| `Alt` + `Enter` | สลับโหมดเต็มหน้าจอ (Fullscreen) |
| `ลูกศรขึ้น/ลง` | ปรับระดับเสียงหลัก |
| `Esc` | ย้อนกลับหรือยกเลิก |
| `P`, `E` หรือ `O` | เปิดเมนู Play, Edit หรือ Options ตามลำดับ |
| `P` สามครั้ง | เข้าสู่โหมดเล่นคนเดียว (Solo) |
| `P` สองครั้ง ตามด้วย `M` | เข้าสู่โหมด [มัลติเพลเยอร์ (Multi)](/wiki/Client/Interface/Multiplayer) |
| `E` สองครั้ง | เข้าสู่โหมดตัวแก้ไข (Edit) |
| `O` | เปิดเมนูการตั้งค่า (Options) |
| `Q` | แสดงหน้าต่างยืนยันการออกจากเกม |
| `Esc` | แสดงหน้าต่างยืนยันการออกจากเกม (เมื่อปิดเมนูตั้งค่าและเมนูเล่นแล้ว) |
| `F` | เปิด/ปิด ตัวนับ FPS |

### เครื่องเล่นเพลง (Jukebox)

| ปุ่มลัด | การทำงาน |
| :-- | :-- |
| `Z` หรือ `F1` | เล่นเพลงก่อนหน้า |
| `X` หรือ `F2` | เล่นเพลงปัจจุบันตั้งแต่เริ่มใหม่ |
| `C` | สลับระหว่างเล่นและหยุดเพลงชั่วคราว |
| `F4` | หยุดเล่นเพลง (รีเซ็ตตำแหน่งเวลาปัจจุบัน) |
| `V` หรือ `F5` | เล่นเพลงถัดไป |
| `J` หรือ `F6` | เปิดเมนูสำหรับข้ามไปยังเพลงที่เลือก |
| `R` | เล่นเพลงแบบสุ่ม |

## หน้าเลือกเพลง (Song select)

*หมายเหตุ: ปุ่มบางปุ่มอาจใช้งานไม่ได้ในหน้าเลือกเพลงของโหมดมัลติเพลเยอร์*

| ปุ่มลัด | การทำงาน |
| :-- | :-- |
| `F1` | เปิดเมนูเลือก [Mod (Game modifiers)](/wiki/Gameplay/Game_modifier) |
| `F2` | สุ่มเลือก Beatmap |
| `Shift` + `F2` | ย้อนกลับไปยัง Beatmap ที่ถูกสุ่มเลือกก่อนหน้า |
| `F3` | เปิดเมนูตัวเลือกของ Beatmap |
| `F5` | ประมวลผล Beatmap ใหม่ (อาจมีหน้าต่างให้ยืนยัน) |
| `Shift` + `Delete` | แสดงหน้าต่างลบ Beatmap ซึ่งจะถามว่าต้องการลบทั้งชุดแมพ, เฉพาะความยากนี้ หรือลบทุกแมพที่มองเห็น |
| `Ctrl` + `1` | สลับโหมดเกมเป็น [osu!](/wiki/Game_mode/osu!) |
| `Ctrl` + `2` | สลับโหมดเกมเป็น [osu!taiko](/wiki/Game_mode/osu!taiko) |
| `Ctrl` + `3` | สลับโหมดเกมเป็น [osu!catch](/wiki/Game_mode/osu!catch) |
| `Ctrl` + `4` | สลับโหมดเกมเป็น [osu!mania](/wiki/Game_mode/osu!mania) |
| `Ctrl` + `Enter` | เล่นแมพที่เลือกด้วย Mod [Auto](/wiki/Gameplay/Game_modifier/Auto) |
| `Ctrl` + `Shift` + `Enter` | เล่นแมพที่เลือกด้วย Mod [Cinema](/wiki/Gameplay/Game_modifier/Cinema) |
| `Ctrl` + `A` | เปิด/ปิด Mod Auto |
| `Ctrl` + `F3` หรือ `Ctrl` + `+` | เพิ่มความเร็วการเลื่อน (Scroll speed) ในโหมด osu!mania |
| `Ctrl` + `F4` หรือ `Ctrl` + `-` | ลดความเร็วการเลื่อน (Scroll speed) ในโหมด osu!mania |
| `Ctrl` + `Backspace` | ลบคำสุดท้ายในช่องค้นหา |
| `ลูกศรขึ้น/ลง` | เลือกความยากถัดไปในทิศทางนั้น |
| `ลูกศรซ้าย/ขวา` | เลือกชุดแมพถัดไปในทิศทางนั้น |
| `Page Up/Down` | เลื่อนหน้าจอขึ้น/ลงตามลำดับ |
| `Enter` | ตกลงในส่วนที่เลือกอยู่ หรือเริ่มเล่นระดับความยากนั้น |
| `Shift` + `ลูกศรซ้าย/ขวา` | เข้าสู่กลุ่มถัดไป/ก่อนหน้า |
| `Shift` + `Enter` | ขยายหรือยุบกลุ่มแมพที่เลือกอยู่ |

## โหมดการเล่น (Play mode)

| ปุ่มลัด | การทำงาน |
| :-- | :-- |
| `+` หรือ `-` | ปรับค่า Local offset (กด `Alt` ค้างไว้เพื่อปรับแบบละเอียด) |
| `Esc` | หยุดเกมชั่วคราว หรือกลับสู่หน้าเลือกเพลงหากกำลังดู Replay หรือใช้ Mod Auto/Cinema |
| `Space` | ข้าม (Skip) ท่อนนำเพลงเข้าสู่ช่วงเล่น |
| `Tab` | เปิด/ปิด ตารางคะแนนระหว่างเล่น |
| `Shift` + `Tab` | เปิด/ปิด การแสดงผลอินเทอร์เฟซระหว่างเล่น |
| `H` | เปิด/ปิด ตารางคะแนนและอินเทอร์เฟซ Replay (เมื่อดู Replay หรือรับชมผู้อื่น) |
| `F3` หรือ `F4` | ปรับความเร็วในโหมด osu!mania (ใช้ได้ในช่วงเริ่มต้นแมพ) |
| `Ctrl` + `R` หรือ `` ` `` | ต้องกดค้างไว้ครู่หนึ่ง เพื่อเริ่มเล่นแมพใหม่ทันที (ข้ามท่อนนำเพลง) |

## โหมด Replay

*ประกาศ: คุณต้องใช้สายการพัฒนาแบบ Cutting Edge เพื่อใช้ฟังก์ชันเหล่านี้*

| ปุ่มลัด | การทำงาน |
| :-- | :-- |
| `5` | หยุด Replay ชั่วคราว |
| `6` หรือ `0` | เล่นต่อหรือตั้งความเร็วเป็น 1x |
| `7` | ตั้งความเร็วเป็น 2x |
| `8` | ตั้งความเร็วเป็น 3x |
| `9` | ตั้งความเร็วเป็น 4x |

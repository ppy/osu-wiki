# การติดตั้งบน macOS (Installation on macOS)

หน้านี้จะอธิบายขั้นตอนเบื้องต้นในการติดตั้งและใช้งาน osu! บนอุปกรณ์ macOS ของคุณ

## ความต้องการขั้นต่ำของระบบ

- เครื่องคอมพิวเตอร์ที่ทำงานด้วยระบบปฏิบัติการ macOS ที่มีประสิทธิภาพเพียงพอ

## วิธีการติดตั้ง osu!

ตัวเกม osu! เวอร์ชัน macOS อาจมีบัคด้านการแสดงผลเล็กน้อยและมีประสิทธิภาพการเล่นที่ต่ำกว่าเวอร์ชัน Windows ตัวเกมอาจทำงานได้ไม่สมบูรณ์แบบในช่วงแรกและต้องใช้เวลาในการปรับจูนเล็กน้อยเพื่อให้คุ้นเคย ในการติดตั้งให้ทำตามขั้นตอนดังนี้:

1. ไปที่ [หน้าดาวน์โหลด Wineskin (แบบไม่เป็นทางการ)](https://osu.ppy.sh/community/forums/topics/1106057) เพื่อดาวน์โหลด Wineskin เวอร์ชันล่าสุดและทำการแตกไฟล์
2. ดาวน์โหลดเครื่องมือ [osu!macOS Agent (แบบไม่เป็นทางการ)](https://osu.ppy.sh/community/forums/topics/1036678) และใช้เพื่อซ่อมแซมไฟล์ `osu!.app` หรือทำตามวิธีสำรองดังนี้:
   1. ย้ายไฟล์ `osu!.app` ไปไว้ที่หน้า Desktop
   2. เปิดโปรแกรม Terminal คัดลอกคำสั่งนี้: `xattr -c 'Desktop/osu!.app'` แล้วกดปุ่ม `Return` (หรือ Enter)
3. ดับเบิลคลิกที่ `osu!.app` เพื่อเปิดโปรแกรม
4. ตัวติดตั้งจะทำงานโดยอัตโนมัติ ตำแหน่งการติดตั้งจะไม่สามารถเปลี่ยนได้และจะถูกเก็บไว้ภายในไฟล์ `osu!.app` โดยตรง การย้ายไฟล์ `osu!.app` จะเป็นการย้ายไฟล์ทั้งหมดที่มากับตัวเกมรวมถึง [Beatmap](/wiki/Beatmap) และ [Skin](/wiki/Skin) ของคุณด้วย คุณสามารถคลิกขวาที่ไฟล์แล้วเลือก `Show Package Contents` เพื่อดูไฟล์ที่อยู่ภายในโฟลเดอร์ได้
5. เมื่อติดตั้งเสร็จสิ้น osu! จะเริ่มทำงานทันที โดยระบบจะใช้ osu!direct เพื่อดาวน์โหลด Beatmap เริ่มต้นมาให้คุณ
6. ตัวเกมจะขอให้คุณลงชื่อเข้าใช้ (Sign in) หรือลงทะเบียน (Register)
   - หากคุณมีบัญชีอยู่แล้ว ให้ลงชื่อเข้าใช้ได้ทันที
   - หากต้องการสมัครใหม่ โปรดดูที่หน้า [การลงทะเบียน (Registration)](/wiki/Registration) จากนั้นจึงค่อยลงชื่อเข้าใช้

## การแก้ไขปัญหา

หากคุณประสบปัญหาในการเปิด osu! ขั้นตอนต่อไปนี้อาจช่วยคุณได้:

1. ตรวจสอบรายงานที่สร้างโดย *osu!macOS Agent* เพื่อดูว่ามีข้อผิดพลาดใดๆ หรือไม่
   - หากคุณใช้ macOS Catalina เวอร์ชันเก่า (10.15.4 หรือเก่ากว่า) คุณจำเป็นต้องอัปเดต macOS (แนะนำ) หรือทำการปิดใช้งาน [System Integrity Protection](https://developer.apple.com/documentation/security/disabling_and_enabling_system_integrity_protection)
2. ใช้โปรแกรมแตกไฟล์ "Archive Utility" ที่มาพร้อมกับ macOS เนื่องจากโปรแกรมอื่นอาจแตกไฟล์ Wineskin ไม่ถูกต้อง
3. หากคุณมีการติดตั้งโปรแกรมป้องกันไวรัส ตรวจสอบให้แน่ใจว่าได้ปิดใช้งานก่อนแตกไฟล์ osu! มิฉะนั้นโปรแกรมอาจกักกันไฟล์ที่จำเป็นสำหรับ Wine
4. หากยังไม่สามารถแก้ไขได้ คุณสามารถโพสต์ถามปัญหาได้ใน [ฟอรัมช่วยเหลือ (Help forum)](https://osu.ppy.sh/community/forums/5)

## osu!(lazer)

[osu!(lazer)](/wiki/Client/Release_stream/Lazer) คือตัวเกมเวอร์ชันถัดไปของ osu! ที่กำลังอยู่ในระหว่างการพัฒนา ในการติดตั้งให้ทำตามขั้นตอนดังนี้:

1. ไปที่ [หน้าดาวน์โหลด osu!(lazer)](https://github.com/ppy/osu/releases/latest) ดาวน์โหลดไฟล์ `osu!.app.zip` และทำการแตกไฟล์
2. ดับเบิลคลิกที่ไฟล์ osu! แล้วกดปุ่ม `Open` เพื่อเปิดโปรแกรม
3. หากมีหน้าต่างขึ้นมาถามเพื่อขอสิทธิ์การเข้าถึงการกดปุ่ม (Keystrokes):
   1. คลิกที่ปุ่ม `Open System Preferences`
   2. คลิกที่รูปแม่กุญแจที่มุมซ้ายล่างแล้วใส่รหัสผ่านเครื่องของคุณ
   3. ติ๊กถูกที่ช่องสี่เหลี่ยมข้างไอคอน osu!

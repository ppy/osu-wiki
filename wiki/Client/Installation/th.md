# การติดตั้ง (Installation)

*ดูเพิ่มเติม: [การติดตั้งบน macOS](/wiki/Client/Installation/macOS)*

หน้านี้จะอธิบายวิธีการติดตั้งและใช้งาน osu! บนอุปกรณ์ Windows ของคุณ นอกจากนี้ยังมี [วิดีโอสอนการใช้งาน](https://www.youtube.com/watch?v=0V5GwzmMhpU) จากซีรีส์ [osu!academy](/wiki/Community/Video_series/osu!academy) ที่อธิบายวิธีการติดตั้งไว้อย่างละเอียด

## ความต้องการขั้นต่ำของระบบ

- .NET Framework 4 ขึ้นไป (ตัวติดตั้งจะแจ้งให้ดาวน์โหลดหากไม่มีในเครื่อง หรือสามารถ [หาดาวน์โหลดได้ที่นี่](https://www.microsoft.com/en-us/download/details.aspx?id=48130))
- การ์ดจอที่รองรับการทำงานในระดับพื้นฐานได้ดี

## วิธีการติดตั้ง osu!

1. ไปที่ [หน้าดาวน์โหลด](https://osu.ppy.sh/home/download) และดาวน์โหลดตัวติดตั้ง
2. ค้นหาไฟล์ที่ดาวน์โหลดมาแล้วรันโปรแกรม
3. ในขณะที่ตัวติดตั้งกำลังทำงาน จะมีช่วงเวลา 10 วินาทีให้คุณเลือกสถานที่ติดตั้ง
   - หากต้องการเปลี่ยนตำแหน่ง ให้คลิกที่ข้อความใต้คำว่า `Hi!` และเลือกโฟลเดอร์ที่ต้องการ
   - osu! จะถูกติดตั้งในตำแหน่งนั้นเมื่อเวลานับถอยหลังสิ้นสุดลง
4. เมื่อการติดตั้งเสร็จสมบูรณ์ osu! จะเริ่มทำงานโดยอัตโนมัติ และจะมีการใช้ osu!direct เพื่อดาวน์โหลด Beatmap เริ่มต้นมาให้คุณ
5. ตัวเกมจะขอให้คุณลงชื่อเข้าใช้ (Sign in) หรือลงทะเบียน (Register)
   - หากคุณมีบัญชีอยู่แล้ว ให้ลงชื่อเข้าใช้ได้ทันที
   - หากต้องการสมัครใหม่ โปรดดูที่หน้า [การลงทะเบียน (Registration)](/wiki/Registration)

## การแก้ไขปัญหา

หากคุณใช้ Windows 7 แบบ 64 บิต คุณอาจพบปัญหาในการติดตั้ง แนะนำให้ลองติดตั้ง .NET Framework เวอร์ชัน 64 บิตแยกต่างหากหากตัวติดตั้งไม่ได้เตรียมไว้ให้ โดยสามารถ [ดาวน์โหลดได้ที่นี่](https://download.microsoft.com/download/2/0/e/20e90413-712f-438c-988e-fdaa79a8ac3d/dotnetfx35.exe)

---

**หากตัวติดตั้งเกิดข้อผิดพลาดในระหว่างการติดตั้ง คุณสามารถลองทำตามขั้นตอนต่อไปนี้:**

- ลองติดตั้ง .NET Framework เวอร์ชันล่าสุดใหม่อีกครั้ง โดย [ดาวน์โหลดได้ที่นี่](https://dotnet.microsoft.com/download)
- ทำความสะอาดระบบคอมพิวเตอร์ของคุณ สแกนหาไวรัส และทำความสะอาด Registry (โปรแกรม CCleaner เป็นตัวเลือกที่ดีสำหรับการล้างไฟล์ขยะและ Registry) การติดตั้งอาจล้มเหลวหากมีข้อมูล Registry หลงเหลือจากการติดตั้ง osu! ครั้งก่อนที่ผิดพลาด

## การเพิ่ม Beatmap

*ดูเพิ่มเติม: [Beatmap](/wiki/Beatmap)*

มีสองวิธีในการเพิ่ม Beatmap เข้าสู่ตัวเกม คือผ่านทางเว็บไซต์ หรือผ่าน osu!direct (เฉพาะผู้ที่มี [osu!supporter](/wiki/osu!supporter))

### ผ่านทางเว็บไซต์

1. ไปที่หน้า [รายการ Beatmap (Beatmap listing)](https://osu.ppy.sh/beatmapsets) และลงชื่อเข้าใช้เว็บไซต์
2. ค้นหาเพลงที่คุณต้องการเล่น
3. คลิกที่ไอคอนดาวน์โหลดเพื่อบันทึก Beatmap ลงในคอมพิวเตอร์ของคุณ
4. ค้นหาและเปิดไฟล์ Beatmap ที่ดาวน์โหลดมา (ไฟล์นามสกุล `.osz`)
   - หรือคุณสามารถย้ายไฟล์ `.osz` ไปไว้ในโฟลเดอร์ `Songs` ภายในไดเรกทอรีที่ติดตั้ง osu! ได้โดยตรง
5. osu! จะจัดการส่วนที่เหลือและโหลด Beatmap เข้าสู่เกมให้โดยอัตโนมัติ
   - หากไม่เห็นเพลงในเกม ให้กด `F5` ใน [หน้าเลือกเพลง (Song Selection)](/wiki/Client/Interface#song-select) เพื่อให้ระบบประมวลผลข้อมูลใหม่

### osu!direct

*หมายเหตุ: คุณจำเป็นต้องมีสถานะ osu!supporter และการเชื่อมต่ออินเทอร์เน็ตเพื่อใช้งาน osu!direct*

1. เปิด osu! แล้วคลิกปุ่ม osu!direct ที่อยู่ทางด้านขวาของเมนูหลัก
2. ค้นหาเพลงที่คุณต้องการเล่น
3. ดับเบิลคลิกที่ชื่อเพลงในรายการ หรือคลิกปุ่ม `Download` ทางด้านขวา
   - หากไม่ต้องการดาวน์โหลดวิดีโอประกอบ ให้คลิกปุ่ม `DL NoVideo` แทน
4. เมื่อดาวน์โหลดเสร็จสิ้น ตัวเกมจะโหลด Beatmap เข้าสู่เกมให้โดยอัตโนมัติ
   - หากไม่เห็นเพลงในเกม ให้กด `F5` ในหน้าเลือกเพลงเพื่อรีเซ็ตข้อมูล

## การเพิ่ม Skin

*ดูเพิ่มเติม: [การทำ Skin (Skinning)](/wiki/Skinning)*

1. ไปที่ [ฟอรัมเกี่ยวกับ Skin](https://osu.ppy.sh/community/forums/15) และเลือก Skin ที่คุณชอบ
2. ทำการดาวน์โหลดไฟล์ Skin นั้นมา
3. วิธีการนำเข้าจะขึ้นอยู่กับรูปแบบของไฟล์ที่ได้รับ:
   - **หากเป็นไฟล์นามสกุล `.osk`:**
     1. เปิดไฟล์นั้น หรือลากไปวางในหน้าต่างตัวเกม osu!
     2. ตัวเกมจะประมวลผลและเลือกใช้งาน Skin นั้นให้คุณโดยอัตโนมัติ
   - **หากเป็นไฟล์บีบอัด (เช่น `.7z`, `.rar`, `.zip`):**
     1. ทำการแตกไฟล์ (Extract) ออกมา
     2. ย้ายโฟลเดอร์ Skin ที่แตกออกมาไปไว้ในโฟลเดอร์ `Skins` ภายในไดเรกทอรีที่ติดตั้ง osu!
     3. หากตัวเกมเปิดอยู่ Skin จะปรากฏในรายการหลังจากที่คุณกดปุ่ม `Ctrl` + `Alt` + `Shift` + `S`
     4. เข้าไปที่เมนู [ตัวเลือกและการตั้งค่า (Options)](/wiki/Client/Options) และค้นหาคำว่า `Skin`
     5. เลื่อนลงมาที่หัวข้อ `Current skin:` และเลือก Skin ที่คุณต้องการใช้งาน

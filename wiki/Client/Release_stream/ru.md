# Ветка обновления

osu! имеет несколько **веток обновлений**, или версий игровых клиентов, которые получают обновления с разной скоростью. Чтобы своевременно выявлять потенциальные проблемы, новые функции и изменения добавляются в предварительные версии, прежде чем они появятся в стабильном клиенте.

| Ветка обновления | Заметка |
| :-- | :-- |
| Stable | Текущая, более популярная версия; [не обновляется по функционалу](https://en.wikipedia.org/wiki/Freeze_(software_engineering)) |
| Cutting Edge | Содержит пару отличий от stable; не обновляется по функционалу |
| Tachyon | Предрелизная и бета-версия osu!(lazer); обычно содержит функционал, который пока не полностью готов |
| [Lazer](Lazer) | Переписанный игровой клиент; активно разрабатывается и содержит [множество изменений и новых функций](/wiki/Client/Release_stream/Lazer/Feature_comparison). |

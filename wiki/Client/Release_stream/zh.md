# 更新通道

osu! 有若干 **更新通道**（即客户端的版本），这些通道以不同的频率接收更新。为了早点发现潜在问题，新的功能和变化在发布到主要版本之前，会先发布到预览版本。

| 更新通道 | 客户端 | 说明 |
| :-- | :-- | :-- |
| `正式版 (Stable)` | osu!(stable) | osu!(stable) 的最新完整版；[不再开发新功能 (feature-locked)](https://zh.wikipedia.org/wiki/Freeze_(software_engineering))；在测试版之后收到更新。 |
| `测试版 (Cutting Edge)` | osu!(stable) | 更早接收 osu!(stable) 更新；不再开发新功能；与 `正式版` 相比多出几项功能；只有 [osu! 支持者](/wiki/osu!supporter)才能进行[多人游戏](/wiki/Client/Interface/Multiplayer)。 |
| [Lazer](Lazer) | osu!(lazer) | 经过完全重写的游戏客户端；[开源](https://github.com/ppy/osu)；开发进程最活跃，拥有[许多新的功能和变化](/wiki/Help_centre/Upgrading_to_lazer#osu!(lazer)-功能变化)。 |
| `Tachyon (Unstable)` | osu!(lazer) | `Lazer` 的预发布版本；更早更快地收到 osu!(lazer) 更新。 |

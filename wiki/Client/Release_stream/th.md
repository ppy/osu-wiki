# สายการพัฒนา (Release stream)

osu! มีการแบ่ง **สายการพัฒนา (Release stream)** หรือเวอร์ชันของตัวเกมออกเป็นหลายระดับ ซึ่งจะได้รับรายการอัปเดตในอัตราความถี่ที่ต่างกัน เพื่อช่วยให้ทีมพัฒนาสามารถตรวจหาและแก้ไขปัญหาได้ตั้งแต่เนิ่นๆ ฟีเจอร์และการเปลี่ยนแปลงใหม่ๆ จะถูกใส่ไว้ในเวอร์ชันทดสอบก่อนที่จะถูกนำมาใช้ในเวอร์ชัน Stable ต่อไป

| สายการพัฒนา | หมายเหตุ |
| :-- | :-- |
| Stable | เวอร์ชันปัจจุบันที่เป็นที่นิยมที่สุด ไม่มีการเพิ่มฟีเจอร์ใหม่ (Feature-locked) และจะได้รับรายการอัปเดตหลังจากผ่านการทดสอบในรุ่น Beta แล้ว |
| Beta | ได้รับรายการอัปเดตหลังจากผ่านการทดสอบในรุ่น Cutting Edge แล้ว เป็นเวอร์ชันที่ไม่มีการเพิ่มฟีเจอร์ใหม่แล้วเช่นกัน |
| Cutting Edge | ได้รับรายการอัปเดตใหม่ล่าสุดอย่างรวดเร็วเพื่อการทดสอบขั้นต้น ไม่มีการเพิ่มฟีเจอร์ใหม่ |
| [Lazer](Lazer) | ตัวเกมเวอร์ชันที่เขียนขึ้นใหม่ทั้งหมด เป็นสายการพัฒนาที่มีความเคลื่อนไหวมากที่สุด และมี [ความแตกต่างรวมถึงฟีเจอร์ใหม่ๆ มากมาย](/wiki/Client/Release_stream/Lazer/Feature_comparison) |

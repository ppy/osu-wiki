# ตัวเลือกและการตั้งค่า (Options)

![เมนูการตั้งค่า](img/options_basic.jpg "เมนูการตั้งค่า (แถบด้านข้างซ้าย)")

ในเมนูหลัก คลิกที่ปุ่ม `Options` หรือกดปุ่ม `O` (หรือกด `Ctrl` + `O` ซึ่งใช้งานได้จากเกือบทุกหน้าจอ) เพื่อเข้าสู่แถบเมนูการตั้งค่าด้านข้างและปรับแต่งส่วนต่างๆ ของ osu! เมื่อเปิดเมนูขึ้นมาแล้ว คุณสามารถพิมพ์ในช่องค้นหาเพื่อหาการตั้งค่าที่ต้องการได้อย่างรวดเร็ว หากคำที่คุณค้นหาไม่มีผลลัพธ์ ตัวหนังสือในช่องค้นหาจะสั่นเพื่อแจ้งให้ทราบ

เมื่อมีการเปลี่ยนค่าของการตั้งค่าที่มีค่าเริ่มต้น (Default) แถบสีเทาแนวตั้งทางด้านซ้ายจะสว่างเป็นสีเหลืองสำหรับการตั้งค่านั้น การคลิกที่แถบสีเหลืองจะเป็นการรีเซ็ตค่ากลับเป็นค่าเริ่มต้น

## General (ทั่วไป)

![ไอคอน General](img/general.jpg "ไอคอน General")

ส่วนนี้เกี่ยวข้องกับบัญชีของคุณ การตั้งค่าภาษา และการอัปเดตของ osu!

### Sign-in (การลงชื่อเข้าใช้)

หากคุณยังไม่ได้ลงชื่อเข้าใช้ ตัวเกมจะเปิดเมนูการตั้งค่านี้ขึ้นมาโดยอัตโนมัติเมื่อเริ่มโปรแกรมเพื่อแจ้งให้คุณลงชื่อเข้าใช้ คุณสามารถข้ามส่วนนี้ได้โดยการกด `Esc`, คลิกปุ่มย้อนกลับ หรือเข้าสู่หน้าเลือกเพลง หากคุณไม่ลงชื่อเข้าใช้ ตัวเกมจะระบุสถานะของคุณเป็น "Guest" (แขก) และคุณจะไม่สามารถดูคะแนนออนไลน์, ส่งคะแนนขึ้นระบบ หรือเล่นโหมด [มัลติเพลเยอร์ (Multi)](/wiki/Client/Interface/Multiplayer) ได้

หากคุณเป็นแขก ตัวเลือกต่อไปนี้จะแสดงขึ้น:

| ชื่อ | คำอธิบาย | ประเภท | ค่าเริ่มต้น |
| :-- | :-- | :-- | :-- |
| `Username` | ช่องสำหรับกรอกชื่อผู้ใช้ | ข้อความ | *(ว่าง)* |
| `Password` | ช่องสำหรับกรอกรหัสผ่าน | รหัสผ่าน | *(ว่าง)* |
| `Remember Username` | หากเลือกไว้ ตัวเกมจะจำชื่อผู้ใช้ที่กรอกไว้ | ช่องติ๊ก | `เปิดใช้งาน` |
| `Remember Password` | หากเลือกไว้ ตัวเกมจะจำรหัสผ่านที่กรอกไว้ และจะลงชื่อเข้าใช้ให้อัตโนมัติในครั้งถัดไป | ช่องติ๊ก | `ปิดใช้งาน` |
| `Sign In` | ลงชื่อเข้าใช้ osu! ด้วยข้อมูลที่กรอกไว้ (สามารถกด `Enter` หลังจากกรอกเสร็จได้) | ปุ่ม |  |
| `Create an account` | เปิดหน้าจอ [การลงทะเบียนบัญชี](/wiki/Registration) | ปุ่ม |  |

---

หากคุณลงชื่อเข้าใช้แล้ว คุณจะเห็น:

| ชื่อ | คำอธิบาย | ประเภท |
| :-- | :-- | :-- |
| `Logged in as {ชื่อผู้ใช้}` | แสดงเมนูผู้ใช้ (รายละเอียดด้านล่าง) | ปุ่ม |

---

หากคุณคลิกที่ปุ่ม `Logged in as {ชื่อผู้ใช้}` เมนูที่มีปุ่มดังต่อไปนี้จะปรากฏขึ้น:

| ชื่อ | คำอธิบาย | ประเภท |
| :-- | :-- | :-- |
| `1. View Profile` | ดูหน้าโปรไฟล์ของคุณบนเว็บไซต์ | ปุ่ม |
| `2. Sign Out` | ลงชื่อออกจากบัญชีในเครื่องนี้ | ปุ่ม |
| `3. Change Avatar` | เปลี่ยนรูปโปรไฟล์ (จะนำคุณไปยังเว็บไซต์) | ปุ่ม |
| `4. Close` | ปิดหน้าต่างนี้ (หรือกด `Esc`) | ปุ่ม |

คุณยังสามารถเข้าถึงเมนูนี้ได้โดยการคลิกที่แถบข้อมูลผู้ใช้ของคุณในทุกที่ที่มีการแสดงผล

## Graphics (กราฟิก)

![ไอคอน Graphics](img/graphics.jpg "ไอคอน Graphics")

ส่วนนี้เกี่ยวข้องกับรูปลักษณ์ของตัวเกม อินเทอร์เฟซ และการตั้งค่าภาพ

### Renderer (การประมวลผลภาพ)

| ชื่อ | คำอธิบาย | ประเภท | ค่าเริ่มต้น |
| :-- | :-- | :-- | :-- |
| `Frame limiter` | ตั้งค่าการจำกัดเฟรมเรต (รายละเอียดด้านล่าง) | รายการเลือก | `Optimal` |
| `Show FPS counter` | เปิด/ปิด ตัวนับ FPS ที่มุมขวาล่างของหน้าจอ | ช่องติ๊ก | `ปิดใช้งาน` |
| `Compatibility mode` | ใช้ระบบวาดภาพแบบเก่าสำหรับคอมพิวเตอร์รุ่นเก่า (ต้องรีสตาร์ทเกมหลังจากเปลี่ยน) | ช่องติ๊ก | `ปิดใช้งาน` |
| `Reduce dropped frames` | ปรับการตั้งค่ากราฟิกเพื่อลดอาการเฟรมกระตุก | ช่องติ๊ก | `ปิดใช้งาน` |
| `Detect performance issues` | แจ้งเตือนหากมีโปรแกรมอื่นในเครื่องที่อาจส่งผลกระทบต่อประสิทธิภาพของเกม | ช่องติ๊ก | `เปิดใช้งาน` |

---

*หมายเหตุ: เมื่อหน้าต่างเกมไม่ได้ถูกเลือกใช้งาน (Inactive) เฟรมเรตจะถูกจำกัดที่ 30fps โดยอัตโนมัติ*\
*ข้อควรระวังสำหรับผู้ใช้โน้ตบุ๊ก: การใช้ `Unlimited (gameplay)` อาจทำให้เครื่องร้อนเกินไป!*

ตัวเลือกในรายการ `Frame Limiter`:

| ชื่อ | คำอธิบาย |
| :-- | :-- |
| `VSync` | จำกัดเฟรมเรตให้เท่ากับความถี่รีเฟรชเรต (Refresh rate) ของหน้าจอคุณ |
| `Power Saving` | จำกัดเฟรมเรตที่ 2 เท่าของรีเฟรชเรตหน้าจอ |
| `Optimal` | จำกัดเฟรมเรตที่ 8 เท่าของรีเฟรชเรตหน้าจอ (สูงสุดไม่เกิน 960fps) |
| `Unlimited (gameplay)` | ปลดล็อกเฟรมเรตสูงสุด (เฉพาะในระหว่างการเล่น) |

- **คำอธิบาย VSync:** คือการบังคับให้เกมรอวาดภาพให้เสร็จทั้งเฟรมก่อนที่จะแสดงผล ช่วยแก้ปัญหา "ภาพฉีก" (Screen tearing) แต่อาจทำให้เกิดอาการหน่วง (Input lag) ได้
- **คำอธิบาย Unlimited (gameplay):** จะปลดล็อกเฟรมเรตเฉพาะตอนเล่นแมพเท่านั้น เมื่อไม่ได้เล่นจะจำกัดที่ 240fps (หรือ 2 เท่าของจอ) ตัวเลือกนี้ **ไม่แนะนำ** เนื่องจากอาจทำให้เกิดอาการกระตุกได้ การใช้ `Optimal` จะให้ค่าความหน่วงที่แทบไม่ต่างกันแต่มีความเสถียรกว่า

### Layout (เลย์เอาต์)

| :-- | :-- | :-- | :-- |
| `Resolution` | ตั้งค่าความละเอียดหน้าจอ | รายการเลือก |  |
| `Fullscreen mode` | เล่นแบบเต็มหน้าจอ (ช่วยลดความหน่วงในการกด) | ช่องติ๊ก | `เปิดใช้งาน` |
| `Render at native resolution` | แสดงผลที่ความละเอียดจริงของหน้าจอแต่ย่อขนาดหน้าต่างเกมไว้ตรงกลาง (Letterboxing) | ช่องติ๊ก | `เปิดใช้งาน` |

| `Vertical position` | ปรับตำแหน่งในแนวตั้ง (เฉพาะเมื่อเปิด Letterboxing) | แถบเลื่อน | `0%` |

## เกร็ดน่ารู้ (Trivia)

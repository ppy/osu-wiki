# osu!api

osu!api dapat digunakan oleh siapapun selama mereka memiliki akun osu!, berbeda dengan perizinan API pada sebelumnnya yang hanya dapat digunakan sesuai dengan permintaan kerena bersifat tertutup.

Anda dapat menemukan dokumentasi API di [Wiki osu!api](https://github.com/ppy/osu-api/wiki). Versi API saat ini menggunakan kunci pribadi untuk mengautentikasi permintaan. Anda dapat mengajukan permohonan Token API di [halaman osu!api](https://osu.ppy.sh/p/api) pada situs web osu!.

Perhatikan bahwa dokumentasi pada wiki osu!api mungkin tidak sepenuhnya lengkap, dan API itu sendiri pengembangannya sudah dalam status tidak aktif. API baru yang jauh lebih kuat sedang dikembangkan di [osu-web](https://github.com/ppy/osu-web).

# osu!api

У веб-сайта osu! есть общедоступный [API](https://ru.wikipedia.org/wiki/API) — набор методов, с помощью которых можно получить различную информацию об игроках, картах и т.д. в машиночитаемом формате.

Примеры данных, а также вся документация к API находятся на [osu!api wiki](https://github.com/ppy/osu-api/wiki).
Для доступа к API используется приватный ключ пользователя, который можно взять на странице [регистрации приложения для osu!](https://osu.ppy.sh/p/api).

**Важно:** документация к osu!api может быть неполной или неточной, как минимум потому, что развитие проекта приостановлено, и все силы разработчиков направлены на [osu-web](https://github.com/ppy/osu-web) и его API.

# Mod request

**Mod request** เป็นคำที่ใช้เมื่อผู้สร้างต้องการความช่วยเหลือและอยากปรับปรุง [บีตแมป](/wiki/Beatmap) ของตัวเอง ทำได้หลายวิธี แต่ส่วนใหญ่จะทำโดยขอ normal mod หรือ [mod for mod](/wiki/Modding/Mod_for_mod) ใน [forum modding queues](https://osu.ppy.sh/community/forums/60) เมื่อ mod request ได้รับการตอบรับแล้ว ขั้นตอน [modding](/wiki/Modding) จะเริ่มขึ้น และบีตแมปจะถูกตรวจหาข้อผิดพลาดที่อาจมี รวมถึงจุดที่ปรับปรุงได้

# 模组入门奖章解锁条件

| 奖章名称 | 条件 |
| :-- | :-- |
| Finality | 在启用 [Sudden Death](/wiki/Gameplay/Game_modifier/Sudden_Death) 模组的情况下，通过一张过审 (Qualified)、上架 (Ranked) 或达标 (Approved) 谱面。 |
| Perfectionist | 在启用 [Perfect](/wiki/Gameplay/Game_modifier/Perfect) 模组的情况下，通过一张过审 (Qualified)、上架 (Ranked) 或达标 (Approved) 谱面。 |
| Rock Around The Clock | 在启用 [Hard Rock](/wiki/Gameplay/Game_modifier/Hard_Rock) 模组的情况下，通过一张过审 (Qualified)、上架 (Ranked) 或达标 (Approved) 谱面。 |
| Time And A Half | 在启用 [Double Time](/wiki/Gameplay/Game_modifier/Double_Time) 模组的情况下，通过一张过审 (Qualified)、上架 (Ranked) 或达标 (Approved) 谱面。 |
| Sweet Rave Party | 在启用 [Nightcore](/wiki/Gameplay/Game_modifier/Nightcore) 模组的情况下，通过一张过审 (Qualified)、上架 (Ranked) 或达标 (Approved) 谱面。 |
| Blindsight | 在启用 [Hidden](/wiki/Gameplay/Game_modifier/Hidden) 模组的情况下，通过一张过审 (Qualified)、上架 (Ranked) 或达标 (Approved) 谱面。 |
| Are You Afraid Of The Dark? | 在启用 [Flashlight](/wiki/Gameplay/Game_modifier/Flashlight) 模组的情况下，通过一张过审 (Qualified)、上架 (Ranked) 或达标 (Approved) 谱面。 |
| Dial It Right Back | 在启用 [Easy](/wiki/Gameplay/Game_modifier/Easy) 模组的情况下，通过一张过审 (Qualified)、上架 (Ranked) 或达标 (Approved) 谱面。 |
| Risk Averse | 在启用 [NoFail](/wiki/Gameplay/Game_modifier/No_Fail) 模组的情况下，通过一张过审 (Qualified)、上架 (Ranked) 或达标 (Approved) 谱面。 |
| Slowboat | 在启用 [Half Time](/wiki/Gameplay/Game_modifier/Half_Time) 模组的情况下，通过一张过审 (Qualified)、上架 (Ranked) 或达标 (Approved) 谱面。 |
| Burned Out | 在启用 [Spun Out](/wiki/Gameplay/Game_modifier/Spun_Out) 模组的情况下，通过一张过审 (Qualified)、上架 (Ranked) 或达标 (Approved) 谱面。由于其他游戏模式没有 `Spun Out` 模组，此奖章只能在 [osu!](/wiki/Game_mode/osu!) 游戏模式中获得。 |

# Unlock requirements for Mod Introduction medals

| Medal name | Requirement |
| :-- | :-- |
| Finality | เคลียร์บีตแมป Qualified, Ranked หรือ Approved โดยเปิดม็อด [Sudden Death](/wiki/Gameplay/Game_modifier/Sudden_Death) |
| Perfectionist | เคลียร์บีตแมป Qualified, Ranked หรือ Approved โดยเปิดม็อด [Perfect](/wiki/Gameplay/Game_modifier/Perfect) |
| Rock Around The Clock | เคลียร์บีตแมป Qualified, Ranked หรือ Approved โดยเปิดม็อด [Hard Rock](/wiki/Gameplay/Game_modifier/Hard_Rock) |
| Time And A Half | เคลียร์บีตแมป Qualified, Ranked หรือ Approved โดยเปิดม็อด [Double Time](/wiki/Gameplay/Game_modifier/Double_Time) |
| Sweet Rave Party | เคลียร์บีตแมป Qualified, Ranked หรือ Approved โดยเปิดม็อด [Nightcore](/wiki/Gameplay/Game_modifier/Nightcore) |
| Blindsight | เคลียร์บีตแมป Qualified, Ranked หรือ Approved โดยเปิดม็อด [Hidden](/wiki/Gameplay/Game_modifier/Hidden) |
| Are You Afraid Of The Dark? | เคลียร์บีตแมป Qualified, Ranked หรือ Approved โดยเปิดม็อด [Flashlight](/wiki/Gameplay/Game_modifier/Flashlight) |
| Dial It Right Back | เคลียร์บีตแมป Qualified, Ranked หรือ Approved โดยเปิดม็อด [Easy](/wiki/Gameplay/Game_modifier/Easy) |
| Risk Averse | เคลียร์บีตแมป Qualified, Ranked หรือ Approved โดยเปิดม็อด [NoFail](/wiki/Gameplay/Game_modifier/No_Fail) |
| Slowboat | เคลียร์บีตแมป Qualified, Ranked หรือ Approved โดยเปิดม็อด [Half Time](/wiki/Gameplay/Game_modifier/Half_Time) |
| Burned Out | เคลียร์บีตแมป Qualified, Ranked หรือ Approved โดยเปิดม็อด [Spun Out](/wiki/Gameplay/Game_modifier/Spun_Out) medal นี้ทำได้เฉพาะในโหมด [osu!](/wiki/Game_mode/osu!) เพราะม็อด `Spun Out` ไม่มีในโหมดเกมอื่น |
| Gear Shift | เคลียร์บีตแมป Qualified, Ranked หรือ Approved โดยเปิดม็อดใดก็ได้จากหมวด [Conversion](/wiki/Gameplay/Game_modifier_(lazer)#conversion) medal นี้ทำได้เฉพาะใน [osu!(lazer)](/wiki/Client/Release_stream/Lazer) เพราะหมวดม็อดนี้มีเฉพาะที่นั่น |
| Game Night | เคลียร์บีตแมป Qualified, Ranked หรือ Approved โดยเปิดม็อดใดก็ได้จากหมวด [Fun](/wiki/Gameplay/Game_modifier_(lazer)#fun) medal นี้ทำได้เฉพาะใน [osu!(lazer)](/wiki/Client/Release_stream/Lazer) เพราะหมวดม็อดนี้มีเฉพาะที่นั่น |

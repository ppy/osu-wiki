# Requisitos de desbloqueo para las medallas de Mod Introduction

| Nombre de la medalla | Requisito |
| :-- | :-- |
| Finality | Pasar un mapa calificado, clasificado o aprobado con el mod [Sudden Death](/wiki/Gameplay/Game_modifier/Sudden_Death) habilitado. |
| Perfectionist | Pasar un mapa calificado, clasificado o aprobado con el mod [Perfect](/wiki/Gameplay/Game_modifier/Perfect) habilitado. |
| Rock Around The Clock | Pasar un mapa calificado, clasificado o aprobado con el mod [Hard Rock](/wiki/Gameplay/Game_modifier/Hard_Rock) habilitado. |
| Time And A Half | Pasar un mapa calificado, clasificado o aprobado con el mod [Double Time](/wiki/Gameplay/Game_modifier/Double_Time) habilitado. |
| Sweet Rave Party | Pasar un mapa calificado, clasificado o aprobado con el mod [Nightcore](/wiki/Gameplay/Game_modifier/Nightcore) habilitado. |
| Blindsight | Pasar un mapa calificado, clasificado o aprobado con el mod [Hidden](/wiki/Gameplay/Game_modifier/Hidden) habilitado. |
| Are You Afraid Of The Dark? | Pasar un mapa calificado, clasificado o aprobado con el mod [Flashlight](/wiki/Gameplay/Game_modifier/Flashlight) habilitado. |
| Dial It Right Back | Pasar un mapa calificado, clasificado o aprobado con el mod [Easy](/wiki/Gameplay/Game_modifier/Easy) habilitado. |
| Risk Averse | Pasar un mapa calificado, clasificado o aprobado con el mod [NoFail](/wiki/Gameplay/Game_modifier/No_Fail) habilitado. |
| Slowboat | Pasar un mapa calificado, clasificado o aprobado con el mod [Half Time](/wiki/Gameplay/Game_modifier/Half_Time) habilitado. |
| Burned Out | Pasar un mapa calificado, clasificado o aprobado con el mod [Spun Out](/wiki/Gameplay/Game_modifier/Spun_Out) habilitado. Esta medalla solo se puede conseguir en el modo de juego [osu!](/wiki/Game_mode/osu!) ya que el mod `Spun Out` no está presente en otros modos de juego. |
| Gear Shift | Pasar un mapa calificado, clasificado o aprobado con cualquier mod de la categoría [Conversión](/wiki/Gameplay/Game_modifier_(lazer)#conversión) activado. Esta medalla solo se puede conseguir en [osu!(lazer)](/wiki/Client/Release_stream/Lazer), ya que la categoría de mods es exclusiva de esta versión. |
| Game Night | Pasar un mapa calificado, clasificado o aprobado con cualquier mod de la categoría [Diversión](/wiki/Gameplay/Game_modifier_(lazer)#diversión) activado. Esta medalla solo se puede conseguir en [osu!(lazer)](/wiki/Client/Release_stream/Lazer), ya que la categoría de mods es exclusiva de esta versión. |

# История osu!

Записи об истории osu!, собранные в osu! wiki.

| Год | Описание |
| :-: | :-- |
| [2007](2007) | Выпуск первой сборки игры |
| [2008](2008) | osu!taiko, osu!catch, больше модов и значительные улучшения |
| [2009](2009) | osu! на iPhone, новые режимы мультиплеера, система кудосу |
| [2010](2010) | Wai-con 2010 |
| [2011](2011) | osu! wiki |
| [2012](2012) | Пять лет osu!, osu!mania, поддержка широкоэкранного режима и визуальные настройки |
| [2013](2013) | Переработка пользовательского интерфейса, HD-скины, улучшенная поддержка широкоформатных экранов, редактор osu!mania, внутриигровые коллекции, moddingV2, Disqus, osu!api, шкала точности, Comic Fiesta 2013, скачивание без ограничений и osu!cuttingedge (только для обладателей osu!mania)|
| [2014](2014) | osu!mania BPM скроллинг, внутриигровые интро/аутро, osu!cuttingedge (для всех)|
| [2015](2015) | редизайн osu!web, osu!weekly, и osu!coins |
| [2016](2016) | scorev2, анонс новой версии osu! — "lazer" — с открытым исходным кодом (ранее известной как "osu!lazer"), а также перезапуск osu! wiki |
| [2017](2017) | Выпуск первой lazer-сборки osu! и интеграция Discord со стабильным клиентом |
| [2018](2018) | osu!next, отказ от старого сайта, индикатор общего времени игры |
| 2019-2021 | *Не задокументировано* |
| [2022](2022) | 15 лет osu!, новая серия ежемесячных новостей osu!monthly|
| 2023 - По настоящее время | *Не задокументировано* |

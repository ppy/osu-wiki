# ประวัติของ osu! (History of osu!)

บันทึกประวัติศาสตร์ของ osu! ทั้งหมดอยู่ใน osu! wiki

| ปี (Year) | รายละเอียด (Description) |
| :-: | :-- |
| [2007](2007) | ตัวเกมเวอร์ชันเริ่มต้น (Initial game build) |
| [2008](2008) | osu!taiko, osu!catch, เพิ่มม็อด และการปรับปรุงครั้งใหญ่ |
| [2009](2009) | osu! iPhone, โหมดผู้เล่นหลายคนใหม่, ระบบ kudosu |
| [2010](2010) | งาน Wai-con 2010 |
| [2011](2011) | เริ่มมี osu! wiki |
| [2012](2012) | ฉลองครบรอบ 5 ปี osu!, เพิ่มโหมด osu!mania, รองรับหน้าจอไวด์สกรีน และการตั้งค่าภาพ |
| [2013](2013) | ปรับปรุง UI ครั้งใหญ่, สกินแบบ HD, ปรับปรุงการรองรับไวด์สกรีน, ตัวแก้ไข osu!mania, ระบบคอลเลกชันในเกม, เริ่มใช้ moddingV2, Disqus, osu!api, accuracy overlays, งาน Comic Fiesta 2013, ดาวน์โหลดได้ไม่จำกัด และ osu!cuttingedge (เฉพาะ osu!supporter เท่านั้น) |
| [2014](2014) | การเลื่อน BPM ใน osu!mania, ลำดับ intro/outro ในเกม, osu!cuttingedge (เปิดให้ทุกคนใช้งาน) |
| [2015](2015) | ปรับปรุงหน้าเว็บ osu! ใหม่, osu!weekly และ osu!coins |
| [2016](2016) | ระบบ scorev2, ประกาศเวอร์ชันโอเพนซอร์สใหม่ "lazer" ของ osu! (เดิมชื่อ "osu!lazer") และการปรับปรุง osu! wiki ครั้งใหญ่ |
| [2017](2017) | ปล่อยตัวเกม lazer เวอร์ชันแรก และการเชื่อมต่อกับ Discord ในตัวเกมเวอร์ชัน stable |
| [2018](2018) | โปรเจกต์ osu!next, ยกเลิกการใช้งานเว็บไซต์เก่า, แสดงเวลาเล่นทั้งหมด |
| 2019–2021 | *ยังไม่มีการบันทึกข้อมูล* |
| [2022](2022) | ฉลองครบรอบ 15 ปี osu!, เริ่มซีรีส์โพสต์ข่าว osu!monthly ใหม่ |
| 2023–ปัจจุบัน | *ยังไม่มีการบันทึกข้อมูล* |

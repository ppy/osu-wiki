# Internet Relay Chat

*สำหรับข้อมูลเพิ่มเติม ดูที่: [Internet Relay Chat (วิกิพีเดีย)](https://en.wikipedia.org/wiki/Internet_relay_chat)*

**Internet Relay Chat** (**IRC**) คือโปรโตคอลในระดับชั้นประยุกต์ (Application layer) ที่ใช้สำหรับการพูดคุยผ่านตัวโปรแกรมออนไลน์ต่างๆ

## osu!Bancho

*ประกาศ: เซิร์ฟเวอร์นี้รองรับโปรโตคอล IRC เพียงบางส่วนเท่านั้น ฟีเจอร์ขั้นสูงบางอย่างของโปรแกรม IRC (เช่น [HexChat](https://hexchat.github.io/)) อาจทำงานไม่ถูกต้องหรือไม่สามารถใช้งานได้เลย*

*osu!Bancho* (บางครั้งเรียกสั้นๆ ว่า *Bancho*) ทำหน้าที่เป็นเกตเวย์สำหรับการเข้าถึง IRC (แชทในเกม) คุณสามารถใช้โปรแกรม IRC สมัยใหม่รุ่นใดก็ได้ในการเชื่อมต่อ (เลื่อนลงไปด้านล่างเพื่อดูรายการโปรแกรมยอดนิยม)

## การเชื่อมต่อ (Connection)

เปิดการตั้งค่าในโปรแกรม IRC ของคุณและกรอกข้อมูลดังนี้ (คุณอาจจำเป็นต้องเพิ่มเซิร์ฟเวอร์ใหม่ก่อน):

- **Server:** `irc.ppy.sh`
- **Port:** `6667`
- **SSL:** ปิดการใช้งาน (Disabled)
- **Username:** ชื่อผู้ใช้ osu! ของคุณ โดยให้เปลี่ยนช่องว่างเป็นเครื่องหมายขีดล่าง (เช่น `beppy master 1000` เปลี่ยนเป็น `beppy_master_1000`)
- **Password:** รหัสผ่านสำหรับ IRC จาก [หน้าการตั้งค่าบัญชี (Account settings)](https://osu.ppy.sh/home/account/edit#legacy-api)

*คำเตือน: รหัสผ่าน IRC ของคุณจะไม่เหมือนกับรหัสผ่านบัญชี osu! **ห้ามแบ่งปันรหัสผ่านนี้ให้ผู้อื่นเด็ดขาด***

## คำสั่ง IRC พื้นฐาน

| คำสั่ง | คำอธิบาย |
| :-- | :-- |
| `/join <#ชื่อแชนแนล>` | เข้าร่วมแชนแนลที่ระบุ |
| `/part <#ชื่อแชนแนล>` | ออกจากแชนแนลที่ระบุ |
| `/me <การกระทำ>` | ส่งข้อความแบบบอกการกระทำ (บุรุษที่ 3) |
| `/ignore <ชื่อผู้ใช้>` | เพิกเฉยต่อผู้ใช้ (ซ่อนข้อความจากคนคนนั้น) |
| `/away <ข้อความ>` | ฝากข้อความไว้สำหรับทุกคนที่พยายามติดต่อคุณในขณะที่คุณไม่อยู่หน้าจอ |
| `/away` | ล้างข้อความสถานะไม่อยู่หน้าจอ |
| `/query <ชื่อผู้ใช้>` | เปิดหน้าต่างแชทส่วนตัวกับผู้ใช้ที่ระบุ (เปลี่ยนช่องว่างเป็นขีดล่าง) |

## การปิดข้อความแจ้งเตือน JOIN/PART/QUIT

โดยปกติแล้ว โปรแกรม IRC หลายตัวจะแจ้งให้คุณทราบเมื่อมีผู้ใช้เข้าร่วมหรือออกจากแชนแนล สำหรับเครือข่ายอย่าง osu!Bancho ที่มีผู้ใช้เข้าออกหลายพันคนต่อวัน ข้อความเหล่านี้จะกลายเป็นข้อความสแปมที่คุณอาจต้องการซ่อน ตารางด้านล่างคือวิธีปิดข้อความเหล่านี้ในโปรแกรมต่างๆ:

| โปรแกรม IRC | วิธีการปิดข้อความ |
| :-- | :-- |
| [HexChat](https://hexchat.github.io/) | ติ๊กที่ "Hide join and part messages" ในส่วน `Settings` -> `Preferences` -> `Chatting` -> `General` |
| [ircII](http://www.eterna.com.au/ircii/) | พิมพ์คำสั่ง `IGNORE * CRAP` |
| [Irssi](https://irssi.org) | พิมพ์คำสั่ง `/ignore * JOINS PARTS QUITS` |
| [Weechat](https://weechat.org/) | พิมพ์คำสั่ง `/filter add joinquit * irc_join,irc_part,irc_quit *` |
| [Konversation](https://konversation.kde.org/) | ติ๊กที่ "Hide Join/Part/Nick Events" ในส่วน `Settings` -> `Configure Konversation...` -> `Behavior` -> `Chat Window` |
| [KVIrc](https://www.kvirc.net/) | (ดูรายละเอียดได้ที่ [หน้าวิกิของ KVIrc](https://github.com/kvirc/KVIrc/wiki/FAQ#how-do-i-suppress-join-part-and-quit-messages)) |
| [mIRC](https://www.mirc.com/) | ไปที่ mIRC Options -> `IRC` -> คลิกปุ่ม `Events...` และเลือก `Hide` สำหรับ joins และ parts |
| [Quassel IRC](https://quassel-irc.org/) | คลิกขวาในหน้าแชทแล้วเลือก Joins/Parts/Quits ในเมนู "Hide Events" |
| [XChat](http://xchat.org/) | พิมพ์คำสั่ง `/set irc_conf_mode 1` |

## คำถามที่พบบ่อย (FAQ)

### ฉันไม่สามารถเข้าสู่ระบบได้

osu!Bancho ใช้การตรวจสอบสิทธิ์แบบข้อความธรรมดา (Plain text) โปรดตรวจสอบให้แน่ใจว่าไม่ได้เลือกโหมดการตรวจสอบสิทธิ์พิเศษใดๆ ในการตั้งค่าโปรแกรม IRC ของคุณ

หรือลองเปลี่ยนที่อยู่เซิร์ฟเวอร์เป็น `cho.ppy.sh` (ซึ่งจะเชื่อมต่อเข้าสู่ osu!Bancho เช่นกัน)

### ฉันได้รับข้อความผิดพลาด "Bad Authentication Token"

ลองทำตามวิธีต่อไปนี้:

1. ตรวจสอบให้แน่ใจว่าใช้รหัสผ่าน IRC ที่ถูกต้องจาก [หน้าการตั้งค่าบัญชี](https://osu.ppy.sh/home/account/edit#legacy-api)
2. หากชื่อผู้ใช้ของคุณมีช่องว่าง ให้เปลี่ยนเป็นเครื่องหมายขีดล่าง (เช่น `My Name` เปลี่ยนเป็น `My_Name`)

### ฉันสามารถใช้ชื่อผู้ใช้อื่นได้หรือไม่?

ไม่ได้

### ฉันจะพิมพ์ชื่อผู้ใช้อื่นอย่างรวดเร็วได้อย่างไร?

พิมพ์ตัวอักษรสองสามตัวแรกของชื่อผู้ใช้ แล้วกดปุ่ม `Tab` เพื่อให้ระบบเติมชื่อให้อัตโนมัติ

### ทำไมชื่อผู้ใช้บางคนถึงมีสัญลักษณ์นำหน้าต่างกัน?

มาตรฐาน IRC มีแนวคิดเรื่อง "โหมดแชนแนล" หรือชุดการกระทำที่ผู้ใช้แต่ละคนสามารถทำได้ ใน osu!Bancho มีสองโหมดที่ใช้สำหรับกลุ่มผู้ใช้พิเศษ:

- `+` หรือ "voice status": ผู้ใช้ที่เชื่อมต่อผ่านโปรแกรม IRC ภายนอก
- `@` หรือ "chat operator status": ผู้ใช้ที่เป็นผู้ดูแลแชท ([GMT](/wiki/People/Global_Moderation_Team) หรือ [NAT](/wiki/People/Nomination_Assessment_Team))

สำหรับผู้ใช้ที่เชื่อมต่อผ่านตัวเกม osu! หรือหน้าเว็บไซต์จะไม่มีสัญลักษณ์นำหน้า

### มีคนส่งข้อความมา แต่ชื่อของเขาไม่อยู่ในรายชื่อผู้ใช้ในแชนแนล!

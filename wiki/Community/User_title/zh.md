# 玩家头衔

*有关获得特殊玩家头衔的玩家列表，请参阅：[拥有特殊头衔的玩家](/wiki/People/Users_with_unique_titles)。*

![Ephemeral 的个人主页截图](img/dev.png?1 "在 Ephemeral 的主页上方可以看到 \"osu!team\" 头衔")

**玩家头衔**（用户头衔）是由 [osu!team](/wiki/People/osu!_team) 设置在玩家个人主页上的小标签。大多数头衔指示了玩家所处的[玩家组](/wiki/People/User_group)身份。除此之外，还有一些[特殊头衔](/wiki/People/Users_with_unique_titles)，获得这些特殊头衔的玩家通常都做出过杰出的贡献或成就。

头衔的颜色通常会根据玩家所处的玩家组身份设置，并且会与玩家名的颜色一致。如果玩家所处的玩家组没有颜色设置，则默认为白色。

## 历史

### 发帖数头衔

![Damnae 在旧网站上的帖子侧边栏截图](img/star-shooter.png "位于 Damnae 已发帖数上方的\"Star Shooter\" 头衔")

在[旧版 osu! 官网](https://old.ppy.sh)的论坛页中，玩家可以通过发送一定量的帖子从而获取头衔：

| 论坛头衔 | 总发帖数 |
| :-: | :-- |
| Rhythm Rookie | 0–4 |
| Tempo Trainee | 5–14 |
| Whistle Blower | 15–29 |
| Cymbal Sounder | 30–49 |
| Beat Clicker | 50–79 |
| Slider Savant | 80–119 |
| Spinner Sage | 120–179 |
| Star Shooter | 180–259 |
| Combo Commander | 260–499 |
| Rhythm Incarnate | 500+ |

这类头衔只会出现在论坛帖子的侧边栏上（不会显示在玩家主页中）。如果设置了其他自定义头衔，这类头衔也不会显示。

## 冷知识

::: Infobox
![](img/multiple-titles.png "VINXIS 的个人主页上同时显示了两个玩家头衔")
:::

- 由于 osu! 论坛曾基于 [phpBB](https://www.phpbb.com/) 架构搭建，根据代码，玩家头衔也曾称为“rank titles”。
- 在 2021 年 3 月之后，如果玩家获得了多个头衔，则会并列展示，中间通过正斜杠连接。但在这之前，只会展示最显眼或最近的玩家头衔。

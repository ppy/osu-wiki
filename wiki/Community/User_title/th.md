# ชื่อตำแหน่งผู้ใช้ (User title)

*สำหรับรายชื่อผู้ที่ได้รับตำแหน่งพิเศษเฉพาะตัว ดูที่ [ผู้ใช้ที่มีตำแหน่งเฉพาะตัว (Users with unique titles)](/wiki/People/Users_with_unique_titles)*

![ภาพถ่ายหน้าจอข้อมูลโปรไฟล์ของ Ephemeral](img/dev.png?1 "ตำแหน่ง \"osu!team\" ปรากฏให้เห็นบริเวณส่วนบนของโปรไฟล์ Ephemeral")

**ชื่อตำแหน่งผู้ใช้ (User title)** คือข้อความสั้นๆ บนหน้าโปรไฟล์ของผู้ใช้ที่ตั้งค่าโดย [ทีมงาน osu!](/wiki/People/osu!_team) ตำแหน่งส่วนใหญ่มักจะบ่งบอกถึงการเป็นสมาชิกใน [กลุ่มผู้ใช้ (User group)](/wiki/People/User_groups) แต่มีผู้ใช้บางคนที่ได้รับ [ตำแหน่งพิเศษเฉพาะตัว](/wiki/People/Users_with_unique_titles) จากความสำเร็จหรือการทำคุณประโยชน์ที่โดดเด่น

สีของชื่อตำแหน่งจะเป็นสีเดียวกับสีชื่อของผู้ใช้ ซึ่งโดยปกติจะถูกกำหนดตามกลุ่มผู้ใช้ที่สังกัด หากผู้ใช้ไม่ได้สังกัดกลุ่มที่มีการกำหนดสี ตำแหน่งจะเป็นสีขาว

## ประวัติความเป็นมา (History)

### ตำแหน่งตามจำนวนการโพสต์ (Post count titles)

![ภาพถ่ายหน้าจอแถบข้อมูลข้างโพสต์ของ Damnae บนเว็บไซต์เก่า](img/star-shooter.png "ตำแหน่ง \"Star Shooter\" ปรากฏอยู่เหนือจำนวนการโพสต์ของ Damnae")

ในส่วนของฟอรัมบน [เว็บไซต์ osu! แบบเก่า](https://old.ppy.sh) ผู้ใช้สามารถได้รับตำแหน่งโดยอัตโนมัติเมื่อทำสถิติจำนวนการโพสต์ถึงเกณฑ์ที่กำหนดดังนี้:

| ชื่อตำแหน่งในฟอรัม | จำนวนการโพสต์รวม |
| :-: | :-- |
| Rhythm Rookie (น้องใหม่หัดจังหวะ) | 0–4 |
| Tempo Trainee (เด็กฝึกหัดจังหวะ) | 5–14 |
| Whistle Blower | 15–29 |
| Cymbal Sounder | 30–49 |
| Beat Clicker | 50–79 |
| Slider Savant | 80–119 |
| Spinner Sage (ผู้อาวุโสเรื่อง Spinner) | 120–179 |
| Star Shooter | 180–259 |
| Combo Commander (ผู้บัญชาการคอมโบ) | 260–499 |
| Rhythm Incarnate (จังหวะในสายเลือด) | 500+ |

ตำแหน่งเหล่านี้จะปรากฏเฉพาะในแถบข้างของโพสต์ในฟอรัมเท่านั้น (ไม่ปรากฏบนหน้าโปรไฟล์) และจะไม่แสดงผลหากผู้ใช้คนนั้นมีตำแหน่งพิเศษอื่นที่ถูกตั้งค่าไว้ก่อนแล้ว

# Титул

*Список игроков, получивших специальные титулы, приведён в статье [Игроки с уникальными титулами](/wiki/People/Users_with_unique_titles).*

![Скриншот информации о профиле Ephemeral](img/dev.png?1 "Титул \"osu!team\" в верхней части профиля Ephemeral.")

**Титул** - это коротка строка в профиле игрока, устанавливаемая [командой osu!](/wiki/People/osu!_team). Большинство титулов указывают на принадлежность к определённой [группе пользователей](/wiki/People/User_groups), но некоторые игроки получают [более уникальные титулы](/wiki/People/Users_with_unique_titles) за выдающиеся достижения или значимый вклад в развитие osu!.

Цвет титула совпадает с цветом игрока, который определяется членством в группе пользователей. Если у игрока нет цвета - титул будет белым.

## История

### Титулы на форуме

![Скриншот боковой панели поста Damnae на старом вебсайте](img/star-shooter.png "Титул \"Star Shooter\" над количеством постов Damnae.")

<!-- the bottom post on https://web.archive.org/web/20131107133826/https://osu.ppy.sh/forum/t/141240 -->

В разделе форума на [старом вебсайте osu!](https://old.ppy.sh)<!-- TODO: wiki page --> игроки могли получать титулы, достигнув определённого количества постов: <!-- https://osu.ppy.sh/community/forums/posts/2306316 -->

| Титул | Количество постов |
| :-: | :-- |
| Rhythm Rookie | 0–4 |
| Tempo Trainee | 5–14 |
| Whistle Blower | 15–29 |
| Cymbal Sounder | 30–49 |
| Beat Clicker | 50–79 |
| Slider Savant | 80–119 |
| Spinner Sage | 120–179 |
| Star Shooter | 180–259 |
| Combo Commander | 260–499 |
| Rhythm Incarnate | 500+ |

Эти титулы были видны только на боковых панелях постов на форуме (не в профилях игроков), а также не отображались, если ранее был установлен более почётный титул.

## Интересные факты

::: Infobox
![](img/multiple-titles.png "Два одновременно отображающихся титула у VINXIS")
:::

- В коде игры титулы называются "ранговыми титулами" (rank titles). Это название осталось с тех времён, когда вебсайт osu! был написан на [phpBB](https://www.phpbb.com/).
- С марта 2021 года игрокам, получившим несколько титулов, теперь присваивается титул, объединяющий их все косой чертой. До этого отображался только самый почётный или самый новый титул.

# กิจกรรม osu! Talk Event

![แบนเนอร์ osu! Talk Event](img/banner.png)

**osu! Talk Event** คือกิจกรรมที่จัดขึ้นโดย [osu! University](/wiki/Community/Discord_servers/osu!_University) ซึ่งเป็นเวทีที่ผู้คนหลากหลายกลุ่มจะมาพูดคุยในหัวข้อต่างๆ ที่น่าสนใจ โดยในแต่ละครั้งจะมี "ธีมหลัก" (Theme) ที่แตกต่างกันออกไป

วิทยากรจะดำเนินการพูดคุยในแชนแนล Stage บน [เซิร์ฟเวอร์ Discord ของ osu!](/wiki/Community/Discord_servers#official) ในระหว่างกิจกรรม จะมีการเปิดเธรด (Thread) ในแชนแนล `#community-meetings` เพื่อใช้ในการร่วมพูดคุยและแลกเปลี่ยนความคิดเห็น

นอกจากนี้ กิจกรรมยังมีการถ่ายทอดสดผ่านทาง [ช่อง Twitch ของ osu! University](https://twitch.tv/osuuniversity) และสามารถรับชมบันทึกย้อนหลังได้ที่ [ช่อง YouTube ของ osu! University](https://www.youtube.com/c/osuuniversity)

หากคุณสนใจเข้าร่วมเป็นวิทยากร หรือต้องการแนะนำบุคคลที่เหมาะสมสำหรับการพูดคุย สามารถกรอกข้อมูลได้ที่ [แบบฟอร์มเสนอชื่อวิทยากร](https://forms.gle/HCD6ac8JwURGh8zx8)

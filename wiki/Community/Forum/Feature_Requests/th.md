# ข้อเสนอแนะเพื่อขอเพิ่มฟีเจอร์ (Feature Requests)

**[Feature Requests](https://osu.ppy.sh/community/forums/4)** คือฟอรัมที่สร้างขึ้นสำหรับรวบรวมข้อเสนอแนะต่างๆ ที่เกี่ยวข้องกับ osu! โดยผู้ใช้สามารถสนับสนุนข้อเสนอแนะและเพิ่มลำดับความสำคัญได้โดยการ [ใช้ดาว (Spending stars)](/wiki/Modding/Star_priority) ที่ได้รับจาก [osu!supporter](/wiki/osu!supporter)

ในปัจจุบัน ฟอรัม Feature Requests นี้ถือว่าล้าสมัยแล้ว และส่วนใหญ่ถูกแทนที่ด้วยระบบการพูดคุยผ่าน [GitHub Discussions](https://github.com/ppy/osu/discussions) แทน

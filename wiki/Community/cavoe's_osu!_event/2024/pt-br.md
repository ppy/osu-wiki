# cavoe's osu! event 2024

![COE 2024 banner](img/banner.jpg)

**cavoe's osu! event 2024** (***COE 2024***) é um evento de osu realizado no  **Brabanthallen in 's-Hertogenbosch (Den Bosch), Holanda**. Esta é a sexta edição do COE

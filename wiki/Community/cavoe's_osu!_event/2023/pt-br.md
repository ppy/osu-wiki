# cavoe's osu! event 2023

![COE 2023 banner](img/banner.png)

**cavoe's osu! event 2023** (***COE 2023***) foi um evento de osu realizado no **Brabanthallen in 's-Hertogenbosch (Den Bosch), Holanda**. foi a quinta edição do COE.

## Eventos

### cavoe's osu! event's Global Taiko Showdown

*Página principal: [cavoe's osu! event's Global Taiko Showdown 2023](/wiki/Tournaments/GTS/COEGTS_2023)*

![COEGTS 2023 banner](/wiki/Tournaments/GTS/COEGTS_2023/img/banner.png)

A equipe do Global Taiko Showdown organizou um torneio 1v1 de osu!taiko em LAN no evento, com uma pool de mapas totalmente customizada com músicas originais para cada beatmap. Jogadores de osu!taiko de qualquer rank puderam participar contanto que eles estivessem presentes durante o torneio

#### Prêmios

| Colocação | Prêmio(s) |
| :-: | :-- |
| ![Coroa dourada](/wiki/shared/crown-gold.png "1st place") | Etiqueta de perfil (pendente), Capa de perfil customizada, grande prêmio físico |
| ![Silver crown](/wiki/shared/crown-silver.png "2nd place") | Capa de perfil customizada, grande prêmio físico |
| ![Bronze crown](/wiki/shared/crown-bronze.png "3rd place") | Capa de perfil customizada, grande prêmio físico |

# cavoe's osu! event 2023

![แบนเนอร์ COE 2023](img/banner.png)

**cavoe's osu! event 2023** (***COE 2023***) คือกิจกรรมงานชุมนุม osu! ที่จัดขึ้นที่ **Brabanthallen ในเมือง 's-Hertogenbosch (Den Bosch) ประเทศเนเธอร์แลนด์** ถือเป็นการจัดงานครั้งที่ 5 ของ cavoe's osu! event

## กิจกรรมภายในงาน

### cavoe's osu! event's Global Taiko Showdown

*หน้าหลัก: [cavoe's osu! event's Global Taiko Showdown 2023](/wiki/Tournaments/GTS/COEGTS_2023)*

ทีมงาน Global Taiko Showdown ได้จัดการแข่งขัน osu!taiko แบบ 1v1 ในรูปแบบ LAN ภายในงาน โดยมีการใช้รายการเพลง (Mappool) ที่แต่งขึ้นมาใหม่ทั้งหมดเป็นพิเศษสำหรับแต่ละแมพ ผู้เล่น osu!taiko ทุกระดับสามารถเข้าร่วมได้หากสามารถเดินทางมาร่วมงานได้ตลอดระยะเวลาการแข่งขัน

#### รางวัล (Prizes)

| อันดับ | รางวัล |
| :-: | :-- |
| ![Gold crown](/wiki/shared/crown-gold.png "1st place") | เหรียญตราหน้าโปรไฟล์, แบนเนอร์หน้าโปรไฟล์แบบพิเศษ, รางวัลใหญ่ (Physical prize) |
| ![Silver crown](/wiki/shared/crown-silver.png "2nd place") | แบนเนอร์หน้าโปรไฟล์แบบพิเศษ, รางวัลพิเศษ (Physical prize) |
| ![Bronze crown](/wiki/shared/crown-bronze.png "3rd place") | แบนเนอร์หน้าโปรไฟล์แบบพิเศษ, รางวัลพิเศษ (Physical prize) |

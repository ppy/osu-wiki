# ศูนย์ช่วยเหลือ (Help centre)

กำลังประสบปัญหาบางอย่างอยู่ใช่ไหม? พวกเราพร้อมช่วยเหลือคุณ! ลองตรวจสอบวิธีแก้ไขปัญหาเบื้องต้นที่พบบ่อยได้จากแถบเมนูด้านซ้ายของหน้าจอ หากปัญหาของคุณไม่ได้ระบุไว้ที่นี่ หรือคุณยังไม่สามารถแก้ไขปัญหาได้ สามารถส่งอีเมลมาหาเราได้ที่ [support@ppy.sh](mailto:support@ppy.sh) หรืออีกช่องทางหนึ่งคือการโพสต์ปัญหาของคุณใน [ฟอรัมความช่วยเหลือ (Help forum)](https://osu.ppy.sh/community/forums/5) ซึ่งจะมีผู้เล่นคนอื่นคอยช่วยเหลือคุณได้เช่นกัน

## ช่วยเราเพื่อให้เราช่วยคุณ {id=diagnostics}

### ไฟล์ Log (Log files) {id=log-files}

**ไฟล์ Log คือบันทึกรายละเอียดการทำงานของตัวเกม Client ในแต่ละช่วงเวลา ซึ่งเราสามารถนำข้อมูลเหล่านี้มาวิเคราะห์เพื่อหาสาเหตุของปัญหาที่เกิดขึ้นกับคุณได้**

ไฟล์เหล่านี้มีประโยชน์อย่างมากและช่วยให้การแก้ปัญหาที่ซับสอนกลายเป็นเรื่องง่าย

หากทีมงานฝ่ายสนับสนุน (Support team) ขอไฟล์ Log เหล่านี้จากคุณ นี่คือขั้นตอนในการรับไฟล์:

1. เปิด osu!
2. คลิกที่ปุ่ม `Options` ในเมนูหลัก หรือกด `Ctrl` + `O`
3. พิมพ์คำว่า `release` ในช่องค้นหาด่วน เพื่อดูว่าตัวเกม Client ของคุณกำลังใช้ Release stream เวอร์ชันใดอยู่
4. ตรวจสอบให้แน่ใจว่าตั้งค่าเป็น `Cutting Edge (Experimental)`
5. คลิกปุ่มรีสตาร์ทที่จะปรากฏขึ้นด้านล่างของหน้าจอเพื่อใช้การตั้งค่าใหม่ (หากคุณมีการเปลี่ยน Release stream)
6. ไปที่เมนู `Options` อีกครั้งแล้วคลิกที่ `Open osu! folder`
7. ค้นหาโฟลเดอร์ `Logs` ในหน้าต่างที่เปิดขึ้นมา
8. เลือกไฟล์ Log ที่ต้องการ (ทีมงานจะระบุชื่อไฟล์ที่ต้องการให้ทราบ) แล้วแนบไฟล์นั้นไปกับข้อความแจ้งปัญหา (Ticket) หรือโพสต์ในฟอรัม

### Event Viewer {id=event-viewer}

**Event Viewer เป็นส่วนประกอบพื้นฐานของ Windows ที่ใช้ค้นหาบันทึกการค้าง (Crash logs) ในกรณีที่ตัวเกม osu! ไม่ได้แจ้งข้อมูลใดๆ ให้คุณทราบ ซึ่งบันทึกเหล่านี้จะช่วยระบุสาเหตุที่ทำให้เกมมีปัญหาได้**

เมื่อพบปัญหา osu! ค้างหรือดับไปเองโดยไม่มีหน้าต่างแจ้งเตือนจากตัวเกม Event Viewer คือที่เดียวที่คุณจะสามารถหาข้อมูลได้

หากทีมงานฝ่ายสนับสนุนขอให้คุณหา Crash log จาก Event Viewer ให้ทำตามขั้นตอนดังนี้:

1. หลังจากที่ osu! ค้างหรือดับไป ให้กดปุ่ม `Win` + `R` เพื่อเปิดหน้าต่าง Run
2. พิมพ์คำว่า `eventvwr` แล้วกด `Enter` เพื่อเปิดโปรแกรม Event Viewer
3. ใน Event Viewer ให้คลิกที่ `Windows Logs` ทางด้านซ้าย แล้วเลือก `Application`
4. ทางด้านขวา ให้คลิกที่ `Filter current log`
5. ในหน้าต่างตัวกรองที่เปิดขึ้นมา ตรวจสอบให้แน่ใจว่าได้ติ๊กถูกที่ช่อง `Error` แล้วคลิก `OK`
6. กด `Ctrl` + `F` แล้วพิมพ์คำว่า `osu!` ในช่องค้นหา เพื่อค้นหา Crash log ล่าสุดของ osu!
7. เข้าไปที่แท็บ `Details` จากนั้นคลิกเพื่อขยายส่วนของ `System` และ `Event Data`
8. คัดลอกข้อความจากส่วนนั้นแล้วนำไปวางในข้อความแจ้งปัญหาหรือโพสต์ในฟอรัม

### กราฟแสดงเวลาเฟรม (Frame Time Graph) {id=frame-time-graph}

**กราฟแสดงเวลาเฟรม (Frame Time Graph) เป็นฟีเจอร์ของ osu! ที่ช่วยให้เรารวบรวมข้อมูลที่เป็นประโยชน์เกี่ยวกับปัญหาด้านประสิทธิภาพที่คุณอาจกำลังประสบอยู่ได้**

เมื่อเผชิญกับปัญหาด้านประสิทธิภาพที่ตรวจสอบได้ยากใน osu! กราฟแสดงเวลาเฟรมจะเป็นเครื่องมือที่ดีในการช่วยวินิจฉัยและหาทางแก้ไขปัญหาที่เกิดขึ้น

หากทีมงานฝ่ายสนับสนุนขอให้คุณถ่ายวิดีโอหรือ Screenshot ขณะที่เปิดกราฟแสดงเวลาเฟรมอยู่ ให้ทำตามขั้นตอนดังนี้:

1. เปิด osu!
2. กด `Ctrl` + `F11` เพื่อเปิดหน้าต่างกราฟแสดงเวลาเฟรม (Frame Time Graph)
3. ทำการทดสอบหรือรอจนกว่าปัญหาด้านประสิทธิภาพจะเกิดขึ้น
4. ถ่ายภาพหน้าจอ (Screenshot) โดยการกด `Shift` + `F12` ตัวเกมจะอัปโหลดภาพไปยังเซิร์ฟเวอร์ของ osu! และเปิดลิงก์รูปภาพในเบราว์เซอร์ให้โดยอัตโนมัติ
5. คัดลอกลิงก์ที่ได้ไปวางในข้อความแจ้งปัญหาหรือโพสต์ในฟอรัม
6. ปิดกราฟแสดงเวลาเฟรมด้วยวิธีเดียวกันกับการเปิด คือกด `Ctrl` + `F11`

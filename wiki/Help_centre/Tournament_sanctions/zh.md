# 赛事处罚

*主页面： [帮助中心](/wiki/Help_centre)*

## 赛事封禁

赛事封禁（禁赛）会阻止玩家参加官方或官方支持的比赛，同时也限制了玩家在这些比赛中担任工作人员或提供其他协助的程度。

赛事封禁极少单独施加，但一般来讲，几乎所有通过标准申诉流程恢复[账号限制](/wiki/Help_centre/Account_restrictions)的玩家，都会被追加至少一年的赛事封禁。

由于比赛的高风险性质，赛事封禁的处理极为严肃，通常不会像账户限制那样获得宽大处理。

### 什么原因会导致临时禁赛？ {id=why-temporary}

任何在[官方支持的比赛](/wiki/Tournaments/Official_support)中发生的严重违反[社区规则](/wiki/Rules)的行为，都可能导致禁赛，即使该行为在竞技环境之外通常只会受到较轻的处罚。

比如一个现实中的例子是，在直播比赛中使用光标烟雾绘制或书写仇恨符号/侮辱性言论（例如纳粹标志）。

此外，所有在标准申诉条款下解除限制后回归游戏的玩家，也会受到至少一年的[临时禁赛](/wiki/Help_centre/Account_restrictions#reasons)，且该期限可由[账号支持团队](/wiki/People/Account_support_team)酌情延长。

### 什么原因会导致永久禁赛？ {id=why-permanent}

任何在官方支持的比赛中使用第三方作弊工具或其他方法以获得不公平优势的行为，都会导致永久禁赛。在某些情况下，尤其是严重滥用的情形，也可能由支持团队酌情给予此类处罚。

这包括以下情况：

- 如上所述，任何使用第三方作弊工具或程序获得优势的行为
- 使用多个账号报名参加同一比赛
- 试图通过使用另一个账号来规避禁言或其他处罚
- 在官方支持的比赛中共享账号
- 合谋获取或利用非公开信息来获取或提供优势（例如在其他队伍之前访问比赛的图池信息等）
- 滥用信任职位以获得或提供优势（操纵赛程、不公平地更改队伍构成，或无正当理由取消参赛者资格）
- 组织针对参赛者或组织者的、明显超出批评范畴的针对性攻击活动

### 禁赛期间我能做什么？ {id=while-banned}

你可以继续参加那些未申请官方支持或由活动组织者自行决定不进行资格审查的社区赛事。

经主办方同意，你仍可在获得官方支持的赛事中担任某些职务，例如直播、解说或美术设计。如果你希望承担其他任务，则相关赛事的主办方**必须**在其初次申请支持时提出豁免请求。根据违规原因和严重程度，并结合你的历史记录，[账号支持团队](/wiki/People/Account_support_team)可能会予以批准。

赛事组织者可以要求查询你的赛事封禁状态信息，无论其赛事是否获得官方支持。

### 常见禁赛原因及期限 {id=reasons}

| 禁赛原因 | 期限 | 备注 |
| :-- | :-- | :-- |
| 在比赛期间违反社区规则 | 1个月或更长 | 账户支持团队可酌情延长 |
| 不尊重工作人员或浪费时间 | 1个月或更长 |  |
| 在比赛中协助他人使用多账号或作弊 | 6个月或更长 |  |
| 在官方支持的比赛中选手行为不当 | 1-2年 |  |
| 根据标准申诉条款回归 | 1-2年 | 账户支持团队可酌情延长 |
| 在官方支持的比赛中使用多个账号或共享账号 | 无限期 | 同时将导致账户限制 |
| 在官方支持的比赛中使用第三方程序或其他作弊工具 | 无限期 | 同时将导致账户限制 |
| 利用非公开/机密的赛事信息获取或提供不公平优势 | 无限期 | 同时可能导致账户限制 |
| 对比赛组织者/工作人员/参与者的严重辱骂或骚扰 | 无限期 | 同时可能导致账户限制 |

## 主办资格观察期

当违反[官方锦标赛支持规则](/wiki/Tournaments/Official_support)的行为无法在比赛结束后通过沟通与主办方解决时，将对其施加主办资格观察期。处于观察期的用户，其下一次申请官方支持的赛事将受到锦标赛委员会在赛中和赛后的严格审查。通常只需通过邮件沟通即可解决的小问题，在观察期内则会导致观察期延长。在观察期内若发生严重违反官方支持规则的行为，将根据委员会的决定导致[主办资格封禁](#主办资格封禁)或更严重的处罚。

组织一场**零**次违反[官方支持](/wiki/Tournaments/Official_support)规则的赛事（包括那些通常只需通过邮件提醒主办方修正的小问题），即可解除主办资格观察期。此类问题的例子包括：

- 论坛帖中缺少工作人员名单
- 未公开资格赛 (Qualifier) 结果
- 未能更新公开的比赛列表（例如对阵图）
- 其他性质类似的基本错误

如果用户反复出现需要委员会通过邮件纠正的小问题，则将无限期保持观察状态。观察期内不努力提高赛事质量的用户，可能会受到委员会决定的更严厉处罚。主办资格观察期不会自动过期。

如果用户受到更严厉的处罚（如主办资格封禁），则在其处罚解除后，观察期也将一并解除。

## 主办资格封禁

主办资格封禁将禁止用户担任任何获得官方支持赛事的主要组织者或管理。但他们仍可担任不涉及组织工作的其他角色。

对于严重违反[官方支持规则](/wiki/Tournaments/Official_support)的情况，尤其是涉及竞技公正性时，锦标赛委员会保留跳过[主办资格观察期](#主办资格观察期)处罚、直接施加主办资格封禁的权利。

## 工作人员资格封禁

工作人员资格封禁将禁止用户在获得官方支持的赛事中担任任何工作人员职务，直播、解说或美术设计工作除外。

## 我可以对赛事处罚提出申诉吗？{id=appeal}

凡非通过[标准申诉条款](/wiki/Help_centre/Account_restrictions#appeal-granted)施加的赛事处罚，均设有 72 小时的申诉窗口期，在此期间可对处罚提出异议。申诉窗口期过后，除无限期赛事封禁外，其他处罚不可再申诉。

无限期赛事封禁可在首次施加后**至少两年**（24个月）提出申诉。提出申诉的用户需证明自己在此期间广泛、积极地参与了 osu! 社区或赛事活动，且行为记录清白。若申诉失败，用户需**至少等待一年**（12个月）方可再次申诉。

尽管可以对无限期赛事封禁进行申诉，但必须强调，任何成功的可能都取决于在**撰写申诉之外**，于更广泛的社区中做出显著贡献。成功申诉无限期赛事封禁的用户名单可[在此论坛主题中查看](https://osu.ppy.sh/community/forums/topics/1798871)。

在极少数情况下，[账户支持团队](/wiki/People/Account_support_team)可自行裁量复查个别案例，并撤销或重新施加适当的处罚，以确保与过往类似案例的处理标准保持一致。

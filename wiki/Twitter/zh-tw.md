# Twitter

osu! 有數個 Twitter 帳號，分別提供不同的功能。大部分的流量都很低，但能夠讓你在 osu! 生態圈外也能獲取新知。

## 服務

| 頭像 | 帳號名稱 | 說明 |
| :-: | :-: | :-- |
| ![osu!status avatar](img/osustatus.jpg) | [@osustatus](https://twitter.com/osustatus) | 在網站或 [Bancho](/wiki/Bancho_(server)) 發生問題時，傳送低流量通知。 |
| ![osu! support avatar](img/osu-support.png) | [@osusupport](https://twitter.com/osusupport) | 協助帳號和/或社群問題。 由[帳號支援團隊](/wiki/People/Account_support_team)經營。 |

## 社群

| 頭像 | 帳號名稱 | 說明 |
| :-: | :-: | :-- |
| ![osu! 頭貼](img/osugame.jpg) | [@osugame](https://twitter.com/osugame) | 官方新聞及公告來源。 |
| ![BanchoBoat 頭貼](img/banchoboat.jpg) | [@banchoboat](https://twitter.com/banchoboat) | 當遊戲伺服器崩潰時，提供一點笑料。 |
| ![NAT 頭貼](img/osu_nat.png) | [@osu_nat](https://twitter.com/osu_nat) | [NAT](/wiki/People/Nomination_Assessment_Team) 發布的新聞、公告，及社群問卷。(並非由 [osu! 團隊](/wiki/People/osu!_team)經營) |
| ![Tournament Committee 頭貼](img/osu_tcomm.jpg) | [@osu_tcomm](https://twitter.com/osu_tcomm) | 由[錦標賽委員會](/wiki/People/Tournament_Committee)發布的各類新聞和公告。(並非由 [osu! 團隊](/wiki/People/osu!_team)經營) |
| ![pp committee 頭貼](img/ppcommittee.png) | [@pp_committee](https://twitter.com/pp_committee) | 所有遊戲模式的難度計算公告，由[表現分委員會](/wiki/People/Performance_Points_Committee)經營。 |

# Grade

*สำหรับความหมายอื่น ดูที่ [Rank (disambiguation)](/wiki/Disambiguation/Rank)*

**Grade** คือชุดตัวอักษรที่แทนผลงานของผู้เล่นบน[บีตแมป](/wiki/Beatmap)หนึ่ง ๆ โดยจะแสดงคู่กับ[คะแนน](/wiki/Gameplay/Score)รวมบนหน้าผลลัพธ์ และปรากฏในรายการต่าง ๆ คู่กับคะแนน

จากต่ำไปสูง grade ที่เป็นไปได้คือ D, C, B, A, S และ SS โดย Silver S และ SS จะทำได้เฉพาะเมื่อใช้ม็อด [Hidden](/wiki/Gameplay/Game_modifier/Hidden), [Flashlight](/wiki/Gameplay/Game_modifier/Flashlight) หรือ [Fade In](/wiki/Gameplay/Game_modifier/Fade_In)

## osu!

| Grade | Condition |
| :-: | :-- |
| SS | accuracy 100% |
| S | มี 300 มากกว่า 90%, มี 50 ไม่เกิน 1%, และไม่มี miss |
| A | มี 300 มากกว่า 80% และไม่มี miss **หรือ** มี 300 มากกว่า 90% |
| B | มี 300 มากกว่า 70% และไม่มี miss **หรือ** มี 300 มากกว่า 80% |
| C | มี 300 มากกว่า 60% |
| D | กรณีอื่นทั้งหมด |

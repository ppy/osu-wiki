# เกรด (Grade)

*สำหรับความหมายอื่น ดูที่ [Rank (แก้ความกำกวม)](/wiki/Disambiguation/Rank)*

**เกรด (Grades)** คือตัวอักษรที่แสดงถึงประสิทธิภาพการเล่นของผู้เล่นใน [บีทแมพ (Beatmap)](/wiki/Beatmap) นั้นๆ โดยเกรดจะปรากฏควบคู่ไปกับ [คะแนนรวม (Score)](/wiki/Gameplay/Score) ในหน้าสรุปผลการเล่น และแสดงในรายการตารางคะแนนต่างๆ

เกรดที่เป็นไปได้เรียงจากต่ำสุดไปสูงสุดคือ D, C, B, A, S และ SS สำหรับเกรด S และ SS สีเงินจะได้รับก็ต่อเมื่อมีการเปิดใช้งานตัวปรับแต่งเกม (Game modifiers) ประเภท [Hidden](/wiki/Gameplay/Game_modifier/Hidden), [Flashlight](/wiki/Gameplay/Game_modifier/Flashlight) หรือ [Fade In](/wiki/Gameplay/Game_modifier/Fade_In) เท่านั้น

## osu!

| เกรด | เงื่อนไข |
| :-: | :-- |
| SS | ความแม่นยำ (Accuracy) 100% |
| S | กดได้ 300 มากกว่า 90%, กดได้ 50 ไม่เกิน 1% และไม่มีการกดพลาด (Miss) |
| A | กดได้ 300 มากกว่า 80% และไม่พลาดเลย **หรือ** กดได้ 300 มากกว่า 90% |
| B | กดได้ 300 มากกว่า 70% และไม่พลาดเลย **หรือ** กดได้ 300 มากกว่า 80% |
| C | กดได้ 300 มากกว่า 60% |
| D | อื่นๆ นอกเหนือจากนี้ |

# ระบบการคิดคะแนนของ osu!catch (osu!catch scoring system)

*ดูเพิ่มเติม: [ระบบการตัดสินของ osu!catch](/wiki/Gameplay/Judgement/osu!catch)*

การคิดคะแนนในโหมด osu!catch ใช้ตัวคูณคะแนนแบบเดียวกับโหมด [osu!](/wiki/Game_mode/osu!) อย่างไรก็ตาม วัตถุที่จะนำมาเพิ่มคะแนนนั้นมีความแตกต่างกัน:

- **ผลไม้ขนาดปกติ (Fruit):** ให้คะแนน 300 คะแนน คูณด้วยตัวคูณคะแนน
- **หยดน้ำขนาดใหญ่ (Large juice drop / Slider tick):** ให้คะแนน 100 คะแนน
- **หยดน้ำขนาดเล็กที่สุด (Droplet / Slider trail):** ให้คะแนน 10 คะแนน
- **กล้วย (Banana):** ให้คะแนนคงที่ 1,100 คะแนนต่อใบ ไม่ว่าจะเปิด Mod ใดหรือมีตัวคูณคอมโบเท่าไหร่ก็ตาม

ในโหมดนี้ คอมโบจะไม่หลุดหากรับหยดน้ำขนาดเล็กพลาด (ในตารางคะแนนเซิร์ฟเวอร์จะระบุว่า *Miss Droplet*) แต่จะทำให้ความแม่นยำ (Accuracy) ลดลงและเสียโอกาสในการเก็บคะแนนจากหยดน้ำนั้น

**สูตรคำนวณคะแนน:**
`Score = Hit Value + [Hit Value * ((Combo multiplier * Difficulty multiplier * Mod multiplier) / 25)]`

| คำศัพท์ | ความหมาย |
| :-: | :-- |
| **Hit Value** | คะแนนจากการกด (10, 100 หรือ 300) รวมถึงคะแนนโบนัสจากกล้วย |
| **Combo multiplier** | (จำนวนคอมโบก่อนการกดนี้ - 1) หรือ 0 แล้วแต่ว่าค่าใดจะสูงกว่า |
| **Difficulty multiplier** | ค่าคงที่ระดับความยากของบีทแมพ |
| **Mod multiplier** | ตัวคูณคะแนนของ Mod ที่เลือกใช้งาน |

**หมายเหตุ:** มีความแตกต่างระหว่างการให้คะแนนของ osu! และ osu!catch ดังนี้:

- เมื่อหมุน Spinner (ห่ากล้วย) จนจบ จะไม่มีการให้คะแนน 300 และไม่เพิ่มจำนวนคอมโบ
- จุดจังหวะที่ลูกศรย้อนกลับ (Reverse slider ticks) จะถูกนับเป็นคะแนนเต็มของการกดหนึ่งครั้ง
  - ในขณะที่โหมด osu! การเก็บลูกศรย้อนกลับจะให้เพียง 30 คะแนนเท่านั้น

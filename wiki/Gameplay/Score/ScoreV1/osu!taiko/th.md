# ระบบคิดคะแนนของ osu!taiko

*ดูเพิ่มเติม: [ระบบการตัดสินของ osu!taiko](/wiki/Gameplay/Judgement/osu!taiko)*

แต่ละ GREAT (ไม่นับโบนัส 1.2x จาก Kiai Time) จะมีค่าเท่ากับ `300 + RoundDown(Combo / 10)` คูณด้วย `n` สูงสุดไม่เกิน `300+10n` คะแนน โดย `n` จะขึ้นอยู่กับระดับความยากของเพลง

แต่ละ *GOOD* จะให้คะแนนครึ่งหนึ่งของ *GREAT* ส่วน *MISS/BAD* จะให้ 0 คะแนน

โน้ตขนาดใหญ่จะให้คะแนนสองเท่าเมื่อ *ตีสำเร็จ* ซึ่งต่างจาก *Taiko no Tatsujin* ตรงที่ต้องตีกลองซ้ายและขวาพร้อมกันอย่างแม่นยำมากเพื่อให้ถือว่าตีโน้ตขนาดใหญ่สำเร็จ

สำหรับโน้ตยาวสีเหลือง การตีแต่ละครั้งจะให้ 300 คะแนนในโน้ตขนาดเล็ก และ 600 คะแนนในโน้ตขนาดใหญ่

สำหรับ shaker notes การเขย่าแต่ละครั้งจะให้ 300 คะแนน และการจบ shaker จะให้คะแนนมากเป็นสองเท่าของ GREAT ณ คอมโบปัจจุบัน

ค่า `n` ทั่วไปสำหรับ difficulty ระดับ 4.5-5 ดาวในระบบ star rating แบบเก่า 5 ดาว คือ 80 ซึ่งทำให้คะแนนสูงสุดอยู่ที่ 1,100/2,200 เมื่อคอมโบถึง 100 และหลังจากนั้น
สำหรับ difficulty ระดับ 4-4.5 ดาว `n` จะเท่ากับ 64 ทำให้คะแนนสูงสุดต่อการกดอยู่ที่ 940/1,880
ในกรณีที่ยากที่สุด `n` จะเท่ากับ 96 ทำให้คะแนนสูงสุดต่อการกดอยู่ที่ 1,260/2,520
สำหรับ difficulty ที่ง่ายกว่านี้จะมีค่า `n` ต่ำลงไปอีก

ต่างจาก [osu!](/wiki/Game_mode/osu!)/[osu!catch](/wiki/Game_mode/osu!catch) การ miss ใน osu!taiko จะไม่ทำให้คะแนนต่างจากคะแนนสูงสุดที่เป็นไปได้แบบ *รุนแรง* นัก (ใน osu!/osu!catch ความเสียหายของคะแนนจากการ miss จะยิ่ง *กว้างขึ้น* เมื่อคอมโบสูงสุดมากขึ้น โดยเฉพาะถ้าคอมโบขาดช่วงประมาณครึ่งหนึ่งของคอมโบสูงสุด)
แทนที่จะเป็นแบบนั้น คะแนนคงที่จำนวน `n` ตามที่อธิบายด้านบนจะถูกหักต่อการ miss หนึ่งครั้ง หากแต่ละ miss ห่างกันมากกว่า 100 คอมโบ
เมื่อมี shaker และโน้ตขนาดใหญ่ร่วมด้วย คะแนนที่เสียไปจะมากขึ้น

ตัวอย่างเช่น ใน difficulty ทั่วไป (`n` = 80) หากคอมโบขาดกลางเพลงโดยไม่คิดโน้ตขนาดใหญ่และสปินเนอร์ จะทำให้เสียคะแนนสูงสุดได้ 44,000 คะแนน เพื่อปั้นคอมโบกลับไปถึง 100 โดยคิดว่าได้ GREAT ทั้งหมด

Kiai Time ส่งผลต่อคะแนนใน osu!taiko ต่างจากโหมดเกมอื่น เพราะมันอ้างอิงถึง *"Go-Go Time"* ใน *Taiko no Tatsujin*
ขณะ Kiai Time ทำงาน กลองมุมซ้ายบนจะเปลี่ยนแอนิเมชัน เพลย์ฟีลด์จะมีพื้นหลังแบบไล่สี และบริเวณกดจะมีกราฟิกไฟล้อมรอบ
นอกจากนี้ โน้ตที่ตีทั้งหมดจะได้รับตัวคูณคะแนน 1.2x รวมถึงโน้ตยาวสีเหลือง (ดรัมโรล) ด้วย ยกเว้นการตี shaker โดยการตีครั้งสุดท้ายจะยังถูกคูณอยู่

สรุปสั้น ๆ: `Score = {ScoreValue + [min(RoundDown(Combo / 10), 10) * RoundDown(taiko score multiplier * raw mod multiplier)]} * Kiai Time`

| คำศัพท์ | ความหมาย |
| :-: | :-- |
| **ScoreValue** | ค่าคะแนนที่ได้รับจากการกด (300/600, 150/300 หรือ 0/0) |
| **Combo** | (คอมโบก่อนการกดครั้งนี้ - 1) หรือ 0 โดยใช้ค่าที่สูงกว่า |
| **taiko score multiplier** | \[ขึ้นอยู่กับระดับความยาก\] ค่าที่เป็นไปได้คือ: 32, 48, 64, 80, 96 |
| **raw mod multiplier** | ตัวคูณ *ดิบ* ของม็อดที่เลือกใช้ (ตัวคูณที่แสดงโดยทั่วไปมักถูกปัดขึ้น) |
| **RoundDown** | ปัดค่าลงเป็นจำนวนเต็ม โดยตัดค่าทศนิยมทั้งหมดออก |
| **min(x, y)** | เลือกค่าที่ต่ำกว่าระหว่าง *x* หรือ *y* เสมอ |
| **Kiai Time** | หาก Kiai Time ทำงานอยู่ ค่านี้จะเป็น 1.2 ไม่เช่นนั้นจะเป็น 1.0 |

ข้อยกเว้น:

- การตีดรัมโรลสำเร็จแต่ละครั้งจะให้ค่า GREAT คงที่ (300/600 สำหรับดรัมโรลขนาดเล็ก/ใหญ่ตามลำดับ) พร้อมโบนัส Kiai Time เท่านั้น
- การตี denden หรือการเขย่า shaker แต่ละครั้งจะให้ค่า GREAT คงที่ (300) โดยไม่มีโบนัส Kiai Time ยกเว้นการตี denden ครั้งสุดท้ายหรือการเขย่า shaker ครั้งสุดท้าย ซึ่งจะให้ค่า GREAT ขนาดใหญ่ (600) พร้อมตัวคูณคอมโบปัจจุบัน

# ระบบการคิดคะแนนของ osu!taiko (osu!taiko scoring system)

*ดูเพิ่มเติม: [ระบบการตัดสินของ osu!taiko](/wiki/Gameplay/Judgement/osu!taiko)*

การกดได้ระดับ GREAT แต่ละครั้ง (ไม่รวมโบนัส 1.2 เท่าจาก Kiai Time) จะมีค่าเท่ากับ `300 + RoundDown(Combo / 10)` คูณกับค่า `n` โดยมีเพดานสูงสุดที่ `300 + 10n` คะแนน ซึ่งค่า `n` นี้จะขึ้นอยู่กับการตั้งค่าระดับความยากของเพลง

- การกดได้ระดับ **GOOD** จะได้คะแนนครึ่งหนึ่งของระดับ **GREAT**
- การกดได้ระดับ **MISS/BAD** จะไม่ได้คะแนนเลย (0 คะแนน)

สำหรับการกดโน้ต **ขนาดใหญ่** สำเร็จ จะได้รับคะแนนเป็นสองเท่า (ต่างจากเกม *Taiko no Tatsujin* ตรงที่ใน osu!taiko คุณต้องกดหน้ากลองหรือขอบกลองทั้งซ้ายและขวาพร้อมกันอย่างแม่นยำมากเพื่อให้ได้รับคะแนนโน้ตขนาดใหญ่)

สำหรับแถบสีเหลืองยาว (Drumroll):
- โน้ตขนาดปกติ: ได้รับ 300 คะแนนต่อการกดถูกจุดจังหวะ (Tick)
- โน้ตขนาดใหญ่: ได้รับ 600 คะแนนต่อการกดถูกจุดจังหวะ

สำหรับ Shaker (Denden):
- การเขย่าแต่ละครั้งได้รับ 300 คะแนน
- เมื่อเขย่าจนเสร็จสมบูรณ์ จะได้รับคะแนนเป็นสองเท่าของคะแนนระดับ GREAT ณ คอมโบปัจจุบัน

ค่ามาตรฐานของ `n` (สำหรับแมพความยาก 4.5–5 ดาวในระบบเก่า) คือ 80 ซึ่งจะให้คะแนนสูงสุดที่ 1,100 คะแนน (2,200 สำหรับโน้ตใหญ่) เมื่อทำคอมโบได้ตั้งแต่ 100 ขึ้นไป
- สำหรับแมพความยาก 4–4.5 ดาว ค่า `n` คือ 64 (คะแนนสูงสุดต่อครั้งคือ 940/1,880)
- สำหรับแมพที่ยากที่สุด ค่า `n` คือ 96 (คะแนนสูงสุดต่อครั้งคือ 1,260/2,520)
- แมพที่ง่ายกว่านี้จะมีค่า `n` ที่ต่ำลงไปตามลำดับ

ต่างจากโหมด [osu!](/wiki/Game_mode/osu!) หรือ [osu!catch](/wiki/Game_mode/osu!catch) ตรงที่การกดพลาดใน osu!taiko จะไม่ทำให้คะแนนรวมสุดท้ายต่างจากคะแนนสูงสุดที่เป็นไปได้มากนัก (ในโหมดอื่น ความเสียหายจากการพลาดคอมโบจะยิ่งสูงขึ้นตามคอมโบสูงสุดของแมพนั้น) ในโหมด Taiko คะแนนที่หายไปจากการพลาดแต่ละครั้งจะคงที่อยู่ที่ค่า `n` (หากการพลาดแต่ละครั้งห่างกันเกิน 100 คอมโบ) แต่ถ้าในแมพนั้นมี Shaker หรือโน้ตใหญ่จำนวนมาก คะแนนที่เสียไปอาจจะสูงขึ้น

ตัวอย่างเช่น ในแมพความยากปกติ (n=80) หากคอมโบหลุดกลางเพลงโดยไม่คิดเรื่องโน้ตใหญ่หรือ Spinner คะแนนที่หายไปสูงสุดจะอยู่ที่ประมาณ 44,000 คะแนน (ซึ่งเป็นคะแนนที่ต้องใช้ในการปั้นคอมโบกลับไปให้ถึง 100 เพื่อให้ได้โบนัสคะแนนสูงสุดอีกครั้ง)

นอกจากนี้ *Kiai Time* ในโหมดนี้มีผลต่อคะแนนโดยตรง (อิงตามระบบ *"Go-Go Time"* ของเกมต้นฉบับ) เมื่อช่วง Kiai ทำงาน มาสคอตที่มุมซ้ายบนจะเปลี่ยนท่าทาง พื้นหลังสนามเล่นจะมีสีไล่ระดับ และบริเวณที่กดจะมีเอฟเฟกต์ไฟ การกดโน้ตทุกประเภทในช่วงเวลานี้จะได้รับตัวคูณคะแนน 1.2 เท่า รวมถึง Drumroll ด้วย ยกเว้นการเขย่า Shaker (แต่การเขย่าครั้งสุดท้ายจะยังได้รับตัวคูณ)

**สรุปสูตรคำนวณ:**
`Score = {ScoreValue + [min(RoundDown(Combo / 10), 10) * RoundDown(taiko score multiplier * raw mod multiplier)]} * Kiai Time`

| คำศัพท์ | ความหมาย |
| :-: | :-- |
| **ScoreValue** | คะแนนพื้นฐานจากการกด (300/600, 150/300 หรือ 0/0) |
| **Combo** | (จำนวนคอมโบก่อนการกดนี้ - 1) หรือ 0 แล้วแต่ว่าค่าใดจะสูงกว่า |
| **taiko score multiplier** | [ขึ้นอยู่กับระดับความยาก] ค่าที่เป็นไปได้คือ: 32, 48, 64, 80, 96 |
| **raw mod multiplier** | ตัวคูณคะแนนที่แท้จริงของ Mod ที่เลือก (ตัวเลขที่แสดงในเกมมักถูกปัดเศษขึ้น) |
| **RoundDown** | การปัดเศษทศนิยมลงให้เป็นจำนวนเต็ม |
| **min(x, y)** | เลือกค่าที่ต่ำที่สุดระหว่าง x หรือ y |
| **Kiai Time** | หากอยู่ในช่วง Kiai จะมีค่าเป็น 1.2 หากไม่ใช่จะมีค่าเป็น 1.0 |

ข้อยกเว้น:-
- การตี Drumroll แต่ละครั้งที่สำเร็จจะได้คะแนน GREAT แบบคงที่ (300/600 ตามขนาด) โดยจะได้รับโบนัสเฉพาะ Kiai Time เท่านั้น
- การเขย่า Shaker แต่ละครั้งจะได้คะแนน GREAT แบบคงที่ (300) โดยไม่ได้รับโบนัส Kiai Time ยกเว้นการเขย่าครั้งสุดท้ายที่จะได้คะแนน 600 และได้รับตัวคูณคอมโบตามปกติ

# Общие очки

*У этого термина есть и другие значения, см. [Score (значения)](/wiki/Disambiguation/Score).*

**Общие очки** (англ. *total score*, жарг. *общий скор*) — это сумма всех очков, полученная за все пройденные карты, включая неудачные попытки.

Если игрок впервые успешно проходит ранкнутую карту, то оба показателя — и [ранкнутые](/wiki/Gameplay/Score/Ranked_score), и общие очки, — увеличиваются на число набранных очков. В дальнейшем общие очки растут с каждым новым результатом, а ранкнутые очки — только в том случае, если игрок улучшил свой рекорд.

## Уровень

Уровень игрока зависит исключительно от его **общего числа очков**, которое для уровня `n` определяется следующей функцией:

```
score(n) = 5,000 / 3 * (4n^3 - 3n^2 - n) + 1.25 * 1.8^(n - 60) для n <= 100
score(n) = 26,931,190,827 + 99,999,999,999 * (n - 100)         для n > 100
```

Обратите внимание, что из-за округления, используемого в игре, общее число очков для уровней ниже 100 не совсем соответствуют формуле. Это связано с тем, что в игре хранится заранее рассчитанный список разницы в очках между уровнями, где каждое значение округляется до ближайшего целого числа. Например, восьмидесятое значение в этом списке, `score(81) - score(80) = 128,927,482.36216`, округляется вниз до 128,927,482. На уровне 100 общая ошибка составляет +1,629 очков, поэтому `score(100) = 26,931,190,828.629` не совпадает со значением 26,931,190,827, рассчитанным по формуле.

Прогресс до следующего уровня отображается с помощью полосы рядом с уровнем игрока.

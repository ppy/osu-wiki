# คะแนนรวม (Total score)

*สำหรับความหมายอื่น ดูที่ [Score (แก้ความกำกวม)](/wiki/Disambiguation/Score)*

**คะแนนรวม (total score)** คือผลรวมของคะแนนทั้งหมดจากทุกบีตแมป รวมถึงรอบที่เล่นไม่ผ่านด้วย

หากผู้เล่นทำสถิติบนบีตแมป ranked เป็นครั้งแรก ทั้ง [ranked score](/wiki/Gameplay/Score/Ranked_score) และ total score ของผู้เล่นจะเพิ่มขึ้นตามคะแนนนั้น หลังจากนั้น total score จะเพิ่มขึ้นทุกครั้งที่ส่งผลการเล่นใหม่ ส่วน ranked score จะเพิ่มขึ้นเฉพาะเมื่อผู้เล่นทำสถิติของตัวเองให้ดีขึ้นเท่านั้น

## เลเวล {#level}

เลเวลของผู้เล่นอิงจาก **total score** ของผู้เล่นเพียงอย่างเดียว คะแนนที่ต้องใช้สำหรับแต่ละเลเวลกำหนดโดยฟังก์ชันต่อไปนี้:

```
score(n) = 5,000 / 3 * (4n^3 - 3n^2 - n) + 1.25 * 1.8^(n - 60) if n <= 100
score(n) = 26,931,190,827 + 99,999,999,999 * (n - 100)         if n > 100
```

โปรดทราบว่าเนื่องจากเกมใช้เทคนิคการปัดเศษ ค่าของเลเวลจนถึง 100 จึงไม่ตรงกับสูตรแบบเป๊ะ ๆ สาเหตุคือเกมเก็บรายการค่าคะแนนที่ต่างกันระหว่างแต่ละเลเวลไว้ล่วงหน้า และแต่ละรายการในลิสต์นี้จะถูกปัดเป็นจำนวนเต็มที่ใกล้ที่สุด ตัวอย่างเช่น รายการลำดับที่ 80 ในลิสต์คือ `score(81) - score(80) = 128,927,482.36216` ซึ่งถูกปัดเป็น 128,927,482 เมื่อถึงเลเวล 100 ความคลาดเคลื่อนรวมจะเป็น +1.629 คะแนน นี่จึงเป็นเหตุผลที่ `score(100) = 26,931,190,828.629` ไม่ตรงกับ 26,931,190,827 ในส่วนที่สองของฟังก์ชัน

ความคืบหน้าไปยังเลเวลถัดไปจะแสดงด้วยแถบข้างเลเวลของผู้เล่น

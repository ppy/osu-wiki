# คะแนนรวม (Total score)

*สำหรับความหมายอื่น ดูที่ [Score (แก้ความกำกวม)](/wiki/Disambiguation/Score)*

**คะแนนรวม (Total score)** คือผลรวมของคะแนนทั้งหมดที่ผู้เล่นทำได้จากทุกบีทแมพ รวมถึงการเล่นที่ไม่ผ่าน (Failed) ด้วย

หากผู้เล่นสร้างสถิติในบีทแมพ Ranked เป็นครั้งแรก ทั้ง [คะแนน Ranked (Ranked score)](/wiki/Gameplay/Score/Ranked_score) และคะแนนรวมจะเพิ่มขึ้นตามจำนวนนั้น หลังจากนั้น คะแนนรวมจะเพิ่มขึ้นทุกครั้งที่มีการส่งผลการเล่นใหม่เข้าระบบ ในขณะที่คะแนน Ranked จะเพิ่มขึ้นเฉพาะเมื่อผู้เล่นสามารถทำคะแนนทำลายสถิติเดิมของตนเองได้เท่านั้น

## เลเวล (Level)

เลเวลของผู้เล่นจะถูกคำนวณจาก **คะแนนรวม** เพียงอย่างเดียวเท่านั้น โดยข้อกำหนดคะแนนที่ต้องใช้ในแต่ละเลเวลจะเป็นไปตามฟังก์ชันทางคณิตศาสตร์ดังนี้:

```
score(n) = 5,000 / 3 * (4n^3 - 3n^2 - n) + 1.25 * 1.8^(n - 60) สำหรับ n <= 100
score(n) = 26,931,190,827 + 99,999,999,999 * (n - 100)         สำหรับ n > 100
```

โปรดทราบว่าเนื่องจากเทคนิคการปัดเศษที่ตัวเกมใช้ ค่าคะแนนจริงสำหรับเลเวลที่ต่ำกว่า 100 อาจจะไม่ตรงกับสูตรเป๊ะๆ เนื่องจากตัวเกมมีการเก็บรายการส่วนต่างของคะแนนระหว่างเลเวลที่คำนวณไว้ล่วงหน้า และแต่ละรายการจะถูกปัดเศษเป็นจำนวนเต็มที่ใกล้เคียงที่สุด ตัวอย่างเช่น รายการที่ 80 คือ `score(81) - score(80) = 128,927,482.36216` ซึ่งถูกปัดเป็น 128,927,482 เมื่อถึงเลเวล 100 ความคลาดเคลื่อนรวมจะอยู่ที่ +1.629 คะแนน ซึ่งเป็นสาเหตุที่ทำให้ `score(100) = 26,931,190,828.629` ไม่ตรงกับค่าคงที่ 26,931,190,827 ในส่วนที่สองของฟังก์ชัน

ความคืบหน้าไปยังเลเวลถัดไปจะแสดงผลเป็นแถบ (Progress bar) ที่อยู่ข้างๆ ตัวเลขเลเวลของผู้เล่น

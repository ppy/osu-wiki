# Keyboard

**Keyboard** คือ input device ที่ใช้เล่น osu! กันบ่อยที่สุด โดยค่าเริ่มต้น ปุ่ม `Z` และ `X` ถูกตั้งไว้สำหรับการคลิก แต่สามารถเปลี่ยนเป็นปุ่มใดก็ได้ตามต้องการใน options ของ osu!

## Keypad

**Keypad** คือคีย์บอร์ดขนาดเล็กที่มีจำนวนปุ่มจำกัด Keypad เป็นที่นิยมใน osu! เพราะมีขนาดเล็กและมีจำนวนปุ่มพอดี โดยมักมี 2 หรือ 4 ปุ่ม

# เมาส์ (Mouse)

**เมาส์ (Mouse)** เป็นอุปกรณ์ที่ได้รับความนิยมอย่างมากในการเล่น osu! โดยผู้เล่นสามารถใช้เมาส์เพื่อเล็งตำแหน่งและคลิกโน้ต ปุ่มกลางของเมาส์ถูกตั้งค่าไว้สำหรับหยุดพักเกม (Pause) (สามารถปิดได้ในเมนูการตั้งค่า) และล้อเมาส์ (Scroll wheel) สามารถใช้เพื่อปรับระดับเสียงได้

## ตัวเลือกที่เกี่ยวข้องกับเมาส์

### Sensitivity (ความไวเมาส์)

ตัวเลือกนี้ช่วยให้คุณสามารถกำหนดความเร็วของเมาส์ที่ต้องการได้ โดยค่าเริ่มต้นจะอยู่ที่ 1x

### Raw input (รับข้อมูลโดยตรง)

Raw input จะทำการข้ามการประมวลผลการเร่งความเร็วเมาส์ (Mouse acceleration) ของ Windows เพื่อให้การเคลื่อนที่ของเมาส์มีความแม่นยำสูงสุด จำเป็นต้องเปิดใช้งานตัวเลือกนี้หากต้องการตั้งค่า Sensitivity ที่นอกเหนือจากค่าปกติบน Windows 7 ขึ้นไป

### Confine mouse cursor (จำกัดเคอร์เซอร์เมาส์)

ตัวเลือกนี้จะช่วยให้เคอร์เซอร์เมาส์ไม่หลุดออกจากหน้าต่างของ osu! โดยค่าเริ่มต้นจะตั้งไว้ที่ `Only when fullscreen` (เฉพาะเมื่อเล่นเต็มหน้าจอ) ตัวเลือกที่มีให้เลือกได้แก่:

- `Never`: ไม่จำกัดเคอร์เซอร์ไว้ในหน้าต่างเกม
- `Only when fullscreen`: จำกัดเคอร์เซอร์ไว้ในหน้าต่างเกมเฉพาะเมื่อเล่นแบบเต็มหน้าจอ
- `Always`: จำกัดเคอร์เซอร์ไว้ในหน้าต่างเกมตลอดเวลา

### Disable mouse wheel in play mode (ปิดใช้งานล้อเมาส์ในระหว่างเล่น)

ในขณะเล่น คุณสามารถใช้ล้อเมาส์เพื่อปรับระดับเสียงหรือหยุดพักเกมได้ การเปิดใช้งานตัวเลือกนี้จะทำการปิดฟังก์ชันดังกล่าวในระหว่างที่คุณกำลังเล่นแมพอยู่

### Disable mouse buttons in play mode (ปิดใช้งานปุ่มเมาส์ในระหว่างเล่น)

ตัวเลือกนี้จะปิดการทำงานของปุ่มคลิกทั้งหมดบนเมาส์ในขณะเล่น ซึ่งมีประโยชน์มากสำหรับผู้เล่นที่ใช้คีย์บอร์ดในการกดโน้ตเพื่อป้องกันการเผลอคลิกโดยไม่ตั้งใจ

### Cursor ripples (เอฟเฟกต์คลื่นน้ำที่เคอร์เซอร์)

เมื่อเปิดใช้งาน จะมีเอฟเฟกต์คลื่นน้ำกระจายออกมาจากเคอร์เซอร์ทุกครั้งที่มีการคลิก

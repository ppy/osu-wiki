# Mouse

**Mouse** เป็นวิธีเล่น osu! ที่ได้รับความนิยมมาก ผู้เล่นสามารถใช้เมาส์เพื่อ aim และคลิกได้ ปุ่มกลางของเมาส์ถูกตั้งไว้สำหรับหยุดเกมชั่วคราว โดยปิดได้ใน options และล้อเลื่อนเมาส์สามารถใช้ปรับระดับเสียงได้

## ตัวเลือกที่เกี่ยวกับเมาส์

### Sensitivity

ตัวเลือกนี้ให้ผู้เล่นตั้งค่าความเร็วเมาส์เองได้ และค่าเริ่มต้นคือ 1x

### Raw input

Raw input จะข้าม mouse acceleration ของ Windows และให้การเคลื่อนที่ของเมาส์ที่แม่นยำที่สุด จำเป็นต้องเปิด raw input หากต้องการตั้งค่า sensitivity ของเมาส์เองบน Windows 7 ขึ้นไป

### Confine mouse cursor

ตัวเลือกนี้ทำให้เคอร์เซอร์เมาส์อยู่ภายในหน้าต่าง osu! และค่าเริ่มต้นคือ `Only when fullscreen` ตัวเลือกที่มี ได้แก่:

- `Never`: เมาส์จะไม่ถูกจำกัดให้อยู่ในหน้าต่าง osu!
- `Only when fullscreen`: เมาส์จะอยู่ในหน้าต่าง osu! เมื่ออยู่ในโหมด fullscreen
- `Always`: เมาส์จะอยู่ในหน้าต่าง osu! เสมอ

### Disable mouse wheel in play mode

ระหว่างเล่น ผู้เล่นสามารถใช้ล้อเมาส์เพื่อปรับระดับเสียงและหยุดเกมชั่วคราวได้ การเปิดตัวเลือกนี้จะปิดฟังก์ชันดังกล่าวระหว่างเล่น

### Disable mouse buttons in play mode

ตัวเลือกนี้จะปิดปุ่มเมาส์ทั้งหมดระหว่างเล่น ซึ่งอาจมีประโยชน์สำหรับผู้เล่นที่ใช้คีย์บอร์ดในการคลิก

### Cursor ripples

เคอร์เซอร์จะแผ่คลื่นออกด้านนอกเมื่อคลิก

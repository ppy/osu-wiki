# 鼠标

**鼠标**是游玩 osu! 的热门方式，同时可以瞄准及点击。鼠标中键用来暂停游戏（可以在设置中停用），滚轮用来调整音量。

## 鼠标相关选项

### 灵敏度

这个选项可以让您自定义您的鼠标移动速度，预设为 1x。

### 原始输入

原始输入会绕过 Windows 内置的鼠标加速，并提供最准确的鼠标位置。在 Windows 7 及以上的版本中，必须启用原始输入才能自定义鼠标灵敏度。

### 限制光标

这个选项让光标只能在 osu! 的窗口中移动，预设为`只在全屏幕限制`。可用的选项有：

- `永不`: 不限制光标位置。
- `只在全屏幕限制`: 只在全屏幕时才限制光标位置。
- `总是`: 光标绝不会超出 osu! 的窗口。

### 游戏中禁用鼠标滚轮

游戏进行时，可以使用鼠标滚轮来调整音量大小及暂停游戏，启用这个选项会禁用这些功能。

### 游戏中禁用鼠标按键

这个选项会在游戏时禁用所有鼠标按键，对用键盘来点击的玩家来说或许会有用。

### 光标涟漪

鼠标在点击时会有一圈涟漪扩散出去。

# Drum controller

![](img/taiko-drum.jpg "Taiko drum")

**Drum controller** คือ input device ที่ออกแบบมาให้ใช้กับเกมกลองสไตล์ Taiko โดยส่วนมากจะมีพื้นที่อินพุต 4 ส่วน ได้แก่ กลางกลองขวา กลางกลองซ้าย ขอบกลองขวา และขอบกลองซ้าย ซึ่งแต่ละอินพุตหรือปุ่มจะทำงานเมื่อใช้ไม้กลองตีลงบนพื้นที่ที่ตรงกัน

## อินพุต

ผู้ที่มี Taiko drum controller สำหรับ [PlayStation 4 (PS4)](https://en.wikipedia.org/wiki/PlayStation_4) หรือ [Nintendo Switch](https://en.wikipedia.org/wiki/Nintendo_Switch) ต้องเสียบคอนโทรลเลอร์เข้าคอมพิวเตอร์ผ่าน USB แล้ว bind ปุ่มที่ถูกต้องใน settings ของ osu! ส่วนผู้ที่มีคอนโทรลเลอร์ Tatacon ของ [Wii](https://en.wikipedia.org/wiki/Wii) ต้อง pair คอนโทรลเลอร์กับคอมพิวเตอร์ก่อน แล้วจึงเปิดตัวเลือก Wiimote/Tatacon support ใน settings ของ osu! drum controller จากผู้ผลิตรายอื่นส่วนใหญ่ก็ใช้กับ osu! ได้เช่นกัน แต่การรองรับอาจแตกต่างกันไป

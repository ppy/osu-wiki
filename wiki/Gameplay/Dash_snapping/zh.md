# 跳对齐

在 [osu!catch](/wiki/Game_mode/osu!catch) 中，所有[白果跳](/wiki/Gameplay/Dash)与[红果跳](/wiki/Gameplay/Hyperdash)均可归类为两种不同的对齐方式：基本对齐与高度对齐。

## 基本对齐

**基本对齐**这个词用来形容[活动物件](/wiki/Beatmapping/Active_object)间，间隔时间不小于做出相应跳动作所需时间的两倍的[白果跳](/wiki/Gameplay/Dash)或[红果跳](/wiki/Gameplay/Hyperdash)。例如在 Platter 难度中，物件间间隔 250 毫秒的红果跳可被定为基本对齐的红果跳。

## 高度对齐

**高度对齐**这个词用来形容不满足基本对齐跳要求（也就是说，物件间隔时间小于基本对齐时间的下限）的[白果跳](/wiki/Gameplay/Dash)或[红果跳](/wiki/Gameplay/Hyperdash)。

## 对齐参考表

| 难度 | 基本对齐白果跳 | 高度对齐白果跳 | 基本对齐红果跳 | 高度对齐红果跳 |
| :-- | :-- | :-- | :-- | :-- |
| **Cup** | - | - | - | - |
| **Salad** | 250 ms 及以上 | 125-249 ms | - | - |
| **Platter** | 125 ms 及以上 | 62-124 ms | 250 ms 及以上 | 125-249 ms |
| **Rain** | 125 ms 及以上 | 62-124 ms | 125 ms 及以上 | 62-124 ms |
| **Overdose** | - | - | - | - |

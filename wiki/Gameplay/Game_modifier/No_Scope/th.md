*สำหรับรายชื่อ Mod ของ [lazer](/wiki/Client/Release_stream/Lazer) ทั้งหมด ดูที่: [Game modifier (lazer)](/wiki/Gameplay/Game_modifier_(lazer))*

**No Scope** mod จะค่อยๆ ซ่อน Cursor ของผู้เล่นหรือ Catcher ลงเมื่อจำนวน [Combo](/wiki/Gameplay/Combo_(score_multiplier)) เพิ่มขึ้น จนกระทั่งล่องหนไปอย่างสมบูรณ์ในที่สุด ซึ่งการทำ [Combo หลุด (Combobreak)](/wiki/Gameplay/Judgement/Combobreak) จะทำให้การมองเห็นกลับมาเป็นปกติอีกครั้ง

นอกจากนี้ การมองเห็นจะถูกกู้คืนชั่วคราวในช่วงระหว่าง [การพัก (Breaks)](/wiki/Beatmap/Break) ของเพลง

## การปรับแต่ง (Customisation)

![No Scope customisation options in the game client](/wiki/Gameplay/Game_modifier_(lazer)/img/customise/NS.png)

* **Hidden at combo** (0–50, ค่าเริ่มต้น: 10): จำนวน Combo ที่ Cursor หรือ Catcher จะล่องหนไปอย่างสมบูรณ์ หากตั้งค่าเป็น 0 จะมองเห็นได้เฉพาะในช่วงพักเพลงเท่านั้น

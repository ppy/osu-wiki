# Dual Stages (ม็อด)

::: Infobox

<!-- lint ignore heading-increment -->

#### Dual Stages

![ไอคอนม็อด Dual Stages](/wiki/Gameplay/Game_modifier_(lazer)/img/mods/DS.png)

*ด่านสองฝั่ง ความสนุกสองเท่า!*

|  |  |
| :-- | :-- |
| ตัวย่อ (Acronym) | DS |

*สำหรับบทความเวอร์ชัน osu!stable ดูที่: [Co-op (mod)](/wiki/Gameplay/Game_modifier/Co-op)*\
*สำหรับรายชื่อม็อด [lazer](/wiki/Client/Release_stream/Lazer) ทั้งหมด ดูที่: [Game modifier (lazer)](/wiki/Gameplay/Game_modifier_(lazer))*

ม็อด **Dual Stages** จะเพิ่มเพลย์ฟีลด์ที่สอง ซึ่งมีจำนวนคีย์เท่ากับเพลย์ฟีลด์แรก โน้ตบนเพลย์ฟีลด์ที่สองนี้ต้องกดด้วยชุดปุ่มควบคุมอีกชุดหนึ่ง สามารถใช้ [ม็อดคีย์](/wiki/Gameplay/Game_modifier/Key_mods_(lazer)) เพื่อเปลี่ยนจำนวนคีย์ของทั้งสองเพลย์ฟีลด์ได้ (เช่น เล่น [บีตแมป](/wiki/Beatmap) 4K พร้อม 7K และ Dual Stages จะกลายเป็น 7K + 7K)

*โปรดทราบว่าม็อดนี้ใช้ได้กับ [คอนเวิร์ต](/wiki/Beatmap/Converts) เท่านั้น*

**ม็อดนี้ไม่สามารถปรับแต่งได้**

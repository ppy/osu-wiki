# Simplified Rhythm (ม็อด)

::: Infobox

<!-- lint ignore heading-increment -->

#### Simplified Rhythm

![ไอคอนม็อด Simplified Rhythm](/wiki/Gameplay/Game_modifier_(lazer)/img/mods/SR.png)

*ทำจังหวะยาก ๆ ให้ง่ายขึ้น!*

|  |  |
| :-- | :-- |
| ตัวย่อ (Acronym) | SR |
| ประเภท (Type) | ลดความยาก (Difficulty Reduction) |
| ปุ่มลัดเริ่มต้น (Default shortcut key) | `T` |
| โหมดเกม (Game modes) | ![][osu!taiko] |
| ตัวคูณคะแนน (Score multiplier) | `0.60x` |
| สถานะ (Status) | Unranked |
| ม็อดที่ใช้ร่วมกันไม่ได้ | ไม่มี |

:::

*สำหรับรายชื่อม็อด [lazer](/wiki/Client/Release_stream/Lazer) ทั้งหมด ดูที่: [Game modifier (lazer)](/wiki/Gameplay/Game_modifier_(lazer))*

ม็อด **Simplified Rhythm** พยายามทำให้เกมเพลย์ของ osu!taiko ง่ายขึ้น โดยแทนที่แพตเทิร์นที่มีจังหวะยาก (เช่น แพตเทิร์น 1/6) ด้วยแพตเทิร์นที่มีจังหวะง่ายกว่า (เช่น จังหวะ 1/4)

## การปรับแต่ง

![ตัวเลือกการปรับแต่ง Simplified Rhythm ในตัวเกม](/wiki/Gameplay/Game_modifier_(lazer)/img/customise/SR.png)

- **การแปลง 1/3 เป็น 1/2** (ค่าเริ่มต้น: ปิด): แพตเทิร์น 1/3 จะถูกแปลงเป็นจังหวะ 1/2
- **การแปลง 1/6 เป็น 1/4** (ค่าเริ่มต้น: เปิด): แพตเทิร์น 1/6 จะถูกแปลงเป็นจังหวะ 1/4
- **การแปลง 1/8 เป็น 1/4** (ค่าเริ่มต้น: ปิด): แพตเทิร์น 1/8 จะถูกแปลงเป็นจังหวะ 1/4

# Moving Fast (ม็อด)

::: Infobox

<!-- lint ignore heading-increment -->

#### Moving Fast

![ไอคอนม็อด Moving Fast](/wiki/Gameplay/Game_modifier_(lazer)/img/mods/MF.png)

*พุ่งเป็นค่าเริ่มต้น ช้าลงหน่อย!*

|  |  |
| :-- | :-- |
| ตัวย่อ (Acronym) | MF |
| ประเภท (Type) | เน้นความสนุก (Fun) |
| โหมดเกม (Game modes) | ![][osu!catch] |
| ตัวคูณคะแนน (Score multiplier) | `1.00x` |
| สถานะ (Status) | Unranked |
| ม็อดที่ใช้ร่วมกันไม่ได้ | [Autoplay (AT)](/wiki/Gameplay/Game_modifier/Autoplay_(lazer)), [Cinema (CN)](/wiki/Gameplay/Game_modifier/Cinema_(lazer)), [Relax (RX)](/wiki/Gameplay/Game_modifier/Relax_(lazer)) |

:::

*สำหรับรายชื่อม็อด [lazer](/wiki/Client/Release_stream/Lazer) ทั้งหมด ดูที่: [Game modifier (lazer)](/wiki/Gameplay/Game_modifier_(lazer))*

ม็อด **Moving Fast** จะสลับการทำงานของปุ่ม [dash](/wiki/Gameplay/Dash) ของผู้เล่น ทำให้ผู้เล่นต้องปล่อยปุ่มเพื่อ dash ส่วนการกดปุ่มค้างไว้จะทำให้ catcher เคลื่อนที่ด้วยความเร็วปกติแทน

**ม็อดนี้ไม่สามารถปรับแต่งได้**

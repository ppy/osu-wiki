# Cover (mod)

::: Infobox

<!-- lint ignore heading-increment -->

#### Cover

![Cover mod icon](/wiki/Gameplay/Game_modifier_(lazer)/img/mods/CO.png)

*Decrease the playfield's viewing area.*

|  |  |
| :-- | :-- |
| Acronym | CO |
| Type | Difficulty Increase |
| Default shortcut key | `K` |
| Game modes | ![][osu!mania] |
| Score multiplier | 1.00x |
| Status | Unranked |
| Incompatible mods | [Fade In (FI)](/wiki/Gameplay/Game_modifier/Fade_In_(lazer)), [Hidden (HD)](/wiki/Gameplay/Game_modifier/Hidden_(lazer)), [Flashlight (FL)](/wiki/Gameplay/Game_modifier/Flashlight_(lazer)) |

:::

*สำหรับรายชื่อม็อด [lazer](/wiki/Client/Release_stream/Lazer) ทั้งหมด ดูที่: [Game modifier (lazer)](/wiki/Gameplay/Game_modifier_(lazer))*

ม็อด **Cover** จะซ่อนส่วนหนึ่งของเพลย์ฟีลด์คล้ายกับ [Fade In](/wiki/Gameplay/Game_modifier/Fade_In_(lazer)) และ [Hidden](/wiki/Gameplay/Game_modifier/Hidden_(lazer)) แต่สามารถปรับแต่งขนาดของพื้นที่ที่ถูกบังได้

## การปรับแต่ง

![Cover customisation options in the game client](/wiki/Gameplay/Game_modifier_(lazer)/img/customise/CO.png)

- **Coverage** (0.2-0.8, ค่าเริ่มต้น 0.5): สัดส่วนความสูงของเพลย์ฟีลด์ที่จะซ่อนโน้ต
- **Direction** (ค่าเริ่มต้น: Along scroll): ทิศทางที่จะใช้ cover
  - **Along scroll**: บังเพลย์ฟีลด์จากด้านบน
  - **Against scroll**: บังเพลย์ฟีลด์จากด้านล่าง

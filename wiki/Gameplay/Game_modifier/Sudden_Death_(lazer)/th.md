*สำหรับบทความนี้ในเวอร์ชัน osu!stable ดูที่: [Sudden Death (mod)](/wiki/Gameplay/Game_modifier/Sudden_Death)*\
*สำหรับรายชื่อ Mod ของ [lazer](/wiki/Client/Release_stream/Lazer) ทั้งหมด ดูที่: [Game modifier (lazer)](/wiki/Gameplay/Game_modifier_(lazer))*

**Sudden Death** mod จะทำให้ผู้เล่นแพ้ทันทีหากพลาด (Miss) Object ใดๆ จาก [Beatmap](/wiki/Beatmap) การพลาด Note, [Slider tick](/wiki/Gameplay/Hit_object/Slider/Slider_tick) และ [Spinner](/wiki/Gameplay/Hit_object/Spinner) จะทำให้ผู้เล่นแพ้ อย่างไรก็ตาม การพลาด [Slidertail](/wiki/Gameplay/Hit_object/Slider/Slidertail) หรือการไม่กดค้างผ่านเส้นทาง Slider ที่ว่างเปล่าจะไม่ทำให้แพ้ ตราบใดที่ [Combo](/wiki/Gameplay/Combo_(score_multiplier)) ยังไม่หลุด

## การปรับแต่ง (Customisation)

![Sudden Death customisation options in the game client](/wiki/Gameplay/Game_modifier_(lazer)/img/customise/SD.png)

- **Also fail when missing a slider tail** (ค่าเริ่มต้น: ปิดใช้งาน): การพลาด [Slidertail](/wiki/Gameplay/Hit_object/Slider/Slidertail) จะส่งผลให้แพ้ทันที
- **Restart on fail** (ค่าเริ่มต้น: ปิดใช้งาน): เริ่มเล่น Map ใหม่ทันทีหลังจากที่ [แถบพลังชีวิต (Health bar)](/wiki/Client/Interface/Health_bar) ลดลงจนถึงจุดต่ำสุด

*การเปลี่ยนตัวเลือกใดๆ ในหน้านี้จะไม่ทำให้ Mod กลายเป็น Unrankable*

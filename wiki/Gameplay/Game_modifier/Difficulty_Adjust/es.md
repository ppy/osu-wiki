# Difficulty Adjust (mod)

::: Infobox

<!-- lint ignore heading-increment -->

#### Difficulty Adjust

![Icono del mod Difficulty Adjust](/wiki/Gameplay/Game_modifier_(lazer)/img/mods/DA.png)

*Modifica los ajustes de dificultad de un beatmap.*

|  |  |
| :-- | :-- |
| Acrónimo | DA |
| Tipo | Conversión |
| Modos de juego compatibles | ![][osu!] ![][osu!taiko] ![][osu!catch] ![][osu!mania] |
| Multiplicador de puntuación ![][osu!] | `0,10x-1,00x` |
| Multiplicador de puntuación ![][osu!taiko] ![][osu!catch] ![][osu!mania] | `0,50x` |
| Estado | No clasificado |
| Mods incompatibles | [Easy (EZ)](/wiki/Gameplay/Game_modifier/Easy_(lazer)), [Hard Rock (HR)](/wiki/Gameplay/Game_modifier/Hard_Rock_(lazer)), [Target Practice (TP)](/wiki/Gameplay/Game_modifier/Target_Practice_(lazer)) |

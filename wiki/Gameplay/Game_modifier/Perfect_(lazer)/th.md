# Perfect (lazer mod)

::: Infobox

#### Perfect

![Perfect mod icon](/wiki/Gameplay/Game_modifier_(lazer)/img/mods/PF.png)

*ต้อง SS เท่านั้น ไม่งั้นเลิกเล่น*

|  |  |
| :-- | :-- |
| ตัวย่อ (Acronym) | PF |
| ประเภท (Type) | เพิ่มความยาก (Difficulty Increase) |
| ปุ่มลัดเริ่มต้น (Default shortcut key) | `D` |
| โหมดเกม (Game modes) | ![][osu!] ![][osu!taiko] ![][osu!catch] ![][osu!mania] |
| ตัวคูณคะแนน (Score multiplier) | 1.00x |
| สถานะ (Status) | Ranked |
| Mod ที่ใช้ร่วมกันไม่ได้ | [No Fail (NF)](/wiki/Gameplay/Game_modifier/No_Fail_(lazer)), [Sudden Death (SD)](/wiki/Gameplay/Game_modifier/Sudden_Death_(lazer)), [Accuracy Challenge (AC)](/wiki/Gameplay/Game_modifier/Accuracy_Challenge), [Cinema (CN)](/wiki/Gameplay/Game_modifier/Cinema_(lazer)) |

:::

*สำหรับบทความนี้ในเวอร์ชัน osu!stable ดูที่: [Perfect (mod)](/wiki/Gameplay/Game_modifier/Perfect)*\
*สำหรับรายชื่อ Mod ของ [lazer](/wiki/Client/Release_stream/Lazer) ทั้งหมด ดูที่: [Game modifier (lazer)](/wiki/Gameplay/Game_modifier_(lazer))*

**Perfect** mod จะทำการเริ่มเล่น [Beatmap](/wiki/Beatmap) ใหม่โดยอัตโนมัติหากผู้เล่นไม่สามารถรักษาระดับการเล่นเพื่อรับ SS grade ได้อีกต่อไป ซึ่งรวมถึงการกดพลาด (Miss) ทั้งใน Note, [Spinner](/wiki/Gameplay/Hit_object/Spinner), [Slider tick](/wiki/Gameplay/Hit_object/Slider/Slider_tick), [Slidertail](/wiki/Gameplay/Hit_object/Slider/Slidertail) หรือการได้รับผลการตัดสินอื่นๆ ที่ไม่ใช่ค่าสูงสุด (GREAT) จาก Object ใดๆ

## การปรับแต่ง (Customisation)

![Perfect customisation options in the game client](/wiki/Gameplay/Game_modifier_(lazer)/img/customise/PF.png)

- **Restart on fail** (ค่าเริ่มต้น: เปิดใช้งาน): เริ่มเล่น Map ใหม่ทันทีหลังจากที่ [แถบพลังชีวิต (Health bar)](/wiki/Client/Interface/Health_bar) ลดลงจนถึงจุดต่ำสุด *การเปิดใช้งานตัวเลือกนี้จะไม่ทำให้ Mod กลายเป็น Unrankable*

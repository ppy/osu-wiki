# Mirror (mod)

![MR mod icon](/wiki/shared/mods/MR.png "Mirror (MR) mod icon")

*สำหรับบทความนี้ในเวอร์ชัน [lazer](/wiki/Client/Release_stream/Lazer) ดูที่: [Mirror (lazer mod)](/wiki/Gameplay/Game_modifier/Mirror_(lazer))*\
*สำหรับรายชื่อ Mod ทั้งหมด ดูที่: [Game Modifier](/wiki/Gameplay/Game_modifier)*

## เกี่ยวกับ (About)

- ตัวย่อ: MR
- ประเภท: พิเศษ (Special)
- ตัวคูณคะแนน: 1.00x
- โหมดเกมที่รองรับ: ![][osu!mania]

## รายละเอียด (Description)

**Mirror** mod คือ [Game modifier](/wiki/Gameplay/Game_modifier) สำหรับ [osu!mania](/wiki/Game_mode/osu!mania) ที่มุ่งเน้นการช่วยเหลือผู้เล่นที่ถนัดมือซ้ายในการเล่น [Beatmaps](/wiki/Beatmap) ที่เดิมทีถูกสร้างขึ้น (Mapped) โดยคำนึงถึงรูปแบบสำหรับคนถนัดขวาเป็นหลัก Mod นี้จะทำการสะท้อน Note ข้ามแกน y (y-axis) เสมือนที่ลากผ่านกึ่งกลางของ osu!mania keyboard หรือกล่าวคือการ "กลับด้าน" (Mirrors) ตำแหน่งของ Note นั่นเอง โดยการสะท้อนตำแหน่งจะมีลักษณะดังนี้:

![MR gameplay](img/MR-comparison-mania.jpg "เปรียบเทียบระหว่างการเล่นปกติ (ซ้าย) กับการเล่นที่เปิดใช้งาน Mirror mod (ขวา) ใน osu!mania")

เหตุผลเบื้องหลังของเรื่องนี้คือ Beatmap ของ osu!mania จำนวนมากถูกสร้างขึ้นโดย Creator ที่ถนัดมือขวา ส่งผลให้รูปแบบ (Patterns) ของ Beatmap ส่วนใหญ่ถูกออกแบบมาให้เล่นได้ถนัดกว่าสำหรับผู้เล่นที่ถนัดขวา ดังนั้นผู้เล่นที่ถนัดมือซ้ายจำนวนมาก — โดยเฉพาะกลุ่มผู้เล่นอันดับสูง — จึงรู้สึกว่าการกลับด้านตำแหน่ง Note ด้วย Mod นี้ ช่วยให้พวกเขาสามารถใช้งานมือซ้ายได้อย่างมีประสิทธิภาพมากขึ้น

## เกร็ดความรู้ (Trivia)

- Mirror mod ถูกเพิ่มเข้ามาในเดือนกันยายน 2019 เพื่อตอบสนองต่อ [คำขอฟีเจอร์ (Feature request)](https://osu.ppy.sh/community/forums/topics/956618) โดยผู้ใช้ [Kamuy](https://osu.ppy.sh/users/7439226)
  - ในช่วงที่มีการส่งคำขอฟีเจอร์ดังกล่าว ได้มี Mod ที่คล้ายกับ Mirror ถูกปรับใช้ใน [osu!(lazer)](/wiki/Client/Release_stream/Lazer) อยู่ก่อนแล้ว แต่เนื่องจากคุณลักษณะของ Mirror mod ที่ช่วยเพิ่มการเข้าถึง (Accessibility) ให้แก่ผู้เล่น มันจึงถูกเพิ่มเข้าไปในเวอร์ชัน Stable ของ osu! ในขณะนั้นด้วย
  - อันที่จริง Mirror mod เคยถูกร้องขอมาแล้วหลายครั้งในช่วงหลายปีก่อนหน้าคำขอของ Kamuy แต่โพสต์ของ Kamuy เป็นเพียงโพสต์เดียวที่ได้รับความสนใจมากพอจนทำให้ Mod นี้ถูกเพิ่มเข้าไปจริงๆ
- Mirror mod เป็น Game modifier เพียงตัวเดียวในหมวด "พิเศษ" (Special) ที่ไม่ส่งผลกระทบต่อทั้งจำนวน [Performance points (pp)](/wiki/Performance_points) และ [คะแนน (Score)](/wiki/Gameplay/Score) ที่ผู้เล่นจะได้รับ ในขณะที่ยังคงสถานะ Rankable (คิดคะแนนจัดอันดับได้)

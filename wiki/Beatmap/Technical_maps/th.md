# Technical maps

*ประกาศ: สำหรับบทความนี้ คำว่า "tech map" จะถูกใช้เป็นคำเรียกรวมๆ ที่ครอบคลุมทุกคำนิยามเหล่านี้*

**Technical maps** (มักเรียกสั้นๆ ว่า "tech map") เป็นคำที่ไม่มีคำนิยามที่แน่ชัดและเป็นที่ยอมรับร่วมกันในชุมชน osu! ขึ้นอยู่กับตัวผู้ใช้เอง โดยคำว่า "tech map" อาจมีความหมายกว้างๆ เพียงแค่ [Beatmap](/wiki/Beatmap) ที่มี [รูปแบบ (Patterns)](/wiki/Beatmap/Pattern) ที่ไม่ซ้ำซากจำเจ หรืออาจมีความหมายเฉพาะเจาะจงถึง Beatmap ที่มีความหนาแน่นของ [วัตถุ (Hit objects)](/wiki/Gameplay/Hit_object) สูง และประกอบด้วย [Slider](/wiki/Gameplay/Hit_object/Slider) รูปทรงประหลาดที่มีการเปลี่ยนความเร็วอย่างรวดเร็วและรุนแรง (มักอยู่ในช่วงที่เพลงมีความเข้มข้นสูง)

Tech map มักจะเกี่ยวข้องกับแนวเพลงที่มีเสียงซับซ้อนและรวดเร็ว (เช่น drum-and-bass, dubstep และ drumstep) เนื่องจากแนวเพลงเหล่านี้ช่วยให้ Mapper สามารถดึงรายละเอียดของแต่ละเสียงออกมาใช้ประโยชน์ในการสร้างแมพได้อย่างเต็มที่ อย่างไรก็ตาม ไม่ได้มีข้อกำหนดว่าต้องใช้เพลงประเภทนี้เท่านั้น

ไม่ว่าจะนิยามอย่างไร ผู้เล่นหลายคนมองว่า Tech map นั้นไม่ยุติธรรม (ในแง่ของทักษะที่ใช้) เนื่องจากความยากส่วนใหญ่มาจากการวางรูปแบบที่ชวนสับสนหรือ Slider ที่ผิดปกติเพียงอย่างเดียว ด้วยเหตุนี้ Tech map จึงไม่เป็นที่นิยมสำหรับการปั๊ม [pp](/wiki/Performance_points) ปริมาณมากในคราวเดียว เพราะระบบในปัจจุบันยังไม่ให้ค่ากับ Beatmap ลักษณะนี้เท่าที่ควร

## คำนิยาม (Definitions)

เนื่องจากคำว่า "tech map" มีความหมายกว้าง จึงมีหลายแง่มุมที่สามารถนำมาอธิบายคำนี้ได้ รายการด้านล่างแสดงคำนิยามต่างๆ จากมุมมองของกลุ่มผู้เล่นยอดนิยม

### นิยามแบบกว้างที่สุด

*ประกาศ: นิยามแบบ "กว้างที่สุด" ต้องการเพียงหนึ่งหรือสองปัจจัยที่ระบุไว้ เพื่อที่จะเรียก Beatmap นั้นว่าเป็น tech map*

- รูปแบบการวางโน้ตที่ไม่ซ้ำซากหรืออ่านยาก (ดูตัวอย่างด้านล่าง)
  - มี "Flow" (การลื่นไหล) ของแมพที่เล่นยาก
- SV sliders (การเปลี่ยนความเร็ว Slider)
  - อาจมีการใช้ Slider art ร่วมด้วย
- Slider รูปทรงประหลาด
- การเปลี่ยนความเร็ว Slider ที่รวดเร็วและเฉียบคม
- ความเข้มข้นโดยรวมสูง
  - มีความหนาแน่นของวัตถุสูงมากในพื้นที่ส่วนใหญ่ของแมพ (ไม่นับรวม [สตรีม (Streams)](/wiki/Beatmap/Pattern/osu!/Stream))

### นิยามแบบเฉพาะเจาะจง/จำกัดที่สุด

*ประกาศ: นิยามแบบ "จำกัดที่สุด" จำเป็นต้องมีปัจจัยที่ระบุไว้ **ครบทุกข้อ** เพื่อที่จะเรียก Beatmap นั้นว่าเป็น tech map*

- Slider รูปทรงประหลาด
- การเปลี่ยนความเร็ว Slider ที่รวดเร็วและเฉียบคม
- SV sliders
- รูปแบบการวางโน้ตที่อ่านยาก (มักถูกอธิบายว่ามี "Flow" ที่ยาก) (ดูตัวอย่างด้านล่าง)

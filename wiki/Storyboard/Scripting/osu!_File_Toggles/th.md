# Storyboard .osu file toggles

*ดูเพิ่มเติม: [skin.ini](/wiki/Skinning/skin.ini)*

ต่อไปนี้คือ toggle ที่สามารถเพิ่มในไฟล์ .osu (และบางกรณีในไฟล์ .osb) เพื่อส่งผลต่อ [skinning](/wiki/Skinning) / [storyboarding](/wiki/Storyboard) toggle ที่ [Beatmap Editor](/wiki/Client/Beatmap_editor) เพิ่ม/ลบให้เองก็สามารถรวมไว้ได้เช่นกัน

## ภายใต้ section \[General\]

`UseSkinSprites: 1`

อนุญาตให้ storyboard อ้างอิง sprite ของสกินปัจจุบันของผู้เล่นผ่านชื่อไฟล์ได้ สิ่งนี้ไม่มีประโยชน์หากคุณบังคับใช้สกินอยู่แล้ว (แค่ใส่ไฟล์ที่จำเป็นลงไป) หรือใช้สกินเฉพาะแมปอยู่แล้ว [ที่มา](https://osu.ppy.sh/community/forums/topics/1869?start=141760)

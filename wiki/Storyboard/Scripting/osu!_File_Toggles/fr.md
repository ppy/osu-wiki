# Bascule du fichier Storyboard .osu

*Voir également : [skin.ini](/wiki/Skinning/skin.ini)*

Les options suivantes peuvent être ajoutées aux fichiers .osu (et dans certains cas, aux fichiers .osb) pour affecter le [skinning](/wiki/Skinning)/[storyboarding](/wiki/Storyboard). Ceux qui sont ajoutés/supprimés par l'[éditeur de beatmap](/wiki/Beatmap_Editor) peuvent également être inclus.

## Sous la section \[General\]

`UseSkinSprites: 1`

Permet aux storyboards de faire référence aux sprites du skin actuel du joueur par leur nom de fichier. Ceci est inutile si vous forcez déjà un skin (incluez simplement les fichiers nécessaires) ou si vous utilisez un skin spécifique à la map. [Source](https://osu.ppy.sh/community/forums/topics/1869?start=141760)

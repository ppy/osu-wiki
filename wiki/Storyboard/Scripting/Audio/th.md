# Storyboard audio samples

**ไฟล์เสียง** (WAV, MP3 และ OGG) สามารถเล่น ณ จุดเวลาที่กำหนดได้ มันคล้ายกับการประกาศ object ไม่ใช่ command ดังนั้นจึงไม่ถูกใช้ใน loop หรือ trigger อย่างไรก็ตาม คุณสามารถวางมันบน layer ใดก็ได้จาก 4 layer (แม้จะไม่มีแนวคิดเรื่อง "overlapping" เพราะเสียงทั้งหมดจะถูก mix รวมกัน) จึงสามารถทำให้เสียงเกิดขึ้นเฉพาะเมื่อผู้เล่นอยู่ในสถานะ Pass หรือ Fail ได้

## การใช้งาน

รูปแบบการประกาศ audio sample คือ:

`Sample,<time>,<layer_num>,"<filepath>",<volume>`

โดย:

- `<time>` คือ timestamp ที่เสียงควรเริ่มเล่น
- `<layer_num>` คือ *ค่าตัวเลข* ที่ตรงกับ layer ที่คุณต้องการให้เสียงอยู่ ค่าที่ใช้ได้อยู่ในรายการด้านล่าง
- `<filepath>` เป็นแนวคิดเดียวกับ sprite แต่ชี้ไปยังไฟล์ .wav, .mp3 หรือ .ogg
- `<volume>` คือเลขตั้งแต่ 1 ถึง 100 เพื่อบอกความดังสัมพัทธ์ของเสียง (เว้นไว้ได้ถ้าต้องการค่า default เป็น 100)

ค่าที่ใช้ได้สำหรับ layer คือ:

| Value | Layer |
| :-: | :-- |
| 0 | Background |
| 1 | Fail |
| 2 | Pass |
| 3 | Foreground |

## ตัวอย่าง

ตัวอย่างเช่น หากต้องการเล่นเพลงสั้น ๆ เมื่อผู้เล่นได้ good ending ในบีตแมป (เช่น จบในสถานะ Pass) แล้วเล่นเอฟเฟกต์เสียงระเบิดเป็นส่วนหนึ่งของฉาก:

```
Sample,163520,2,"Audio\Best End.mp3",80
Sample,167201,2,"Audio\sfx\EndG-SmallExplosion.wav",100
```

สังเกตว่า volume ของเพลงต่ำกว่าเพื่อให้เอฟเฟกต์เสียงเด่นขึ้น และ layer ของทั้งสองรายการคือ 2 (Pass)

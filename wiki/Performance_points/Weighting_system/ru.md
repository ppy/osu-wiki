# Система весов

В osu! [общее количество очков производительности](/wiki/Performance_points/Total_performance_points) (пп) на аккаунте определяется с помощью **системы весов**. Эта система определяет, сколько [очков производительности](/wiki/Performance_points) в профиль даётся за каждый отдельный рекорд в сравнении с другими. В результате работы **системы весов** ваш второй, третий и так далее по счёту рекорды дают меньше очков производительности, чем самый высокий. Это сделано для того, чтобы игрок не мог постоянно продвигаться в рейтинге, играя самые легкие карты, и заставляет его улучшать свои навыки. Эта самая система весов работает по следующей формуле:

`общее количество pp = pp[1] * 0.95^0 + pp[2] * 0.95^1 + pp[3] * 0.95^2 + ... + pp[m] * 0.95^(m-1)`

Объяснение:

- `pp` обозначает сумму пп каждого отдельного рекорда.
- `pp[n]` обозначает `n`-ое значение пп для каждого рекорда после их сортировки по убыванию.
- `n` варьируется от 1 до `m`, где `m` - количество рекордов, которые имеются у игрока на аккаунте.

## Бонусные пп

Система весов может дать игроку в сумме до 413.894 дополнительных пп, если он будет ставить больше рекордов. Количество рекордов, необходимых для получения всех бонусных пп, равно 1000.

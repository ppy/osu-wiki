# ppv1

Performance score รุ่นแรก (ย่อว่า **ppv1**) คือระบบจัดอันดับโบราณสำหรับผู้เล่นทั่วโลกในทุกโหมดของ osu! มันแทนที่ระบบเก่าแบบ *ranking by total score* และถูกยกเลิกหลังมีการนำ [ppv2](/wiki/Performance_points/ppv2) มาใช้

## ประวัติ

ppv1 เปิดตัวในเดือนเมษายน 2012 และในช่วงทดสอบถูกกล่าวถึงบนโปรไฟล์ผู้เล่นด้วยชื่อ ``???`` หลังจากนั้นจึงถูกเปลี่ยนชื่อเป็น **pp** (performance points) เมื่อวันที่ 17 เมษายน 2012 ระบบนี้อยู่ร่วมกับระบบ ranking เดิม (*ranking by total score*) และสุดท้ายกลายเป็นระบบเดียวที่ใช้จริงในวันที่ 24 กรกฎาคม 2012 ระหว่างการปล่อย osu! client เวอร์ชัน [20120722-24](https://osu.ppy.sh/community/forums/posts/1687719 "20120722-24")

เดิมระบบนี้อัปเดตเป็นรอบ ๆ (โดยปกติทุก 24 ชั่วโมง) จนกระทั่งทำงานแบบ real-time ในวันที่ 16 สิงหาคม 2012

ppv1 ถูกยกเลิกและแทนที่ด้วย **ppv2** เมื่อวันที่ 27 มกราคม 2014 เหตุผลของการเปลี่ยนแปลงมาจากเสียงวิจารณ์ของผู้เล่น:

- Algorithm ขาดความโปร่งใส
- ส่งเสริมระดับความยาก ``Hard`` พร้อมม็อด
- ระดับความยาก insane มีค่าน้อย

นอกจากนี้ยังทำให้ peppy แยกงานดูแลและอัปเดต filing system ออกมาได้ จึงลดภาระงานลง

## Algorithm

รายละเอียดการคำนวณไม่ได้เปิดเผยต่อสาธารณะ อย่างไรก็ตาม มีข้อมูลบางส่วนเกี่ยวกับการทำงานภายในของระบบถูกเปิดเผย:

- ppv1 เน้น skill และ farm ไม่ได้
- Score ถูกประเมินเทียบกับความยากของแมป โดยอิงจากสถิติที่ไม่ได้เปิดเผยต่อสาธารณะ
- จำนวนคะแนนที่ได้ตามธรรมชาติจะลดลงตามเวลา (ดังนั้นผู้เล่นที่ไม่ active อาจเสียคะแนนได้ถึงครึ่งหนึ่งภายในหนึ่งปี)
- ไม่มี penalty สำหรับ scores ที่แย่ มีแต่ reward สำหรับ scores ที่ดี
- แมป *Ranked* และ *Approved* ทั้งหมดถูกนำมาคิดในการคำนวณ
- Accuracy ของผู้เล่นถูกนำมาคิด
- Multipliers ของม็อดแตกต่างจากที่ประกาศในเกม
- ระดับความยากเฉพาะโหมดได้รับความสำคัญ
- หาก score ไม่ติด top 500 ของแมป จะไม่นับ

## การไต่อันดับ

Ranking เกี่ยวข้องกับ performance ในแต่ละแมปเป็นหลัก ทำให้ผู้เล่นจำนวนมากโฟกัสแค่การฝึกระดับความยากเดี่ยว ๆ ppv1 ส่งเสริมธีมการฝึกที่พบได้ทั่วไปในหมู่ผู้เล่น:<!-- the list below is totally vague -->

- เล่นให้ดีขึ้นและทำ grades ที่ดีขึ้นบนบีตแมป
- ปรับ accuracy ให้ดีขึ้น แม้เปอร์เซ็นต์เล็ก ๆ ก็สร้างความต่างได้
- ทำ rank บนแมปยอดนิยม ซึ่งเป็นจุดที่แย่งอันดับยากกว่า
- เล่นแมปยาก
- ปรับปรุง scores เก่า
- ทำ grades SS

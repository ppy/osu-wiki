# Cómo iniciar un proyecto de un beatmap

*Aviso: Este artículo trata sobre la sección del foro [Beatmap Projects](https://osu.ppy.sh/community/forums/53) y cómo abordarla. Asegúrate de leer las [reglas del foro](https://osu.ppy.sh/community/forums/topics/453937) antes de publicar.*

## Crear un título informativo para el hilo

*¡Sí, el título de tu hilo determinará si otros usuarios de osu! aceptarán o rechazarán tu proyecto de beatmap, incluso antes de hacer clic en él!*

`[Discography]` / `[Album]` / `[Collab]` debería añadirse antes del título. Hay muchas posibilidades de que tu proyecto sea ignorado o trasladado a [Abandoned Wasteland](/wiki/Community/Forum/Abandoned_Wasteland) si el título del hilo no es lo suficientemente descriptivo.

Buenos ejemplos:

- `[Album] Pandora Voxx -complete-`
- `[Discography] Deemo`

Mal ejemplo:

- `starting a new project, need your help with this album`

### ¿Cuál debo usar?

**Discography:** Lista de álbumes o canciones lanzadas por el cantante, grupo o compositor, o como banda sonora de un juego.

Ejemplos:

- [\[Discog\] osu! needs more Duca!](https://osu.ppy.sh/community/forums/topics/98205)
- [\[Discography\] Linkin Park](https://osu.ppy.sh/community/forums/topics/121592)
- [\[Discography\] Cytus](https://osu.ppy.sh/community/forums/topics/177446)

**Album:** Lista de canciones del álbum.

Ejemplos:

- [\[Album\] Anamanaguchi - Scott Pilgrim vs The World (the game)](https://osu.ppy.sh/community/forums/topics/37908)
- [\[Album\] Daft Punk - Random Access Memories](https://osu.ppy.sh/community/forums/topics/132592)

**Collab:** Crear un proyecto [de mapping en grupo](/wiki/Beatmap/Beatmap_collaborations) para una canción determinada. Puede variar desde una solicitud para un [storyboard](/wiki/Storyboard) hasta mapear partes de un [maratón](/wiki/Beatmap/Marathon).

Ejemplo:

- [Within Temptation - The Unforgiving](https://osu.ppy.sh/beatmapsets/29157)

# วิธีเริ่มต้นโปรเจกต์บีทแมพ (How to start a beatmap project)

*ข้อสังเกต: บทความนี้เกี่ยวกับส่วนของฟอรัม [Beatmap Projects](https://osu.ppy.sh/community/forums/53) และวิธีการเข้าถึงส่วนนี้ ตรวจสอบให้แน่ใจว่าได้อ่าน [กฎของฟอรัม](https://osu.ppy.sh/community/forums/topics/453937) ก่อนที่จะโพสต์*

## การตั้งชื่อหัวข้อกระทู้ให้มีข้อมูลครบถ้วน (Making an informative thread title)

*ใช่แล้ว ชื่อหัวข้อกระทู้ของคุณจะเป็นตัวกำหนดว่าผู้ใช้ osu! คนอื่นๆ จะตอบรับหรือปฏิเสธโปรเจกต์บีทแมพของคุณ แม้กระทั่งก่อนที่พวกเขาจะคลิกเข้าไปดู!*

ควรเพิ่ม `[Discography]` / `[Album]` / `[Collab]` ไว้ก่อนชื่อเรื่อง มีโอกาสสูงที่โปรเจกต์ของคุณจะถูกเพิกเฉยหรือถูกย้ายไปยัง [Abandoned Wasteland (ดินแดนรกร้างที่ถูกทอดทิ้ง)](/wiki/Community/Forum/Abandoned_Wasteland) หากชื่อหัวข้อกระทู้ไม่มีคำอธิบายที่เพียงพอ

ตัวอย่างที่ดี:

- `[Album] Pandora Voxx -complete-`
- `[Discography] Deemo`

ตัวอย่างที่ไม่ดี:

- `starting a new project, need your help with this album` (กำลังเริ่มโปรเจกต์ใหม่ ต้องการความช่วยเหลือจากคุณสำหรับอัลบั้มนี้)

### ฉันควรใช้อันไหนดี? (Which one do I use?)

**Discography (รายชื่อผลงาน):** รายชื่ออัลบั้มหรือเพลงที่ปล่อยออกมาโดยนักร้อง, กลุ่มศิลปิน หรือนักประพันธ์เพลง หรือเป็นเพลงประกอบสำหรับเกม

ตัวอย่าง:

- [\[Discog\] osu! needs more Duca!](https://osu.ppy.sh/community/forums/topics/98205)
- [\[Discography\] Linkin Park](https://osu.ppy.sh/community/forums/topics/121592)
- [\[Discography\] Cytus](https://osu.ppy.sh/community/forums/topics/177446)

**Album (อัลบั้ม):** รายชื่อเพลงในอัลบั้ม

ตัวอย่าง:

- [\[Album\] Anamanaguchi - Scott Pilgrim vs The World (the game)](https://osu.ppy.sh/community/forums/topics/37908)
- [\[Album\] Daft Punk - Random Access Memories](https://osu.ppy.sh/community/forums/topics/132592)

**Collab (ร่วมมือ):** การสร้าง [โปรเจกต์การทำแมพแบบกลุ่ม](/wiki/Beatmap/Beatmap_collaborations) สำหรับเพลงใดเพลงหนึ่ง อาจมีตั้งแต่การร้องขอ [storyboard](/wiki/Storyboard) ไปจนถึงการทำแมพในบางส่วนของ [marathon map (แมพมาราธอน)](/wiki/Beatmap/Marathon)

ตัวอย่าง:

- [Within Temptation - The Unforgiving](https://osu.ppy.sh/beatmapsets/29157)

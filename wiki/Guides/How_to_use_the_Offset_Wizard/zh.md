# 如何使用偏移量设置向导

*另见: [偏移量设置向导](/wiki/Client/Options/Offset_Wizard)和[全局偏移量](/wiki/Offset/Universal_offset)*

如果[打击音效](/wiki/Beatmapping/Hitsound)与你所玩的所有谱面音乐都不同步，你很可能需要调节[全局偏移量](/wiki/Offset/Universal_offset)。要做到这一点，最简单的方法是使用**偏移量向导**。

根据下面的步骤，来使用偏移量向导调节你的全局偏移量。

## 打开偏移量设置向导

1. 打开 osu!。
2. 转到 `Options` 菜单，或按下 `Ctrl` + `O`。
3. 在搜索栏中搜索“偏移量”（"offset"），然后点击`偏移量（offset）设置向导`按钮。

偏移量设置向导将会打开，并播放在 osu! 中之前播放的歌曲。屏幕上将有绿色的条滑过，同时伴随节拍器的滴答声。

## 校准全局偏移量

1. 仔细听歌曲和节拍器
2. 使用相应的按键（或者使用鼠标滚轮），将全局偏移量调高或调低。
   - 如果游戏内打击音效听起来播放过早，你需要*降低*你的全局偏移量。
   - 如果游戏内打击音效听起来播放过迟，你需要*升高*你的全局偏移量。
3. 反复调节，直到节拍器与歌曲完美配合。用屏幕上滚动的长条来帮助你做到这一点。
4. 校准好全局偏移量后，点击屏幕左下角的`返回`（`Back`）按钮来退出偏移量设置向导。

如果节拍器相对于音乐听起来过快或过慢，你可能需要启用屏幕左上角的`两拍一响`（`Half tick`）或`每拍两响`（`Double tick`）来使调节更容易。

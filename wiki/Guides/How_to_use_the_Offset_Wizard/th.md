# วิธีใช้งาน Offset Wizard (How to use the Offset Wizard)

*ดูเพิ่มเติมที่: [Offset Wizard](/wiki/Client/Options/Offset_Wizard) และ [Universal offset](/wiki/Offset/Universal_offset)*

หาก [เสียงตอนกด (hitsounds)](/wiki/Beatmapping/Hitsound) รู้สึกว่าไม่ตรงกับจังหวะเพลงในทุกๆ บีทแมพที่คุณเล่น เป็นไปได้สูงว่าคุณจำเป็นต้องเปลี่ยนค่า [universal offset](/wiki/Offset/Universal_offset) ของคุณ วิธีที่ง่ายที่สุดในการทำเช่นนี้คือการใช้ **Offset Wizard**

ทำตามขั้นตอนต่อไปนี้เพื่อปรับค่า universal offset ของคุณโดยใช้ Offset Wizard

## การเปิด Offset Wizard

1. เปิด osu!
2. ไปที่เมนู `Options` หรือกด `Ctrl` + `O`
3. ค้นหาคำว่า "offset" ในช่องค้นหาด่วน (quick-search) และคลิกปุ่ม `Offset wizard`

จากนั้น Offset Wizard จะเปิดขึ้น โดยจะเล่นเพลงที่เคยเล่นล่าสุดใน osu! ในขณะที่มีแถบสีเขียวเลื่อนผ่านหน้าจอและมีเสียงเมโทรนอมดังเป็นจังหวะ



## การปรับจูนค่า universal offset

1. ฟังเพลงและเสียงเมโทรนอมที่เล่นไปพร้อมกันอย่างตั้งใจ
2. เริ่มปรับค่า universal offset ขึ้นหรือลง โดยใช้ปุ่มลูกศรที่เกี่ยวข้อง หรือวงล้อเลื่อนของเมาส์
   - หากเสียงตอนกดดูเหมือนจะเล่นเร็วเกินไปในเกม คุณจะต้อง *ลด (decrease)* ค่า universal offset ของคุณ
   - หากเสียงตอนกดดูเหมือนจะเล่นช้าเกินไปในเกม คุณจะต้อง *เพิ่ม (increase)* ค่า universal offset ของคุณ
3. ปรับจนกว่าคุณจะได้ยินเสียงเมโทรนอมตรงกับจังหวะเพลงอย่างสมบูรณ์ โดยใช้แถบที่เลื่อนบนหน้าจอเพื่อช่วยคุณ
4. เมื่อคุณปรับจูนค่า universal offset สำเร็จแล้ว ให้กดปุ่ม `Back` ที่มุมซ้ายล่างของหน้าจอเพื่อออกจาก Offset Wizard

หากเสียงเมโทรนอมรู้สึกว่าเร็วหรือช้าเกินไปสำหรับเพลง คุณอาจต้องการเปิดใช้งาน `Half tick` (จังหวะครึ่ง) หรือ `Double tick` (จังหวะคูณสอง) ที่มุมซ้ายบนของหน้าจอเพื่อให้สิ่งต่างๆ ง่ายขึ้น

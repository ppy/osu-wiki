# วิธีเปลี่ยนชื่อศิลปินหรือชื่อเพลงของแผนที่ (How to change a map's artist or title)

*[Full Submission with Title/Artist Change -- Mistakes by NoHitter (hijacked by peppy)](https://osu.ppy.sh/community/forums/topics/24443)*

เมื่อมีการเปลี่ยน **ชื่อเพลง (Title) หรือชื่อศิลปิน (Artist)** ในแผนที่ อาจเกิดปัญหาที่ทำให้คุณไม่สามารถส่ง (Submit) แผนที่ได้โดยไม่เกิดแผนที่ซ้ำซ้อน (Duplicate)

**ห้ามทำการ Submit หากเมนู "Web" ใน Editor ถูกปิดใช้งาน (เป็นสีเทา) เพราะจะทำให้แผนที่ของคุณเกิดการ Duplicate ทันที**

วิธีแก้ไขเมื่อแผนที่เกิดการ Duplicate ให้ทำดังนี้:

- วิธีแบบง่าย (สำหรับกรณีที่ไม่มีการแก้ไขข้อมูลในเครื่องล่าสุด):
  1. ลบแผนที่ฉบับสำเนาในเครื่องทิ้ง
  2. ดาวน์โหลดแผนที่ฉบับสำเนาจากเว็บไซต์ osu! มาใหม่
  3. เข้าไปที่ Editor และตรวจสอบให้แน่ใจว่าเมนู "Web" สามารถใช้งานได้ (ไม่เป็นสีเทา)
  4. แก้ไขข้อมูลชื่อศิลปิน/ชื่อเพลง (Artist/Title)
  5. **ห้ามออกจาก Editor เด็ดขาด**
  6. ใช้เมนู `File` > `Open...` เพื่อเปลี่ยนชื่อศิลปิน/ชื่อเพลงในระดับความยาก (Difficulty) อื่นๆ ให้ครบถ้วน
  7. ทำการ Full Submit

- วิธีแบบยาก (สำหรับกรณีต้องการเก็บสิ่งที่แก้ไขไว้ในเครื่อง):
  1. ไปที่ [หน้าข้อมูลบีทแมพ (Beatmap info page)](/wiki/Beatmap_information) บนเว็บไซต์ osu! สำหรับแผนที่ของคุณ
  2. เปิด Song Setup ในแต่ละระดับความยาก แล้วตั้งค่าชื่อศิลปิน/ชื่อเพลงกลับไปเป็นค่าเดิมที่อยู่บนเว็บไซต์
  3. ไปที่หน้าเลือกเพลง (Song Select) แล้วกดปุ่ม `F5` จากนั้นรอสักครู่
  4. เข้าไปที่ Editor และตรวจสอบให้แน่ใจว่าเมนู "Web" สามารถใช้งานได้ (ไม่เป็นสีเทา)
  5. แก้ไขข้อมูลชื่อศิลปิน/ชื่อเพลง (Artist/Title)
  6. **ห้ามออกจาก Editor เด็ดขาด**
  7. ใช้เมนู `File` > `Open...` เพื่อเปลี่ยนชื่อศิลปิน/ชื่อเพลงในระดับความยากอื่นๆ ให้ครบถ้วน
  8. ทำการ Full Submit

หากคุณมีแผนที่ที่เกิดการ Duplicate ไปแล้ว โปรดติดต่อสมาชิกของทีม [NAT](/wiki/People/Nomination_Assessment_Team) เพื่อขอรับการแก้ไขครับ

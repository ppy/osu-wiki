# 音频编辑指南

[osu! 学院](/wiki/Community/Video_series/osu!academy)视频的[第 15 章：音频编码 (4:02, YouTube)](https://www.youtube.com/watch?v=muu3HkG38kk) 包括了本文内容。章节中还包含安装 Audacity 与借助 LAME 导出 `.mp3` 文件的方法。

本文可充当帮助你出于作图目的，对音频文件进行小幅编辑的指南。进一步扩展的话，你就能帮其他作图者创建适应任何需要的音频文件了。

*本文只列出了作者添加的软件，你并不是只能使用这些软件。如果你了解其他可用工具，并且能解释如何将它们用于以下小节，也请添加进来。*

## Audacity

*备注：本文使用的是 Audacity 3.4.2。*

[Audacity](https://www.audacityteam.org/download) 是一款免费、开源的音频编辑与录制软件。它使用 [LAME](https://lame.sourceforge.io) 编码库导出 MP3 格式的音频文件，此编码库在 Windows 与 macOS 平台的 Audacity 是内置的。

大多数 Linux 发行版会提供安装 Audacity 所需的 LAME，但某些版本也可能不会提供。因此 Linux 用户应当阅读 [Audacity 参考手册](https://manual.audacityteam.org/man/installing_and_updating_audacity_on_linux.html#linlame)获取更多信息。

### 降低比特率

*对于压缩相关的一般信息，参见：[压缩文件](/wiki/Guides/Compressing_files)*

安装并打开 Audacity，按如下步骤操作：

1. 打开要降低比特率的 `.mp3` 文件。
2. 按下 `Ctrl` + `Shift` + `E`，或者点击 `文件` -> `导出` -> `导出音频...`。
3. 在`格式 (Format)：`一栏选择 `MP3 文件`。
4. 修改`音频设置 (Audio options)` 中的如下设置：
   1. `比特率模式 (Bit Rate Mode)`：`预设`
   2. `质量`：`中等，145-185 kbps`
5. 编辑`文件名 (File Name):` 和`文件夹 (Folder):`，决定要保存的文件位置与名称。
   - 也可以点击`浏览 (Browse)...` 按钮进行导航。
6. 如果要编辑文件元数据，点击`编辑元数据 (Edit Metadata)...`。
7. 元数据编辑完成后点击 `OK`。
8. 点击`导出 (Export)` 以导出音频文件。

### 循环

安装并打开 Audacity，按如下步骤操作：

1. 打开要进行循环的 `.mp3` 文件。
2. 点击并拖动光标，选中要循环的部分。
   - 有时音乐从头到尾都要循环。
   - 副歌或高潮部分有时循环效果最好。
3. 按下 `Ctrl` + `C`，或者点击`编辑` -> `复制`。
4. 找到可以循环播放复制部分的位置。
   - 可以是选中区域末端。
5. 按下 `Ctrl` + `V`，或者点击`编辑` -> `粘贴`。
6. 播放整首音乐，确保循环效果听起来不错。
7. 按需重复操作。
8. [导出](#降低比特率)音频文件。

### 剪裁

安装并打开 Audacity，按如下步骤操作：

1. 打开要剪裁的 `.mp3` 文件。
2. 点击并拖动光标，选中要剪去的部分。
   - 应该是不想作图的较长尾奏部分。
3. 按下 `Delete` 键。
4. 点击并拖动光标，选中音频结束前的 3 到 5 秒。
5. 点击`效果` -> `淡入淡出` -> `淡出`。
6. [导出](#降低比特率)音频文件。

## mp3DirectCut

*备注：本文使用的是 mp3DirectCut 2.36。*

[mp3DirectCut](https://mpesch3.de) 是一款免费使用的音频编辑软件，可以在不重新编码的情况下直接编辑 MP3 文件，通常会避免音频质量损失。推荐在需要调整音频音量或剪裁音频时使用。

### 降低比特率

*注意：这个功能需要 32 位的编码库文件。*

1. 打开要降低比特率的 `.mp3` 文件。
2. 点击`设置` -> `配置 (Configuration)`，或者按下 `F11`。
3. 转到`编码器 (Encoder)`标签页。
4. 启用下列可用的编码库之一。
   - `System`: ACM 编码库
   - `Lame`: 安装目录下的 `Lame_enc.dll` 文件
5. 将最低比特率更改为 `128 kbit/s`，最高比特率更改为 `192 kbit/s`。
6. 点击`保存`.
7. 按下 `Ctrl` + `W`，或者点击`文件` -> `保存全部音频...`。
8. 导航到要保存文件的位置。
   - 你也可以重命名文件。
9. 点击`保存`。

### 调整数值

安装并打开 mp3DirectCut，按如下步骤操作：

1. 打开要进行调整的 `.mp3` 文件。
2. 按下 `Ctrl` + `A` 全选。
3. 按下 `Ctrl` + `G`，或者点击`编辑` -> `增益...`。
4. 勾选`锁定滑动轴`复选框。
5. 选中左侧滑块并向下拖动。
   - 设置的 dB（分贝）值会有所不同，试试各种音量，觉得合适即可。
6. 满意后按下 `OK`。
7. 按下 `Ctrl` + `W`，或者点击`文件` -> `保存全部音频...`。
8. 导航到要保存文件的位置。
   - 你也可以重命名文件。
9. 点击`保存`。

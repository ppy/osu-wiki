# คู่มือการตัดต่อเสียง (Audio editing guide)

[osu!academy](/wiki/Community/Video_series/osu!academy) ได้เคยพูดถึงเรื่องนี้ไว้ใน [Episode 15: Audio Encoding (4:02)](https://www.youtube.com/watch?v=muu3HkG38kk) ซึ่งในวิดีโอนั้นจะรวมถึงวิธีการติดตั้งและใช้งาน Audacity ร่วมกับปลั๊กอิน LAME เพื่อส่งออกไฟล์เป็น `.mp3`

บทความนี้จะทำหน้าที่เป็นคู่มือเพื่อช่วยให้คุณทำการแก้ไขไฟล์เสียงเล็กๆ น้อยๆ สำหรับการสร้าง Beatmap โดยการเพิ่มเติมเนื้อหาในส่วนนี้จะช่วยให้เพื่อนๆ นักสร้างแผนที่คนอื่นสามารถสร้างไฟล์เสียงที่เหมาะสมกับความต้องการได้

*ข้อมูลในนี้ไม่ใช่ซอฟต์แวร์เพียงอย่างเดียวที่คุณสามารถใช้งานได้ แต่เป็นเพียงรายชื่อโปรแกรมที่มีผู้นำมาเขียนไว้ หากคุณรู้จักเครื่องมืออื่นๆ ที่ใช้งานได้และสามารถอธิบายวิธีใช้ในส่วนต่างๆ ด้านล่างนี้ได้ โปรดช่วยกันเพิ่มเติมข้อมูลได้เลย*

## Audacity

*หมายเหตุ: บทความนี้อ้างอิงจาก Audacity เวอร์ชัน 3.4.2*

[Audacity](https://www.audacityteam.org/download) เป็นซอฟต์แวร์บันทึกและตัดต่อเสียงแบบโอเพนซอร์สที่ใช้งานได้ฟรี โปรแกรมนี้ใช้ไลบรารี [LAME](https://lame.sourceforge.io) ในการส่งออกไฟล์เสียงในรูปแบบ MP3 ซึ่งจะถูกติดตั้งมาพร้อมกับ Audacity ในระบบปฏิบัติการ Windows และ macOS อยู่แล้ว

สำหรับผู้ใช้งาน Linux โปรดอ้างอิงจาก [คู่มือการใช้งาน Audacity](https://manual.audacityteam.org/man/installing_and_updating_audacity_on_linux.html#linlame) สำหรับข้อมูลเพิ่มเติม เนื่องจาก Linux บางรุ่นอาจไม่ได้ให้ LAME มาพร้อมกับการติดตั้ง Audacity แต่ส่วนใหญ่จะมีมาให้

### การลดบิตเรต (Lowering Bit Rate)

*สำหรับข้อมูลเกี่ยวกับการบีบอัดไฟล์โดยทั่วไป ดูที่: [การบีบอัดไฟล์ (Compressing files)](/wiki/Guides/Compressing_files)*

ติดตั้งและเปิดโปรแกรม Audacity จากนั้นทำตามขั้นตอนดังนี้:

1. เปิดไฟล์ `.mp3` ที่คุณต้องการลดบิตเรต
2. กดปุ่ม `Ctrl` + `Shift` + `E` หรือคลิกที่ `File` -> `Export Audio...`
3. ในช่อง `Format:` ให้เลือกเป็น `MP3 Files`
4. ในส่วน `Audio options` ให้ตั้งค่าดังนี้:
   1. `Bit Rate Mode`: `Preset`
   2. `Quality`: `Medium, 145-185 kbps`
5. แก้ไข `File Name:` และ `Folder:` เพื่อกำหนดตำแหน่งและชื่อไฟล์ที่ต้องการบันทึก
   - คุณสามารถใช้ปุ่ม `Browse...` เพื่อหาที่จัดเก็บไฟล์ได้
6. หากคุณต้องการแก้ไข Metadata ของไฟล์ ให้คลิกที่ `Edit Metadata...`
7. คลิก `OK` เมื่อแก้ไข Metadata เสร็จสิ้น
8. คลิก `Export` เพื่อส่งออกไฟล์เสียง

### การทำลูป (Looping)

ติดตั้งและเปิดโปรแกรม Audacity จากนั้นทำตามขั้นตอนดังนี้:

1. เปิดไฟล์ `.mp3` ที่คุณต้องการทำลูป
2. คลิกและลากเพื่อคลุมดำ (Highlight) ส่วนที่ต้องการทำลูป
   - น้อยครั้งที่จะเป็นการทำลูปตั้งแต่ต้นจนจบเพลง
   - ส่วนใหญ่มักจะทำลูปในส่วนของฮอรัส (Chorus) หรือท่อนแยก (Refrain) จะให้ผลลัพธ์ที่ดีที่สุด
3. กดปุ่ม `Ctrl` + `C` หรือคลิกที่ `Edit` -> `Copy`
4. หาตำแหน่งที่คุณต้องการวางส่วนที่คัดลอกมาเพื่อทำลูป
   - อาจจะเป็นช่วงท้ายของส่วนที่คุณคลุมดำไว้ก่อนหน้า
5. กดปุ่ม `Ctrl` + `V` หรือคลิกที่ `Edit` -> `Paste`
6. ลองฟังเพลงทั้งหมดและตรวจสอบให้แน่ใจว่าเสียงที่ลูปนั้นฟังดูเนียนและลื่นไหล
7. ทำซ้ำตามต้องการ
8. [ส่งออกไฟล์เสียง](#การลดบิตเรต-lowering-bit-rate)

### การตัดบางส่วนของเพลง (Cropping)

ติดตั้งและเปิดโปรแกรม Audacity จากนั้นทำตามขั้นตอนดังนี้:

1. เปิดไฟล์ `.mp3` ที่คุณต้องการตัดออก
2. คลิกและลากเพื่อคลุมดำส่วนที่ต้องการลบออก
   - เช่น ส่วน Outro ที่ยาวเกินไปซึ่งคุณไม่ต้องการสร้างแผนที่ในส่วนนั้น
3. กดปุ่ม `Delete` หรือ `Ctrl` + `K` หรือคลิกที่ `Edit` -> `Delete`
4. คลิกและลากคลุมช่วง 3 ถึง 5 วินาทีสุดท้ายของไฟล์เสียง
5. คลิกที่ `Effect` -> `Fading` -> `Fade Out`
6. [ส่งออกไฟล์เสียง](#การลดบิตเรต-lowering-bit-rate)

## mp3DirectCut

*หมายเหตุ: บทความนี้อ้างอิงจาก mp3DirectCut เวอร์ชัน 2.36*

[mp3DirectCut](https://mpesch3.de) เป็นซอฟต์แวร์ตัดต่อเสียงที่ใช้งานได้ฟรี ซึ่งสามารถแก้ไขไฟล์ MP3 ได้โดยตรงโดยไม่ต้องทำการ Encoding ใหม่ ซึ่งมักจะช่วยป้องกันไม่ให้คุณภาพเสียงสูญเสียไป แนะนำให้ใช้เมื่อต้องการเพิ่ม/ลดความดัง หรือตัดบางส่วนของเพลง

### การลดบิตเรต (Lowering Bit Rate)

*หมายเหตุ: จำเป็นต้องมีไฟล์ไลบรารีแบบ 32-bit เพื่อใช้งานฟีเจอร์นี้*

1. เปิดไฟล์ `.mp3` ที่คุณต้องการลดบิตเรต
2. คลิกที่ `Settings` -> `Configuration` หรือกดปุ่ม `F11`
3. ไปที่แถบ `Encoder`
4. เปิดใช้งานไลบรารี Encoding ตัวใดตัวหนึ่งที่มีอยู่:
   - `System`: ACM encoding library
   - `Lame`: ไฟล์ `Lame_enc.dll` ที่อยู่ในโฟลเดอร์ที่ติดตั้งโปรแกรม
5. เปลี่ยนค่า Minimum bitrate เป็น `128 kbit/s` และ Maximum เป็น `192 kbit/s`
6. คลิก `Close`
7. กดปุ่ม `Ctrl` + `W` หรือคลิกที่ `File` -> `Save complete audio...`
8. เลือกตำแหน่งที่ต้องการบันทึกและตั้งชื่อไฟล์
9. คลิก `Save`

### การปรับระดับความดัง (Adjusting the Volume)

ติดตั้งและเปิดโปรแกรม mp3DirectCut จากนั้นทำตามขั้นตอนดังนี้:

1. เปิดไฟล์ `.mp3` ที่คุณต้องการปรับ
2. กดปุ่ม `Ctrl` + `A` เพื่อเลือกทั้งหมด
3. กดปุ่ม `Ctrl` + `G` หรือคลิกที่ `Edit` -> `Level...`
4. ติ๊กถูกที่ช่อง `Lock Sliders`
5. เลื่อนแถบเลื่อน (Slider) ด้านซ้ายลงเพื่อลดความดัง
   - ค่า dB (decibels) ที่เหมาะสมจะแตกต่างกันไป ให้ลองปรับและฟังจนกว่าจะพอใจ
6. เมื่อพอใจแล้วให้คลิก `OK`
7. กดปุ่ม `Ctrl` + `W` หรือคลิกที่ `File` -> `Save complete audio...`
8. เลือกตำแหน่งที่ต้องการบันทึกและคลิก `Save`

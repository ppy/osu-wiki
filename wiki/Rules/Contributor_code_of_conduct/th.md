# Contributor Covenant Code of Conduct

## Our Pledge

เราในฐานะ members, contributors และ leaders ให้คำมั่นว่าจะทำให้การมีส่วนร่วมในคอมมูนิตี้ของเราเป็นประสบการณ์ที่ปราศจากการคุกคามสำหรับทุกคน ไม่ว่าจะมีอายุ รูปร่าง ความพิการที่มองเห็นหรือมองไม่เห็น ชาติพันธุ์ sex characteristics, gender identity and expression, ระดับประสบการณ์ การศึกษา สถานะทางเศรษฐกิจและสังคม สัญชาติ รูปลักษณ์ภายนอก เชื้อชาติ ศาสนา หรือ sexual identity and orientation แบบใดก็ตาม

เราให้คำมั่นว่าจะปฏิบัติและมีปฏิสัมพันธ์ในวิธีที่ช่วยสร้างคอมมูนิตี้ที่เปิดกว้าง ต้อนรับ หลากหลาย inclusive และดีต่อสุขภาพ

## Our Standards

ตัวอย่างพฤติกรรมที่ช่วยสร้างสภาพแวดล้อมเชิงบวกให้คอมมูนิตี้ของเรา ได้แก่:

- แสดงความเข้าอกเข้าใจและความเมตตาต่อคนอื่น
- เคารพความคิดเห็น มุมมอง และประสบการณ์ที่แตกต่าง
- ให้และรับ constructive feedback อย่างสุภาพ
- รับผิดชอบและขอโทษผู้ที่ได้รับผลกระทบจากความผิดพลาดของเรา และเรียนรู้จากประสบการณ์นั้น
- โฟกัสสิ่งที่ดีที่สุดไม่ใช่แค่สำหรับเราในฐานะปัจเจก แต่สำหรับคอมมูนิตี้โดยรวม

ตัวอย่างพฤติกรรมที่ยอมรับไม่ได้ ได้แก่:

- การใช้ภาษาหรือภาพทางเพศ และความสนใจหรือการเข้าหาทางเพศทุกรูปแบบ
- Trolling, คอมเมนต์ดูหมิ่นหรือเหยียดหยาม และการโจมตีส่วนตัวหรือทางการเมือง
- การคุกคามในที่สาธารณะหรือส่วนตัว
- การเผยแพร่ข้อมูลส่วนตัวของผู้อื่น เช่น ที่อยู่จริงหรือที่อยู่อีเมล โดยไม่ได้รับอนุญาตอย่างชัดเจน
- พฤติกรรมอื่นที่สามารถมองได้อย่างสมเหตุสมผลว่าไม่เหมาะสมในพื้นที่ professional

## Enforcement Responsibilities

community leaders รับผิดชอบในการชี้แจงและบังคับใช้ standards ของพฤติกรรมที่ยอมรับได้ และจะดำเนินการแก้ไขที่เหมาะสมและเป็นธรรมเพื่อตอบสนองต่อพฤติกรรมใด ๆ ที่พวกเขาเห็นว่าไม่เหมาะสม คุกคาม offensive หรือเป็นอันตราย

community leaders มีสิทธิ์และความรับผิดชอบในการนำออก แก้ไข หรือปฏิเสธ contributions อื่นที่ไม่สอดคล้องกับ Code of Conduct นี้ และจะสื่อสารเหตุผลของ moderation decisions เมื่อเหมาะสม

## Scope

Code of Conduct นี้ใช้ในพื้นที่คอมมูนิตี้ทั้งหมด และยังใช้เมื่อบุคคลหนึ่งเป็นตัวแทนคอมมูนิตี้อย่างเป็นทางการในพื้นที่สาธารณะ ตัวอย่างการเป็นตัวแทนคอมมูนิตี้ของเรา ได้แก่ การใช้ official e-mail address, การโพสต์ผ่าน official social media account หรือการทำหน้าที่เป็นตัวแทนที่ได้รับแต่งตั้งใน event ออนไลน์หรือออฟไลน์

## Enforcement Guidelines

community leaders จะทำตาม Community Impact Guidelines เหล่านี้ในการกำหนดผลลัพธ์สำหรับ action ใด ๆ ที่พวกเขาเห็นว่าละเมิด Code of Conduct นี้:

### 1. Correction

**Community Impact**: การใช้ภาษาที่ไม่เหมาะสมหรือพฤติกรรมอื่นที่ถือว่าไม่ professional หรือไม่เป็นที่ต้อนรับในคอมมูนิตี้

**Consequence**: คำเตือนแบบส่วนตัวเป็นลายลักษณ์อักษรจาก community leaders ซึ่งให้ความชัดเจนเกี่ยวกับลักษณะของการละเมิดและคำอธิบายว่าทำไมพฤติกรรมนั้นไม่เหมาะสม อาจมีการขอให้ขอโทษต่อสาธารณะ

### 2. Warning

**Community Impact**: การละเมิดจาก incident เดียวหรือชุดของ actions

**Consequence**: คำเตือนพร้อมผลที่ตามมาหากยังมีพฤติกรรมต่อเนื่อง ห้ามมี interaction กับผู้ที่เกี่ยวข้อง รวมถึง unsolicited interaction กับผู้ที่บังคับใช้ Code of Conduct เป็นระยะเวลาที่กำหนด ซึ่งรวมถึงการหลีกเลี่ยง interactions ในพื้นที่คอมมูนิตี้และช่องทางภายนอกอย่าง social media การละเมิดเงื่อนไขเหล่านี้อาจนำไปสู่ temporary หรือ permanent ban

### 3. Temporary Ban

**Community Impact**: การละเมิด standards ของคอมมูนิตี้อย่างร้ายแรง รวมถึงพฤติกรรมไม่เหมาะสมต่อเนื่อง

**Consequence**: temporary ban จาก interaction หรือการสื่อสารสาธารณะกับคอมมูนิตี้ทุกรูปแบบเป็นระยะเวลาที่กำหนด ห้ามมี public หรือ private interaction กับผู้ที่เกี่ยวข้อง รวมถึง unsolicited interaction กับผู้ที่บังคับใช้ Code of Conduct ในช่วงเวลานี้ การละเมิดเงื่อนไขเหล่านี้อาจนำไปสู่ permanent ban

### 4. Permanent Ban

**Community Impact**: แสดง pattern ของการละเมิด standards ของคอมมูนิตี้ รวมถึงพฤติกรรมไม่เหมาะสมต่อเนื่อง การคุกคามบุคคล หรือ aggression ต่อหรือ disparagement of กลุ่มบุคคล

**Consequence**: permanent ban จาก public interaction ทุกประเภทภายในคอมมูนิตี้

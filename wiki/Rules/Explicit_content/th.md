# Explicit content

![สกรีนช็อตของบีตแมปที่มี explicit content](img/explicit-tag.jpg "ตัวอย่างบีตแมปที่ถูก mark ด้วย tag 'explicit'")

mappers สามารถประกาศว่าคอนเทนต์ของบีตแมปตัวเองเป็น *explicit* ได้ด้วยการ tick checkbox `Explicit Content` ที่อยู่ใน [genre and language popup menu](/wiki/Beatmap/Genre_and_language) บน web listing ของบีตแมปนั้น ฟีเจอร์นี้มีความคาดหวังและกฎบางอย่างกำกับการใช้งาน

ไม่ว่า audio จะเป็นลักษณะใด โปรดตรวจให้แน่ใจว่ามันทำตาม [general song content rules](/wiki/Rules/Song_content_rules) ด้วย

## อะไรนับเป็น explicit content?

explicit content หมายถึง **audio content ของบีตแมป** เป็นหลัก ส่วนใหญ่เกี่ยวกับ theme, subject matter หรือการใช้ภาษาหยาบแบบ *หนัก* สิ่งนี้ **ไม่** ใช้กับ visual elements ของบีตแมป เพราะสิ่งเหล่านั้น **ต้อง** ทำตาม [Visual Content Considerations](/wiki/Rules/Visual_content_considerations) หรือส่วนอื่นของบีตแมปที่ผู้ใช้เปลี่ยนได้ (เช่น tags หรือชื่อ difficulty ของบีตแมป) ซึ่ง **ต้อง** ทำตาม[กฎคอมมูนิตี้ osu!](/wiki/Rules) ตามปกติ

โดยหลักทั่วไป provision ของ explicit content จะสร้างข้อยกเว้นเฉพาะให้คอนเทนต์ที่ผู้ใช้ไม่สามารถแก้ไขได้อย่างสมเหตุสมผลด้วยเครื่องมือที่เกมให้มา และยังอาจถูกเพิกถอนเป็นราย track ตามดุลพินิจของ [osu! support team](/wiki/People/Account_support_team)

โดยทั่วไป เพลงส่วนใหญ่สามารถใช้ในบีตแมปได้ ยกเว้นน้อยมาก ตราบใดที่ถูก mark อย่างเหมาะสม

## อะไรนับเป็นการใช้ explicit language แบบหนัก?

การใช้คำหยาบทั่วไปแบบเบา ๆ (หรือครั้งเดียว) ยังไม่พอให้บีตแมปเข้าข่ายต้องใช้ explicit content marker บีตแมปที่มีภาษาแบบนี้สามารถปล่อยไว้โดยไม่ tag เป็น explicit ได้ ตราบใดที่ภาษานั้นเป็นสิ่งที่คนทั่วไปคาดได้อย่างสมเหตุสมผลจากคอนเทนต์ที่จัดว่า "parental guidance" (PG-13) หากมีข้อถกเถียงว่าเรื่องนี้เกี่ยวข้องกับ track นั้นไหม ให้ถือว่าไม่เกี่ยวข้อง

การใช้คำหยาบต้องต่อเนื่อง มีนัยสำคัญ ยั่วยุอย่างหนัก หรือซ้ำมาก จึงจะนับเป็น explicit

นอกจากนี้ การพูดถึงหัวข้อ controversial อย่างชัดเจนและต่อเนื่อง หรืออะไรก็ตามที่คนมีเหตุผลจะมองว่า "หนัก" ควรถูกถือเป็น explicit ตัวอย่างบางส่วน (แต่ไม่ใช่ทั้งหมด) ได้แก่:

- ภาพ ผลกระทบ หรือผลลัพธ์ของการฆ่าตัวตาย
- ความรุนแรงที่สื่อเป็นนัยอย่างหนัก
- การบรรยาย/พูดถึงผลลัพธ์ของความรุนแรงแบบมีรายละเอียด gore
- การอ้างอิงทางเพศหรือ allusions ถึง sexual acts อย่างชัดเจนมาก

วิธีที่ดีในการแยกว่าสิ่งใดควรถูก tag เป็น explicit หรือไม่ คือดูว่า major audio streaming services มีการระบุไว้แบบนั้นบนแพลตฟอร์มของตัวเองหรือไม่ [Spotify](https://www.spotify.com) เป็นจุดเริ่มต้นที่ดีมาก

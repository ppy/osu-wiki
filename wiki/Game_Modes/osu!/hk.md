簡介
====

甚麼是「應援團」？
------------------

[押忍！戰鬥！應援團](http://zh.wikipedia.org/wiki/%E6%8A%BC%E5%BF%8D%EF%BC%81%E6%88%B0%E9%AC%A5%EF%BC%81%E6%87%89%E6%8F%B4%E5%9C%98) 是任天堂DS上的一款節奏遊戲。其遊戲內容僅包含三種元素：在觸屏上點擊圓圈、在固定的路徑上拖動小球以及高速轉動轉盤。所有的這些動

作都是跟隨著日本流行歌曲的節奏進行的。在DS平台上的截圖如下：

![](Ouendan.jpg "Ouendan.jpg")

遊戲中，上屏表示主人公的努力過程，下屏作用是遊戲操作，背景有應援團團員做的各種動作。每關的基本內容是關於一個在生活中遇到困難的人。

你將通過應援團的神奇力量，為他們加油助威，使他們走出困境。

如何進行遊戲？
--------------

由於大部分的PC玩家並沒有安裝觸屏等設備，因此本遊戲可以使用普通的鼠標，或者你能想到的任何輸入設備來進行操作。不同的遊戲方法可以在

[遊玩風格頁面中找到](HK:Play_Styles "wikilink")。遊戲中有一些歌曲可能會挑戰你的鼠標（或者你的手）的極限，但是遊戲所有（Ranked或Approved）的

譜面都經過測試，（大部分）都能使用鼠標過關。

### 遊戲基礎

教程譜面：[（預置於PC版 osu! 遊戲中）原版教程譜面](http://osu.ppy.sh/s/3756)，[Ranked版本教程譜面](http://osu.ppy.sh/s/19928) 這款遊戲主要需要你的：

-   反應能力（及時點擊出現的物件）
-   對歌曲節奏的掌握
-   對鼠標的控制能力（如果使用數位板則是運筆的能力了）
-   對於物件在屏幕上出現時間的估測能力
-   使用鼠標（手指、筆）高速轉動的能力
-   耐力（特別是馬拉松歌曲）
-   鼠標定位和讀圖能力
-   對於各種硬件的熟悉

這些是你在成為 osu! 神觸之路上將會學習和掌握到的一些重要技巧。在你成為神觸之前，你需要學習一些關於擊打物件的基礎知識。

下面是關於這些物件的一些基礎知識。

#### 圓圈

<img src="Hitcircles.jpg" title="Hitcircles.jpg" alt="Hitcircles.jpg" width="400" height="300" />

-   圖中的實心圓圈就是需要在相應時間點擊的物件。
    -   實心圓圈外面會有一個大圓圈，當大圈正好收縮到小圈的白色邊緣的那一時刻，就應當點擊這個物件。
-   為了深入理解圓圈的擊打規律，你需要做的不僅僅是用眼睛去判定，更重要的是需要了解這些物件是跟隨的歌曲的節奏的，需要用耳朵來判斷。

#### 滑條

<img src="Slider.jpg" title="Slider.jpg" alt="Slider.jpg" width="400" height="300" />

-   圖中的物件就是一個滑條。
    -   首先，你要像點擊一個圓圈一樣點擊在滑條的頭部，但是此時並不要鬆手。
    -   此時一個小球將會出現，沿著滑條的軌跡移動。
    -   你必須保持鼠標點擊／鍵盤按下（或者手指／筆尖觸板）的狀態，使得光標位於小球外圍的橙色圓圈中，並跟著小球一起移動。
        -   小球的移動不受你的控制，所以你必須跟著它來移動。
        -   當你保持按住鼠標／鍵盤的狀態下，橙色圓圈才會出現，否則它不可見。
    -   有時，比如圖中的情況，當你到達滑條末端時還需要原路返回。
    -   滑條末端的箭頭形狀可以指示上一條所說的情況。

#### 轉盤

<img src="Spinner.jpg" title="Spinner.jpg" alt="Spinner.jpg" width="400" height="300" />

-   圖中的物件是一個轉盤。
    -   當它出現時你需要按下鼠標/鍵盤上表示點擊的鍵，並使用鼠標瘋狂地轉圈直到表示倒計時的藍色圓圈完全收縮。
        -   轉盤外面的條顯示你還差多少通過這個轉盤。
        -   轉盤下面的小橫條上顯示當前的轉速，表示當前的速度下每分鐘可以轉多少圈。
        -   如果你在轉的時候不按下表示擊打的鍵，轉圈不會有任何用。
-   在DS上，這是毀屏幕的絕佳方法（尤其是高難度的曲子）。
-   在 osu! 中，轉盤的難度進行了一些調整，不至於太難完成。
    -   可以達成轉速的上限是477圈每分，這也是開啟Auto模式後達到的速度。開啟Spun Out模式時，速度有所降低，為287轉每分。
    -   學著嘗試去轉小圈（像Auto所做的一樣，但如果不習慣的話會把你的手玩壞）。
-   ［鼠標用戶請看］'''如果無論你怎麼轉，光標並不是在旋轉而是隨機亂跑，請在以下方法中選擇：查一下鼠標的移動速度上限、在平而光滑的木製表

面上使用鼠標、換個好點的鼠標墊。以上方法都不好用時，請買一個新的XD'''

-   -   地攤鼠可能經常遇到這種問題，最好**由慢到快地轉而不是突然高速旋轉**保證鼠標能夠適應。
    -   **建議使用光滑的木質表面或鼠標墊**。除非你的鼠標支持，否則最好不要在光滑的玻璃表面或坑洼的表面上使用。

### 遊戲時界面

![](Hierophobicintro.jpg "Hierophobicintro.jpg")

這是 osu! 模式遊戲時的界面。

-   左上角的長條是血條。
    -   生命值會按照固定的速度減少（取決於作圖者的設定），但是你可以通過正確的擊打物件來補充生命值。
    -   完全準確的擊打（300判定或「激」判定）補充的生命值要多於不准確的擊打（比如50判定）。
    -   完全未命中的擊打會導致大量扣血。
-   血條右側顯示你的得分。
-   得分下面表示準確率。
-   準確率旁邊的餅圖表示整個歌曲的進度，可以在「[設定](HK:Options#詳細設定 "wikilink")」中的「進度條位置」選項中修改。
-   屏幕左下角的數字表示連擊數。

得分計算
--------

本節說明了所有關於分數計算的方法，包括詳細的計算公式。

### 準確率

| colspan=2|準確率＝總擊打得分數÷（總擊打數×300） |
|-------------------------------------------------|
| 總擊打得分數                                    |
| 總擊打數                                        |

也就是說，每個Miss計0分，50判定計50分，100判定計100分，300判定計300分。這些分數的總和除以假設所有判定都是300的分數總和即為準確率。

比如你所有的判定都是300，準確率就是100%，而所有判定都是100時，則得到33.33%的準確率。

### 分數

每個圓圈和滑條尾端的計算公式如下：

| colspan=2|分數＝擊打判定值+擊打判定值×（連擊加成×難度加成×Mod加成）÷25 |
|------------------------------------------------------------------------|
| 擊打判定值                                                             |
| 連擊加成                                                               |
| 難度加成                                                               |
| Mod加成                                                                |

另外，在每個滑條的起始、結束和折返端都會得到30分，每個滑條中間點得到10分，轉圈每轉一圈得到100分。此外，轉盤轉滿後，每轉一圈還會得

到1000分的額外獎勵。

### 物件判定

**<u>圓圈</u>**

-   根據你擊打的準確程度，你將取得300、100、50的得分判定。
-   如果你根本沒有擊打或者擊打得太早，會得到miss判定。

**<u>滑條</u>**

-   滑條的判定不依賴於起始擊打的準確程度。
-   滑條包含中間點，包括開頭、結尾以及中間重複的小點。
-   如果命中所有的中間點，獲得300判定；如果命中至少一半，獲得100判定；只要命中一個以上，就可以獲得50判定。
-   **過早地點擊滑條**並不會導致Miss，但是將會**把分數加成置為0**。
-   只有當一個中間點都沒命中時，才會得到Miss判定。

**<u>轉盤</u>**

-   對於轉盤，會根據你轉的圈數與轉盤的總長度相比較來判定你得到300、100或50的判定。
-   如果你在轉盤結束之前未填滿轉盤能量條，則得到Miss判定。

### 分數加成

下列每一項都會為分數加成係數加一：

-   成功擊中圓圈
-   成功擊中滑條頭
-   成功擊中滑條尾
-   成功命中滑條中間點
-   成功完成或完成一半轉盤

以下每項都會將分數加成係數歸零：

-   圓圈獲得miss判定
-   過早擊中滑條頭
-   未擊中滑條頭或滑條中間點
-   未能充滿轉盤能量條的一半

以下情況不會影響分數加成係數：

-   未命中滑條尾（這將導致一個“喝”判定或100／50判定）
-   轉盤的1000分獎勵

### 評分

**<u>一般評分</u>**

-   SS = 100% 準確率
-   S = 超過90%的300判定，少於1%的50判定並且沒有miss判定
-   A = 超過80%的300判定並且無miss判定**或者**超過90%的300判定
-   B = 超過70%的300判定並且無miss判定**或者**超過80%的300判定
-   C = 超過60%的300判定
-   D = 其他情況

<u>**特殊評分**</u>

-   銀SS（SSH） = 100%準確率並開啟'[Hidden](HK:Hidden "wikilink")'和'[FlashLight](HK:Flashlight "wikilink")'兩個mod的任意組合。
-   銀S（SH） = 超過90%的300判定，少於1%的50判定並且無Miss判定並開啟'[Hidden](HK:Hidden "wikilink")'和'[FlashLight](HK:Flashlight "wikilink")'兩個Mod的任意組合。

<u>**關於個人頁面的“Ranks”數據**</u>

-   個人頁面的Ranks區域顯示的評分數由你在線打圖的評分數量確定。
    -   包括你在線時所進行的**離線遊戲**都**不計入**評分數。
-   **評分數與每個譜面難度是相關的，不是整個歌曲的**。
    -   如果譜面有3個難度，你分別都取得了SS，你將得到3個SS判定數而不是一個。

### 生命值

生命值的計算系統較為複雜所以不再詳細解釋。這些均與作圖者自己的設定有關。

下列情況會獲得生命值：

-   擊中一個圓圈。 （好的判定獲得更多的生命值。同一顏色的Combo中，最後一個圓圈獲得更多的生命值。）
-   擊中滑條頭、尾或折返端。 （滑條尾上，好的判定獲得更多的生命值。）
-   旋轉或完成轉盤（好的判定獲得更多的生命值）
-   完成轉盤後繼續旋轉（獲得的生命值取決於轉速）

下列情況會損失生命值：

-   固定扣血
-   在圓圈或滑條上取得miss判定
-   未完成轉盤
-   完成轉盤後未繼續轉

下列情況不會改變生命值：

-   休息時間
-   完成轉盤後繼續旋轉

常見問題
========

哪裡能找到譜面？
----------------

''關於譜面的詳細解釋位於條目[譜面中](HK:Beatmaps "wikilink")。 ''

你可以在主頁上的"[Beatmaps](http://osu.ppy.sh/p/beatmaplist)" 中進行搜索。

-   你需要登錄你的 osu! 賬戶，服務器不允許未登錄的用戶從官方網站進行下載。

我可以關閉Kiai時段嗎？
----------------------

不行。不像視頻，它們是遊戲的一部分，所以不能被關閉。

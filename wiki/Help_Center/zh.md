账号相关

我的国籍不对！
--------------

在注册时，会根据注册账号时的IP判断账号的所在国，并且根据该所在国进行排名。但是，在某种“你懂的”原因导致网站大姨妈的情况下，使用了一个“你懂的”方式进入了网站并且注册了账号，这就会导致国籍出错。虽然这并不影响游戏，但可能会影响你参与某些活动。

当然，我们可以**在获得能够充分证明你的国籍的情况下**受理修改国籍的请求。比如身份证、护照、驾驶证等等，然后找一张纸写下你的osu!账号（英文版是如此写的），并将两者提供给我们（accounts@ppy.sh）。在我们认同你的证据后，将会修改你的账号到正确的国籍。

**不要提供伪造的证据！不要提供伪造的证据！不要提供伪造的证据！**不然会遭到长达一个月的禁言甚至是永久封禁。

如果你连你的注册邮箱不能登录了，或者你已经真的不再记得你的账号是什么了。你也可以尝试发送邮件给我们（accounts@ppy.sh，下同），将你对你的账号的描述写下来，或许我们能够帮你找回账号。（包括但不限于你注册时使用的邮箱，或者改名之前的账号名，等等）

我被封号了，怎么办？
--------------------

**不作死就不会死**。首先请想想自己有没有做过以下作死的行为：使用外挂（包括但不限于键盘/鼠标模拟、修改封包、注入方式降低难度）、开小号、代打、账号交易、违反社区规则等等。

如果有，那就不用多想了，我们肯定是找到充分的理由去封禁你了！

如果你坚信这是我们误判，欢迎你发邮件给我们以证明你的清白。据统计，所有的解封请求中，只有万分之五是误封或特殊理由封禁的（比如早期的pp计算bug，因为某些人疯狂刷pp而暂时封号）。在封号期间，企图使用新账号或者伪造证据的行为规避封禁将**绝不会对账号进行解封**。

这个封禁操作，将**至少需要**维持半年时间。

这段时间过后，我们可以受理你的解封请求，使你重新回到 osu! 社区。但是，你要在请求中保证今后不会再次作死。这样我们才有充分的把握去解封你的账号。

*当然，我们会审核你以前的行为（比如事件的严重性、对其他用户的影响，等等），保留拒绝解封的权利。*

我们审核通过后，你就可以再次登录游戏。解封后将会由一个月的观察期，期间不能进行提交分数进行游戏排名。如果在观察期之后，你都没有做出作死的行为，你的账号将回到正常状态。

如果在解封后再次作死，那很抱歉，我们将会直接删除你的账号，并且不再受理任何有关该账号的请求。

封禁期间/被删号而永远离开之后你依然可以离线游戏、作图甚至是将你的beatmap、replay发布到一些osu!非官方社区或者交给你的朋友代为发布，等等。**但一定要记住不要再使用任何的旁门左道去获得成果了，没有人会乐意接受这样的结果的**。

我可以开小号/砍掉重练吗？
-------------------------

对于这个问题，一直以来，我们都**严禁玩家使用多个账号**进行游戏。使用小号进行游戏就**相当于人肉外挂**，对那些不断练习甚至开了废人模式（不提倡使用，这对你的健康不好）最终跻身前列的玩家并不公平。

除此之外，也**不允许任何的代打/代练行为**。如果你想介绍这款游戏给你的朋友，请**务必**在给你的朋友试玩游戏前，注销你的账号。如果你的朋友决定入坑，也请你引导你的朋友进行注册账号，并告知你的朋友：“这个账号只能你自己使用，不要给任何人使用，包括我。”

以下**均不是**注册小号的理由（也不限于下列现象）：

-   砍掉重练（这情况只发生过 3 例，而且都是一些特殊的原因）
-   因为被封号 / 删号而重新注册
-   区别对待某些玩家

**你只能使用你自己注册的第一个账号**。除此之外，不允许以任何理由再拥有其他账号。若你这样做，将会面临封禁。

如果你并不知情，注册了小号，在意识到不能使用小号之后并且在被发现之前，请立即停止使用你的小号，同时欢迎你发邮件给我们主动承认错误。我们会考虑减轻你的处罚，并且删除你的小号。

我的得分不见了！
----------------

是不是跟下列情况符合：

-   **我重新安装过游戏。**

重新安装游戏会清除**所有的本地成绩**（离线打图练级压游戏次数的要注意），不过在线的记录仍然保留在服务器上。如果你的 supporter 状态还有效，你还可以根据玩过的图在[这里](http://osu.ppy.sh/p/beatmaplist&s=4&r=0)筛选。

-   **我觉得我肯定玩过这首歌！**

是不是游戏模式选错了？每个模式的得分均不通用（因为计算方式不相同），切换游戏模式看看，是不是回来了？

-   **我离线玩游戏，完了之后登录，我的成绩呢？**

很遗憾，只会在在线时提交分数，而本地游戏的分数不会提交也无法提交。不过如果你在线游戏，却用着某个渣网络，这也不用担心，只要你是在线玩的，在退出游戏前，都会自动尝试提交分数（直到提交成功或者退出游戏为止）。

-   **我玩过很多图，可是我却看不到以前玩过的成绩！**

这个问题问的好，你在玩其他的图的时候，游戏会自动下载最近的排行榜。只需保持网络连接，就能下载完了。

我玩下落式模式时，经常都是某个键按不到！
----------------------------------------

早期键盘为了降低成本，使用了薄膜键盘（是的，在这个方法出现之前，都是机械键盘），因为传输方式的限制，会出现“键位冲突”的现象。大部分的薄膜键盘都**不能同时按下6个键以上**（甚至一部分还只能同时按下3个键）。

为了避免这个现象，rank 准则有“不能同时按下超过 6 个键”的规则。

如果真的是连6个键都不能同时按，那可能要考虑换一个键盘（十几块的地摊货都行）进行游戏，或者改一下键位避免按键冲突。

我经常都用最新版的客户端，怎么还提交不到成绩？

1.  网络是否正常？服务器要跨洋，丢包总是难免会发生的。
2.  删除所有的 dll（动态链接库）文件然后更新一遍试试。
3.  看看是不是被防火墙拦截或者被某些软件大量消耗带宽了。
4.  也可能是服务器大姨妈了，再等一下看看能不能上传？（1小时之内都会尝试提交成绩）

对于因为超时而无法提交成绩的问题，我们深感抱歉。我们会不断优化成绩的提交。如果在超时之后仍然无法提交，可以尝试以后再玩一遍，总能提交上去的！

我在游戏期间，鼠标晃来晃去怎么办？
----------------------------------

如果你能够在游戏中看到同时看到光标在某亮点，或者经常晃来晃去等不科学的移动轨迹，请在游戏中将“鼠标灵敏度”调整为 **1.0x** 。如果鼠标慢的像蜗牛，可以尝试下列调整方式（取决于你使用的设备）：

-   **鼠标**：有些鼠标（多数是游戏鼠标）设置了一个调整 DPI 的键，可以按下这个调整。如果没有这个键，可以在游戏中调整鼠标灵敏度。
-   **数位板**：请参考你的数位板的说明书，调整映射区域。
-   **osu! Tablet**：官方设备经过优化，只需在游戏中打开“原生输入”（Raw Input）即可。如果还不满意，还可以调整鼠标灵敏度。

Cursor stops when using tablet, but I'm still able to tap.
----------------------------------------------------------

In one of the recent updates, a feature called **Raw Input** was introduced. Sadly, tablets manufactured by **Wacom** and some mouses that use proprietary **drivers** are not supported by this feature yet, and using it causes user cursor to stop moving. To fix this issue, simply go to **Options** using input device that still works (mostly likely your mouse) and search for **Raw Input** checkbox and disable it.

Basic Troubleshooting Steps
---------------------------

-   **Step 1: Requirements**

Check to see if you're missing something. A good number of issues have been a result of a .Net Framework error, which should not be an issue at all if you've taken a close look at the [System Requirements](http://osu.ppy.sh/index.php?p=download) page. For those of you running any Windows environment, you can generally find most of these details by right-clicking your "My Computer" (or just "Computer" for Vista users) and selecting Properties from the mouse menu.

-   **Step 2: Updates**

Check to see if you have the latest version of osu!. This is obvious. Just run the updater. Also make sure all of your hardware drivers are up to date. The latest version of most GPU/Sound cards/other things can be found on the manufacturer's website. Failing that, you can just run a google search of your hardware model number and include "driver" in the search. It's pretty easy to find driver updates.

-   **Step 3: Operating System**

If you're all good with the above two steps, there's a good chance the problem you're having has to do with either the way your Operating System handles things, or better yet the problem has something to do with your machine specifically. While saying "I run XP" is a good start, don't forget to specify the service pack, and whether it's home/professional etc. Many operating systems have an entire army of flavours, and any issue you're having may be due to your specific hardware and software combination.

-   **Step 4: Environment**

In the event that you've made it through the first three steps and everything is fine, it's likely got to do with your computer's environment. Chances are, there are background processes running on your machine that are somehow interfering with osu!. Try closing downevery non-critical process on your machine before loading osu! and see if that solves the problem. Also, if you are having connection-specific issues, disable your firewall/internet security temporarily to see if it helps.

-   **步骤5: FAQ**

如果还是有问题没法解决的话，[读一下FAQ](FAQ "wikilink")。最好能阅读FAQ和按照FAQ的方法解决问题，不要用已有的问题浪费大家的时间。

-   **Step 6: Making the Thread**

If you still haven't solved the problem, the next thing you should do is run a search query in the Technical Support forum searching a few keywords relating to your error. Make sure your error is unique before you post a new thread about it. If it's not unique, and the thread is less than a couple months old, read the thread and post in there if necessary, Otherwise, you'll have to make a new thread - make sure the title is descriptive!

On to making the actual post. Here's what you should be posting, in no particular order:

1.  osu! build version (this can be located at the top-right of your options window, [as shown in this picture](http://puu.sh/4bDl8/c7a9a2562a.png)
2.  For issues unrelated to gameplay, System/OS base specifications (these can be easily found in [System Information](http://puu.sh/4bDaP/dffdbf900f.png) - easily accessible by Windows 7 and Vista users by simply typing 'system information' into the taskbar search - XP users will have to find it in Administrative Tools in the Control Panel - after you have that open, click OS name then shift-click the bottom most entry and hit Ctrl+C ([picture](http://puu.sh/4bDaP/dffdbf900f.png)))
3.  How exactly you discovered the bug with clear instructions on how to replicate it - include pictures if possible and any error messages you may have received - these are EXTREMELY important to the process. If we cannot replicate the problem, we can't even begin to fix it!

**[Make your post here](http://osu.ppy.sh/forum/posting.php?mode=post&f=5)**

Game Lag: Common Causes and Solutions
-------------------------------------

Simply telling us that you are experiencing "lag" is very unhelpful. "Lag" could be used to describe one of many possible symptoms, each having many different possible causes. Here are some of the different kinds of lag you might encounter:

-   **The game's behaviour (either audio, video, or both) is delayed from realtime.**

This could be a result of input lag - your mouse's actions are delayed. Possibly there are too many devices on your USB channel? If the lag is system-global, this is probably the case. Having desktop composition (Aero) causes a few milliseconds of input lag, so be sure to disable it if you wish for a perfect experience. If you don't wish to turn it off, running osu! in fullscreen will also do the trick. It could also be a result of output lag. Check that your video isn't set to prerender too many frames, and that there aren't too many filters, splitters, etc. between your PC and your monitor.

-   **The game's performance is choppy.**

This might simply be a result of low machine specs. Go into options and enable the FPS counter. If it's a value below 60 and the frame limiter is off, your graphics card might not be powerful enough to handle the game. Also check that other 3d apps in the background aren't eating up some of your precious graphics performance. If this is not enough, try checking the Low End PC option. DO NOT try switching to OpenGL to improve performance! At present, osu!'s OpenGL engine is neither as optimized nor as feature-rich as the Direct3D renderer. It is there for compatability reasons.

-   **Occasional momentary freezeups.**

Check if another background process is monopolizing your system for brief moments. This could be caused by other games, web browsers, or server apps (including file sharing/torrents). Close all other background programs and tray icons and see if the problem persists.

-   **FPS drops sharply after an update.**

This is known to happen to some users after some updates. Restart your PC to fix it in 99% of cases.

Some other quick solutions to improve performance.

-   你可以尝试使用全屏模式，在osu!设置里面更改分辨率下方选择使用，通常来说这会使你osu!的FPS上升。

## บทนำ

osu! ดำเนินงานโดยนิติบุคคลในออสเตรเลีย (ppy Pty Ltd) โดยเซิร์ฟเวอร์ส่วนใหญ่ทำงานจากสหรัฐอเมริกา

osu! ให้บริการหลากหลายแก่ผู้ใช้จากเกือบทุกประเทศทั่วโลก พร้อมคอมมูนิตี้ที่มีชีวิตชีวาและโอกาสมากมายให้ผู้ใช้แบ่งปันความคิดสร้างสรรค์กับผู้อื่นผ่านโปรไฟล์ผู้ใช้ บีตแมป (ด่านในเกม), ฟอรัม, ข้อความส่วนตัว, แชตในเกม และคอมเมนต์รีเพลย์

เพื่อให้บริการเหล่านี้แก่คุณ เรามักต้องเก็บรวบรวม จัดเก็บ และส่งต่อข้อมูลส่วนบุคคล เราพยายามโปร่งใสให้มากที่สุดในทุกด้านของการดำเนินงาน osu! นโยบายความเป็นส่วนตัวนี้จัดทำขึ้นด้วยเจตนาที่จะให้เราสอดคล้องกับเป้าหมายนั้น

เอกสารนี้มีเป้าหมายเพื่ออธิบายว่าเราเก็บข้อมูลใดบนเครือข่ายของเราและในการใช้งานผลิตภัณฑ์และบริการของเรา เราใช้ข้อมูลนั้นอย่างไร และเรามีตัวเลือกอะไรให้คุณควบคุมข้อมูลส่วนบุคคลของคุณ

การใช้บริการของเราถือว่าคุณยืนยันว่าคุณได้อ่านและเข้าใจ[ข้อกำหนดการให้บริการ](/wiki/Legal/Terms)และนโยบายความเป็นส่วนตัวนี้แล้ว รวมถึงวิธีและเหตุผลที่เราใช้ข้อมูลของคุณ และการใช้บริการของคุณอยู่ภายใต้ข้อกำหนดการให้บริการและนโยบายความเป็นส่วนตัวที่เกี่ยวข้อง หากคุณไม่ต้องการให้เราเก็บรวบรวมหรือประมวลผลข้อมูลส่วนบุคคลของคุณตามที่อธิบายไว้ที่นี่ คุณมีหลายทางเลือก รวมถึงจำกัดข้อมูลที่เราเก็บเกี่ยวกับคุณ หรือไม่ใช้เครือข่าย ผลิตภัณฑ์ และบริการของเรา

## ข้อมูลที่เราเก็บรวบรวม

### เมื่อสมัครบัญชี

แม้จะใช้งานบางส่วนได้โดยไม่ต้องมีบัญชี แต่บ่อยครั้งผู้ใช้จำเป็นต้องสมัครบัญชีเพื่อใช้งานบริการบางอย่าง เมื่อสมัครบัญชี เราจะจัดเก็บ

- username ของคุณ
- ที่อยู่อีเมลของคุณ
- รหัสผ่านของคุณ (bcrypt+salt)
- ที่อยู่ IP และประเทศของคุณ

ข้อมูลส่วนบุคคลนี้ ยกเว้น username และประเทศของคุณ จะไม่ถูกแชร์สู่สาธารณะ

### เมื่ออัปเดตโปรไฟล์

เมื่อสร้างโปรไฟล์ผู้ใช้ของคุณ (ซึ่งผู้ใช้คนอื่นทั้งหมดเห็นได้สาธารณะ) คุณสามารถระบุข้อมูลต่อไปนี้ได้ตามต้องการ

- ตำแหน่งปัจจุบันของคุณ
- ความสนใจของคุณ
- อาชีพของคุณ
- ตัวตนบนโซเชียลมีเดียของคุณ (Twitter, Discord, website)
- รูป avatar และรูป cover โปรไฟล์ของคุณ
- ลายเซ็นของคุณ

ช่องทั้งหมดข้างต้นเห็นได้สาธารณะ แต่สามารถถอนออกได้ทันทีและถาวรเมื่อใดก็ได้จาก[หน้า settings](https://osu.ppy.sh/home/account/edit)

### เมื่ออัปโหลดคอนเทนต์ที่ผู้ใช้ส่งเข้ามา

เมื่อโพสต์บนฟอรัม เข้าร่วมแชต หรืออัปโหลดคอนเทนต์ไปยังบริการของเรา เช่น บีตแมป คุณกำลังเผยแพร่คอนเทนต์ที่ส่งเข้ามาทั้งหมดอย่างชัดเจน ในกรณีส่วนใหญ่ คุณสามารถแก้ไขและลบได้หลังส่งตามดุลพินิจของคุณ แต่ในบางกรณี ฟังก์ชันนี้อาจถูกล็อกไว้เพื่อรักษาความเกี่ยวข้องและการให้บริการแก่ฐานผู้ใช้ของเรา

ตัวอย่างเช่น หากคุณอัปโหลดบีตแมปและบีตแมปนั้นถูก "ranked" บีตแมปนั้นจะกลายเป็นฐานให้ผู้ใช้จำนวนมากทำคะแนนบนแมปดังกล่าว ณ จุดนั้น ตัวเลือกในการลบผลงานที่ส่งเข้ามาจะถูกเพิกถอน

### เมื่อเข้าสู่ระบบไคลเอนต์เกม

เมื่อเชื่อมต่อกับบริการของเราจากไคลเอนต์เกม osu! จะมีการส่ง string เฉพาะไคลเอนต์มาเพื่อช่วยให้เราระบุสภาพแวดล้อมการเล่นปัจจุบันของคุณ string นี้สร้างจากการผสมผสาน identifier จากการตั้งค่า hardware และ software ของคุณ แล้วถูก hash ในลักษณะที่ไม่มีข้อมูลที่ระบุตัวตนได้โดยตรง แต่ใช้ติดตามการเข้าถึงของคุณตลอดการเข้าสู่ระบบบริการของเราได้

วัตถุประสงค์หลักของสิ่งนี้คือเพื่อรักษาระบบจัดอันดับที่ยุติธรรม และช่วยให้เราบังคับใช้ความปลอดภัยของบัญชีหากบัญชีของคุณถูกเข้าถึงจากตำแหน่งที่ถูก compromise ข้อมูลนี้ถือเป็นข้อมูลส่วนตัวและจัดเก็บไว้ตราบเท่าที่เห็นว่ายังเกี่ยวข้องเท่านั้น นอกจากนี้ยังถ่ายโอนไม่ได้ และไม่มีความหมายภายนอก ecosystem ของ osu!

### เมื่อเล่นเกมและส่งคะแนน

เมื่อจบเซสชันเกม (ไม่ว่าจะผ่านหรือไม่ผ่านบีตแมป) รายละเอียดเกี่ยวกับ performance ของคุณจะถูกส่งไปยังเซิร์ฟเวอร์ของเราโดยอัตโนมัติ ส่วนคะแนนของข้อมูลที่ส่งนี้รวมถึงข้อมูล replay ของเกม และอาจแสดงต่อสาธารณะใน Global Leaderboards และบน User Profile ของคุณ รวมถึงไม่สามารถลบหรือแก้ไขได้

### Anti-cheat

osu! มีโค้ดที่รันได้ซึ่งใช้ตรวจจับการใช้งานโปรแกรมโกง เป้าหมายของ anti-cheat ของ osu! คือการรักษาสภาพแวดล้อมการเล่นที่ยุติธรรมและแข่งขันได้สำหรับผู้เล่นทุกคน โดยไม่กระทบ performance การเล่นหรือความเป็นส่วนตัวของผู้ใช้

- การตรวจจับโปรแกรมโกงเกิดขึ้นบนอุปกรณ์ของคุณ และไม่มีข้อมูลที่ไม่จำเป็นถูกส่งไปยังเซิร์ฟเวอร์ของเราในระหว่างกระบวนการตรวจจับ
- หาก anti-cheat ตัดสินว่าคุณกำลังโกง ข้อเท็จจริงนี้พร้อมหลักฐานในรูป diagnostics และ metadata จะถูกส่งไปยังเซิร์ฟเวอร์ของเราเพื่อตรวจสอบ หากคุณไม่ได้โกง จะไม่มีการส่งข้อมูล anti-cheat
- แม้เมื่อมีการตรวจจับ เราจะพยายามอย่างดีที่สุดที่จะไม่ส่งข้อมูลใด ๆ ภายนอก ecosystem ของ osu! ที่สามารถใช้ระบุตัวตนของคุณได้
- metadata ที่ส่งมาจะถูกเก็บไว้บนเซิร์ฟเวอร์ของเราเฉพาะช่วงเวลาที่เป็นประโยชน์ โดยทั่วไปคือหลายชั่วโมงถึงหลายวันระหว่างที่เราวิเคราะห์คอนเทนต์ที่ถูกรายงาน
- การวิเคราะห์ส่วนใหญ่เป็นอัตโนมัติ metadata ที่ส่งมาไม่สามารถเข้าถึงได้โดยสมาชิก support team และต้องเข้าถึงผ่านระบบความปลอดภัยหลายชั้น เฉพาะ database server admins เท่านั้นที่ดู metadata ได้

เราเคารพและให้คุณค่ากับความเป็นส่วนตัวของคุณ และไม่ต้องการทำให้ผู้ใช้ที่เล่นถูกต้องรู้สึกหวาดกลัวจากการมี anti-cheat อยู่ในเกม

### Logging

เราใช้ error logging ซึ่งเก็บข้อมูลทางเทคนิคและข้อมูลการใช้งานระหว่างที่คุณใช้บริการของเรา ข้อมูลนี้อาจรวมถึง IP address, username ของคุณ, ประเภทและเวอร์ชันของ browser, การตั้งค่า time zone และตำแหน่ง, operating system และ platform รวมถึงรายละเอียดอื่นเกี่ยวกับอุปกรณ์ที่คุณใช้เข้าถึงบริการของเรา

ข้อมูลที่เก็บรวบรวมนี้จะถูกรวมเป็น aggregate และเก็บไว้เฉพาะตราบเท่าที่ยังเป็นประโยชน์ โดยทั่วไป ระยะเวลาจัดเก็บข้อมูลที่ไม่ได้ aggregate จะน้อยกว่าหนึ่งเดือน พร้อมกฎ purge อัตโนมัติ

## การเปิดเผยข้อมูลส่วนบุคคลของคุณ

เราไม่ทำ marketing, advertising หรือส่งอีเมลที่ไม่ได้ร้องขอใด ๆ อีเมลเดียวที่คุณจะได้รับจากเราคือผลจากการกระทำบนบริการของเรา (เช่น การขอ two-factor authentication, การซื้อสินค้า หรือการเปิด notifications สำหรับ discussion)

เราไม่แชร์ข้อมูลส่วนบุคคลของคุณกับบุคคลที่สามใด ๆ analytics ทั้งหมดโฮสต์ภายในองค์กร ข้อมูล diagnostic ที่จำกัดอาจถูกเปิดให้ developer ที่เชื่อถือได้ในทีมใช้วิเคราะห์และแก้ปัญหาของเกมหรือเว็บไซต์

โปรดทราบว่ามีบุคคลที่สามบางรายที่คุณอาจพบเมื่อทำบางการกระทำ:

- การส่ง support ticket ทางอีเมลจัดการผ่าน Enchant
- การสั่งซื้อสินค้าจริงจัดการผ่าน Shopify หรือ partner ด้านสินค้ารายอื่น
- supporter tags และการซื้อที่ไม่ใช่สินค้าจริงอื่น ๆ จัดการโดย Xsolla หรือ PayPal เราไม่เก็บข้อมูล fulfilment หรือข้อมูลการชำระเงินใด ๆ

## สิทธิ์และการควบคุมของคุณ {#your-rights-and-control}

ในฐานะผู้ใช้ คุณมีสิทธิ์ย้าย อัปเดต หรือลบข้อมูลส่วนบุคคลของคุณ สิ่งนี้ทำได้หลัก ๆ จาก [user settings](https://osu.ppy.sh/home/account/edit) หรือหากไม่มีให้ใช้ฟีเจอร์ "Edit" ที่ localized อยู่ในส่วนที่เกี่ยวข้องของเว็บไซต์เรา หากคุณต้องการดึงข้อมูลบัญชีทั้งหมดของคุณด้วยโปรแกรม กรุณาใช้ [public API](https://github.com/ppy/osu-api/wiki) ของเรา

ในหลายกรณี user submissions เช่น โพสต์ฟอรัมและบีตแมป สามารถลบได้เป็นรายรายการ คุณจะพบปุ่มลบที่เชื่อมกับรายการที่ลบได้โดยตรง

คุณมีตัวเลือกในการลบบัญชีของคุณออกจากบริการของเรา โปรดทราบว่าปัจจุบันนี่เป็นกระบวนการแบบ manual และอาจใช้เวลาหลายวันจึงเสร็จสมบูรณ์ ([ติดต่อเรา](mailto:privacy@ppy.sh) เพื่อยื่นคำขอ) ในกรณีลบบัญชี บางส่วนของผลงานสาธารณะของคุณอาจยังคงอยู่ตามรายละเอียดใน "ข้อมูลที่เราเก็บรวบรวม"

เนื่องจากเรามีนโยบายเข้มงวดแบบหนึ่งบัญชีต่อผู้ใช้หนึ่งคนเพื่อรักษา leaderboard ที่ยุติธรรม หลังลบบัญชี คุณอาจไม่สามารถกลับมาใช้บริการของเราได้ เพื่อบังคับใช้นโยบายนี้ เราจะเก็บ unique identifying string ตามที่กล่าวไว้ใน "ข้อมูลที่เราเก็บรวบรวม" ต่อไปแม้หลังลบบัญชีแล้ว สิ่งนี้ทำขึ้นในฐานะ [legitimate interest ตามที่ระบุไว้ใน GDPR](https://ico.org.uk/for-organisations/guide-to-data-protection/guide-to-the-general-data-protection-regulation-gdpr/legitimate-interests/when-can-we-rely-on-legitimate-interests/) เพื่อบังคับใช้กฎของเรา โปรดมั่นใจว่าข้อมูลนี้ถูก hash ทางเดียว และไม่สามารถใช้ระบุตัวคุณภายนอก osu! หรือเกินบริบทเฉพาะของการบังคับใช้กฎนี้ได้

## Cookies

เราใช้ cookies เช่นเดียวกับอินเทอร์เน็ตส่วนใหญ่ เราใช้มันอย่างเคร่งครัดเพื่อรักษาสถานะ session และ login credentials ของคุณข้ามหลาย session หากคุณแพ้ cookies กรุณาอย่าใช้บริการของเรา
